## <a name="what-about-microservices"></a><span data-ttu-id="123ae-136">Und Microservices...?</span><span class="sxs-lookup"><span data-stu-id="123ae-136">What about microservices?</span></span>

<span data-ttu-id="123ae-138">Die Microservices-Architektur ist ein komplexer Ansatz, den Sie für Anwendungen verwenden können, die von Grund auf neu erstellt werden, oder wenn Sie vorhandene Anwendungen in Richtung cloudnative Anwendungen weiterentwickeln.</span><span class="sxs-lookup"><span data-stu-id="123ae-138">The microservices architecture is an advanced approach that you can use for applications that are created from scratch or when you evolve existing applications toward cloud-native applications.</span></span> <span data-ttu-id="123ae-139">Sie können damit beginnen, dass Sie vorhandenen Anwendungen ein paar Microservices hinzufügen, um mehr über die neuen Paradigmen von Microservices zu erfahren.</span><span class="sxs-lookup"><span data-stu-id="123ae-139">You can start by adding a few microservices to existing applications to learn about the new microservices paradigms.</span></span> <span data-ttu-id="123ae-140">Natürlich müssen Sie aber auch speziell auf diesen Architekturansatz hin die Architektur entwickeln und programmieren.</span><span class="sxs-lookup"><span data-stu-id="123ae-140">But clearly, you need to architect and code, especially for this type of architectural approach.</span></span>

<span data-ttu-id="123ae-141">Microservices sind allerdings für keine neue oder moderne Anwendung obligatorisch.</span><span class="sxs-lookup"><span data-stu-id="123ae-141">However, microservices are not mandatory for any new or modern application.</span></span> <span data-ttu-id="123ae-142">Bei Microservices handelt es sich nicht um eine „Wunderwaffe“, und sie sind nicht die einzige, optimale Methode, um jede Anwendung zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="123ae-142">Microservices are not a "magic bullet," and they aren't the single, best way to create every application.</span></span> <span data-ttu-id="123ae-143">Wie und wann Sie Microservices verwenden, hängt vom Typ der Anwendung ab, die Sie erstellen müssen.</span><span class="sxs-lookup"><span data-stu-id="123ae-143">How and when you use microservices depends on the type of application that you need to build.</span></span>
