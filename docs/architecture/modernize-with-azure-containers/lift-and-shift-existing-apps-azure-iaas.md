## <a name="why-migrate-existing-net-web-applications-to-azure-iaas"></a><span data-ttu-id="ed42a-113">Warum vorhandene .NET-Webanwendungen zu Azure IaaS migrieren?</span><span class="sxs-lookup"><span data-stu-id="ed42a-113">Why migrate existing .NET web applications to Azure IaaS</span></span>

<span data-ttu-id="ed42a-116">Nachdem Sie die Entscheidung getroffen haben, Ihre Apps in die Cloud zu verlagern, ist der Hauptgrund für die Wahl von IaaS anstelle von komplexeren Optionen wie PaaS einfach der, dass die IaaS-Umgebung vertrauter sein wird.</span><span class="sxs-lookup"><span data-stu-id="ed42a-116">After you make the decision to move your apps to the cloud, the main reason why you might choose IaaS instead of more advanced options like PaaS is simply that the IaaS environment will be more familiar.</span></span> <span data-ttu-id="ed42a-117">Beim Umzug in eine Umgebung, die Ihrer aktuellen lokalen Umgebung ähnelt, bietet die lokale Umgebung eine flachere Lernkurve, was dies zum schnellsten Weg in die Cloud macht.</span><span class="sxs-lookup"><span data-stu-id="ed42a-117">Moving to an environment that's similar to your current, on-premises environment offers a lower learning curve, which makes it the quickest path to the cloud.</span></span>
