### <a name="working-with-the-orchestration-client"></a><span data-ttu-id="ed86c-124">Arbeiten mit dem Orchestrierungsclient</span><span class="sxs-lookup"><span data-stu-id="ed86c-124">Working with the Orchestration client</span></span>

<span data-ttu-id="ed86c-125">Orchestrierungen sind im Vergleich zu anderen Arten von ausgelösten Vorgängen in Azure Functions eindeutig.</span><span class="sxs-lookup"><span data-stu-id="ed86c-125">Orchestrations are unique when compared to other styles of triggered operations in Azure Functions.</span></span> <span data-ttu-id="ed86c-126">Durable Functions ermöglicht die Ausführung von Funktionen, die bis zu ihrem Abschluss mehrere Stunden oder sogar Tage dauern können.</span><span class="sxs-lookup"><span data-stu-id="ed86c-126">Durable Functions enables the execution of functions that may take hours or even days to complete.</span></span> <span data-ttu-id="ed86c-127">Diese Art von Verhalten bringt die Notwendigkeit mit sich, den Status einer aktiven Orchestrierung zu überprüfen, sie vorzeitig zu beenden oder Benachrichtigungen zu externen Ereignissen zu senden.</span><span class="sxs-lookup"><span data-stu-id="ed86c-127">That type of behavior comes with the need to able to check the status of a running orchestration, preemptively terminate, or send notifications of external events.</span></span>

<span data-ttu-id="ed86c-128">Für solche Fälle stellt die Durable Functions-Erweiterung die `DurableOrchestrationClient`-Klasse bereit, die Ihnen die Interaktion mit orchestrierten Funktionen ermöglicht.</span><span class="sxs-lookup"><span data-stu-id="ed86c-128">For such cases, the Durable Functions extension provides the `DurableOrchestrationClient` class that allows you to interact with orchestrated functions.</span></span> <span data-ttu-id="ed86c-129">Sie erhalten Zugriff auf den Orchestrierungclient, indem Sie die `OrchestrationClientAttribute`-Bindung verwenden.</span><span class="sxs-lookup"><span data-stu-id="ed86c-129">You get access to the orchestration client by using the `OrchestrationClientAttribute` binding.</span></span> <span data-ttu-id="ed86c-130">Im Allgemeinen würden Sie dieses Attribut in einen anderen Triggertyp einbinden, z.B. `HttpTrigger` oder `ServiceBusTrigger`.</span><span class="sxs-lookup"><span data-stu-id="ed86c-130">Generally, you would include this attribute with another trigger type, such as an `HttpTrigger` or `ServiceBusTrigger`.</span></span> <span data-ttu-id="ed86c-131">Nachdem die Quellfunktion ausgelöst wurde, kann der Orchestrierungsclient zum Starten einer Orchestratorfunktion verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="ed86c-131">Once the source function has been triggered, the orchestration client can be used to start an orchestrator function.</span></span>

```csharp
[FunctionName("KickOff")]
public static async Task<HttpResponseMessage> Run(
    [HttpTrigger(AuthorizationLevel.Function, "POST")]HttpRequestMessage req,
    [OrchestrationClient ] DurableOrchestrationClient<orchestrationClient>)
{
    OrderRequestData data = await req.Content.ReadAsAsync<OrderRequestData>();

    string instanceId = await orchestrationClient.StartNewAsync("PlaceOrder", data);

    return orchestrationClient.CreateCheckStatusResponse(req, instanceId);
}
```

> <span data-ttu-id="ed86c-140">*Fehlerbehandlungscode wurde aus Platzgründen ausgelassen*</span><span class="sxs-lookup"><span data-stu-id="ed86c-140">*Error handling code has been left out for brevity*</span></span>

```csharp
[FunctionName("PlaceOrder")]
public static async Task<string> PlaceOrder([OrchestrationTrigger] DurableOrchestrationContext context)
{
    OrderRequestData orderData = context.GetInput<OrderRequestData>();

    await context.CallActivityAsync<bool>("CheckAndReserveInventory", orderData);
    await context.CallActivityAsync<string>("ProcessPayment", orderData);

    string trackingNumber = await context.CallActivityAsync<string>("ScheduleShipping", orderData);
    await context.CallActivityAsync<string>("EmailCustomer", trackingNumber);

    return trackingNumber;
}
```

<span data-ttu-id="ed86c-152">`ActivityTriggerAttribute` wird verwendet, um einen Funktionsparameter vom Typ `DurableActivityContext` zu kommentieren.</span><span class="sxs-lookup"><span data-stu-id="ed86c-152">The `ActivityTriggerAttribute` is used to annotate a function parameter of type `DurableActivityContext`.</span></span> <span data-ttu-id="ed86c-153">Mithilfe der Anmerkung wird der Laufzeit mitgeteilt, dass die-Funktion als Aktivitätsfunktion verwendet werden soll.</span><span class="sxs-lookup"><span data-stu-id="ed86c-153">Using the annotation informs the runtime that the function is intended to be used as an activity function.</span></span> <span data-ttu-id="ed86c-154">Eingabewerte für Aktivitätsfunktionen werden mit der `GetInput<T>`-Methode des `DurableActivityContext`-Parameters abgerufen.</span><span class="sxs-lookup"><span data-stu-id="ed86c-154">Input values to activity functions are retrieved using the `GetInput<T>` method of the `DurableActivityContext` parameter.</span></span>
