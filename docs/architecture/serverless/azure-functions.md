<span data-ttu-id="72228-167">In der folgenden Beispieldatei *functions.json* wird ein Trigger und eine Bindung definiert:</span><span class="sxs-lookup"><span data-stu-id="72228-167">The following example *functions.json* file defines a trigger and a binding:</span></span>

```json
{
  "bindings": [
    {
      "name": "myBlob",
      "type": "blobTrigger",
      "direction": "in",
      "path": "images/{name}",
      "connection": "AzureWebJobsStorage"
    },
    {
      "name": "$return",
      "type": "queue",
      "direction": "out",
      "queueName": "images",
      "connection": "AzureWebJobsStorage"
    }
  ],
  "disabled": false
}
```

<span data-ttu-id="72228-168">In diesem Beispiel wird die Funktion durch eine Änderung am Blobspeicher im `images`-Container ausgelöst.</span><span class="sxs-lookup"><span data-stu-id="72228-168">In this example, the function is triggered by a change to blob storage in the `images` container.</span></span> <span data-ttu-id="72228-169">Die Informationen für die Datei werden übergeben, sodass der Trigger auch als Bindung fungiert.</span><span class="sxs-lookup"><span data-stu-id="72228-169">The information for the file is passed in, so the trigger also acts as a binding.</span></span> <span data-ttu-id="72228-170">Eine weitere Bindung ist vorhanden, um Informationen in eine Warteschlange namens `images`einzufügen.</span><span class="sxs-lookup"><span data-stu-id="72228-170">Another binding exists to put information onto a queue named `images`.</span></span>

<span data-ttu-id="72228-171">Dies ist das C#-Skript für die Funktion:</span><span class="sxs-lookup"><span data-stu-id="72228-171">Here is the C# script for the function:</span></span>

```csharp
    log.Info($"C# Blob trigger function Processed blob\n Name:{name} \n Size: {myBlob.Length} Bytes");
    return name;
}
```

<span data-ttu-id="72228-172">Das Beispiel ist eine einfache Funktion, die den Namen der Datei annimmt, die geändert oder in den Blobspeicher hochgeladen wurde, und sie zur späteren Verarbeitung in einer Warteschlange platziert.</span><span class="sxs-lookup"><span data-stu-id="72228-172">The example is a simple function that takes the name of the file that was modified or uploaded to blob storage, and places it on a queue for later processing.</span></span>
