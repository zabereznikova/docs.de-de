<span data-ttu-id="7df82-104">*Das Entwickeln von Containeranwendungen mit Microservice bedeutet, dass Sie Anwendungen mit mehreren Containern erstellen. <span data-ttu-id="7df82-105">Zuvor wurde die Frage aufgeworfen, ob Docker erforderlich ist, um eine Microservice-Architektur zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="7df82-105">Earlier we raised the question "Is Docker necessary when building a microservice architecture?"</span></span> <span data-ttu-id="7df82-106">Die Antwort ist ein klares Nein.</span><span class="sxs-lookup"><span data-stu-id="7df82-106">The answer is a clear no.</span></span> <span data-ttu-id="7df82-107">Docker ist ein Hilfsmittel, das bedeutende Vorteile bietet. <span data-ttu-id="7df82-109">Wenn Sie jedoch eine auf Microservices und Docker-Containern basierte Anwendung entwerfen und entwickeln können, können Sie auch ein anderes, einfacheres Anwendungsmodell entwerfen und entwickeln.</span><span class="sxs-lookup"><span data-stu-id="7df82-109">However, if you know how to design and develop a microservices-based application that is also based on Docker containers, you will be able to design and develop any other, simpler application model.</span></span> <span data-ttu-id="7df82-110">Sie können beispielsweise eine Anwendung mit drei Ebenen entwerfen, für die mehrere Container erforderlich sind.</span><span class="sxs-lookup"><span data-stu-id="7df82-110">For example, you might design a three-tier application that also requires a multi-container approach.</span></span> <span data-ttu-id="7df82-111">Deshalb, und weil Microservice-Architekturen ein wichtiger Trend innerhalb der Container-Welt sind, konzentriert sich dieser Abschnitt auf die Implementierung von Microservice-Architekturen mithilfe von Docker-Containern.</span><span class="sxs-lookup"><span data-stu-id="7df82-111">Because of that, and because microservice architectures are an important trend within the container world, this section focuses on a microservice architecture implementation using Docker containers.</span></span>
