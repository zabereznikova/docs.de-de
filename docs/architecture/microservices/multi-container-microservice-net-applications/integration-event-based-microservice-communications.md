<span data-ttu-id="b65b2-178">Zunächst werden Implementierungscode für die Ereignisbusschnittstelle und mögliche Implementierungen zu Untersuchungszwecken veranschaulicht.</span><span class="sxs-lookup"><span data-stu-id="b65b2-178">Let's start with some implementation code for the event bus interface and possible implementations for exploration purposes.</span></span> <span data-ttu-id="b65b2-179">Die Schnittstelle sollte wie die folgende Schnittstelle allgemein und einfach gehalten sein.</span><span class="sxs-lookup"><span data-stu-id="b65b2-179">The interface should be generic and straightforward, as in the following interface.</span></span>

```csharp
public interface IEventBus
{
    void Publish(IntegrationEvent @event);

    void Subscribe<T, TH>()

        where TH : IIntegrationEventHandler<T>;

    void SubscribeDynamic<TH>(string eventName)

    void Unsubscribe<T, TH>()
        where TH : IIntegrationEventHandler<T>
        where T : IntegrationEvent;
}
```

<span data-ttu-id="b65b2-183">Die `Subscribe`-Methoden (abhängig von den Argumenten können mehrere Implementierungen verwendet werden) werden von den Microservices verwendet, die Ereignisse empfangen möchten.</span><span class="sxs-lookup"><span data-stu-id="b65b2-183">The `Subscribe` methods (you can have several implementations depending on the arguments) are used by the microservices that want to receive events.</span></span> <span data-ttu-id="b65b2-184">Diese Methode weist zwei Argumente auf.</span><span class="sxs-lookup"><span data-stu-id="b65b2-184">This method has two arguments.</span></span> <span data-ttu-id="b65b2-185">Beim ersten Argument handelt es sich um das Integrationsereignis, das abonniert werden kann (`IntegrationEvent`).</span><span class="sxs-lookup"><span data-stu-id="b65b2-185">The first is the integration event to subscribe to (`IntegrationEvent`).</span></span> <span data-ttu-id="b65b2-186">Beim zweiten Argument handelt es sich um den Integrationsereignishandler (oder die Rückrufmethode) mit dem Namen `IIntegrationEventHandler<T>`.
