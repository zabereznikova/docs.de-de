## <a name="deploying-a-monolithic-application-as-a-container"></a><span data-ttu-id="9c624-145">Bereitstellen einer monolithischen Anwendung als Container</span><span class="sxs-lookup"><span data-stu-id="9c624-145">Deploying a monolithic application as a container</span></span>

<span data-ttu-id="9c624-146">Das Verwenden von Containern zur Verwaltung monolithischer Anwendungsbereitstellungen hat Vorteile.</span><span class="sxs-lookup"><span data-stu-id="9c624-146">There are benefits to using containers to manage monolithic application deployments.</span></span> <span data-ttu-id="9c624-147">Das Skalieren von Containerinstanzen ist wesentlich schneller und einfacher als die Bereitstellung zusätzlicher VMs.</span><span class="sxs-lookup"><span data-stu-id="9c624-147">Scaling container instances is far faster and easier than deploying additional VMs.</span></span> <span data-ttu-id="9c624-148">Selbst wenn Sie VM-Skalierungsgruppen verwenden, benötigt der Start der VMs Zeit.</span><span class="sxs-lookup"><span data-stu-id="9c624-148">Even if you use virtual machine scale sets, VMs take time to start.</span></span> <span data-ttu-id="9c624-149">Bei der Bereitstellung als herkömmliche Anwendungsinstanzen anstelle von Containern wird die Konfiguration der Anwendung im Rahmen der VM verwaltet.
