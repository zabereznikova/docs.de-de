### <a name="why-razor-pages"></a><span data-ttu-id="adec6-126">Was spricht für Razor Pages?</span><span class="sxs-lookup"><span data-stu-id="adec6-126">Why Razor Pages?</span></span>

## <a name="mapping-requests-to-responses"></a><span data-ttu-id="adec6-145">Zuordnen von Anforderungen zu Antworten</span><span class="sxs-lookup"><span data-stu-id="adec6-145">Mapping requests to responses</span></span>

<span data-ttu-id="adec6-146">Im Wesentlichen dienen ASP.NET Core-Apps dazu, eingehende Anforderungen ausgehenden Antworten zuzuordnen.</span><span class="sxs-lookup"><span data-stu-id="adec6-146">At its heart, ASP.NET Core apps map incoming requests to outgoing responses.</span></span> <span data-ttu-id="adec6-147">Auf niedriger Ebene wird für diese Zuordnung Middleware verwendet. <span data-ttu-id="adec6-168">Nachdem eine bestimmte Anforderung einer Route zugeordnet wurde, aber noch bevor die Aktionsmethode aufgerufen wird, führt ASP.NET Core MVC Vorgänge zur
