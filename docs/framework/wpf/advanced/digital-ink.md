# <a name="digital-ink"></a>Freihandeingaben
In diesem Abschnitt erläutert die Verwendung von Freihandeingaben in der [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]. Bisher nur im Tablet PC SDK gefunden, ist Freihandeingaben jetzt im Windows Presentation Foundation-Kern verfügbar. Dies bedeutet, dass Sie vollständige Tablet PC-Anwendungen jetzt Dank der Leistungsfähigkeit von Windows Presentation Foundation entwickeln können.
