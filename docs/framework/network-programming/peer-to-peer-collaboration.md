<span data-ttu-id="6819c-103">Peer-zu-Peer Networking ist die Verwendung relativ leistungsfähiger Computer (Personalcomputer), die am Rand des Internets für mehr als nur clientbasierte Rechenaufgaben vorhanden sind.</span><span class="sxs-lookup"><span data-stu-id="6819c-103">Peer-to-peer networking is the utilization of the relatively powerful computers (personal computers) that exist at the edge of the Internet for more than just client-based computing tasks.</span></span> <span data-ttu-id="6819c-104">Der moderne Personalcomputer (PC) verfügt über einen sehr schnellen Prozessor, einen beträchtlichen Arbeitsspeicher und eine große Festplatte, die beim Ausführen von allgemeinen Rechenaufgaben, wie z.B. E-Mail und Webbrowsen allesamt nicht voll ausgelastet werden.</span><span class="sxs-lookup"><span data-stu-id="6819c-104">The modern personal computer (PC) has a very fast processor, vast memory, and a large hard disk, none of which are being fully utilized when performing common computing tasks such as email and Web browsing.</span></span> <span data-ttu-id="6819c-105">Der moderne PC kann einfach sowohl als Client als auch als Server (kein Peer) für viele Arten von Anwendungen fungieren.</span><span class="sxs-lookup"><span data-stu-id="6819c-105">The modern PC can easily act as both a client and server (a peer) for many types of applications.</span></span>  
  
<span data-ttu-id="6819c-106">Die Peer-zu-Peer-Kollaborationsinfrastruktur ist eine vereinfachte Implementierung der Microsoft Windows-Peer-zu-Peer-Infrastruktur, die den Dienst „Personen in meiner Umgebung“ in Windows Vista und späteren Plattformen nutzt.</span><span class="sxs-lookup"><span data-stu-id="6819c-106">The Peer-to-Peer Collaboration Infrastructure is a simplified implementation of the Microsoft Windows Peer-to-Peer Infrastructure that leverages the People Near Me service in Windows Vista and later platforms.</span></span> <span data-ttu-id="6819c-107">Sie wird am besten für Peer-fähige Anwendungen innerhalb eines Subnetzes verwendet, für die der „Personen in meiner Umgebung“-Dienst ausgeführt wird, obwohl sie auch Internetendpunkte oder Kontakte bedienen kann.</span><span class="sxs-lookup"><span data-stu-id="6819c-107">It is best used for peer-enabled applications within a subnet for which the People Near Me service operates, although it can service internet endpoints or contacts as well.</span></span> <span data-ttu-id="6819c-108">Sie beinhaltet den gängigen Contact Manager, der vom Live Messenger und anderen Live-fähigen Anwendungen verwendet wird, um die Kontaktendpunkte, die Verfügbarkeit und die Präsenz zu bestimmen.</span><span class="sxs-lookup"><span data-stu-id="6819c-108">It incorporates the common Contact Manager that is used by Live Messenger and other Live-aware applications to determine contact endpoints, availability, and presence.</span></span>  
  
## <a name="collaboration-applications"></a><span data-ttu-id="6819c-109">Anwendungen für die Zusammenarbeit</span><span class="sxs-lookup"><span data-stu-id="6819c-109">Collaboration applications</span></span>

 <span data-ttu-id="6819c-110">Eine typische Peer-zu-Peer-Kollaborationsanwendung umfasst die folgenden Schritte:</span><span class="sxs-lookup"><span data-stu-id="6819c-110">A typical peer-to-peer collaboration application is comprised of the following steps:</span></span>  
  
- <span data-ttu-id="6819c-111">Der Peer bestimmt die Identität eines Peers, der sich für das Hosten einer Kollaborationssitzung interessiert.</span><span class="sxs-lookup"><span data-stu-id="6819c-111">Peer determines the identity of a peer who is interested in hosting a collaboration session</span></span>  
  
- <span data-ttu-id="6819c-112">Eine Anforderung zum Hosten einer Sitzung wird aus irgendeinem Grund gesendet, und der Peer erklärt sich dazu bereit, die Kollaborationstätigkeit zu verwalten.</span><span class="sxs-lookup"><span data-stu-id="6819c-112">A request to host a session is sent, somehow, and the host peer agrees to manage collaboration activity.</span></span>  
  
- <span data-ttu-id="6819c-113">Der Host lädt Kontakte im Subnetz (einschließlich des Anforderers) zu einer Sitzung ein.</span><span class="sxs-lookup"><span data-stu-id="6819c-113">The host invites contacts on the subnet (including the requestor) to a session.</span></span>

- <span data-ttu-id="6819c-116">Alle Peers, die kollaborieren möchten, abonnieren den Hostpeer.</span><span class="sxs-lookup"><span data-stu-id="6819c-116">All peers who want to collaborate will subscribe to the host peer.</span></span>  
  
- <span data-ttu-id="6819c-117">Während die Peers ihre anfängliche Kollaborationsaktivität ausführen, kann der Hostpeer Remotepeers zu seinem Kontaktmanager hinzufügen.</span><span class="sxs-lookup"><span data-stu-id="6819c-117">While the peers are performing their initial collaboration activity, the host peer may add remote peers to its contact manager.</span></span> <span data-ttu-id="6819c-118">Er verarbeitet auch alle Antworten auf Einladungen, um zu bestimmen, wer die Einladung akzeptiert, wer sie abgelehnt, und wer nicht geantwortet hat.</span><span class="sxs-lookup"><span data-stu-id="6819c-118">It also processes all invitation responses to determine who has accepted, who has declined, and who has not answered.</span></span>  <span data-ttu-id="6819c-119">Er kann möglicherweise Einladungen an jene, die nicht geantwortet haben, widerrufen oder einige andere Aktivitäten durchführen.</span><span class="sxs-lookup"><span data-stu-id="6819c-119">It may cancel invitations to those who have not answered, or perform some other activity.</span></span>  
  
- <span data-ttu-id="6819c-120">An diesem Punkt kann der Hostpeer eine Kollaborationssitzung mit allen eingeladenen Peers starten oder eine Anwendung mit der Kollaborationsinfrastruktur registrieren.</span><span class="sxs-lookup"><span data-stu-id="6819c-120">At this point, the host peer can start a collaboration session with all invited peers, or register an application with the collaboration infrastructure.</span></span>  <span data-ttu-id="6819c-121">P2P-Anwendungen verwenden die Infrastruktur für die Peer-zu-Peer-Kollaboration und den <xref:System.Net.PeerToPeer.Collaboration>-Namespace, um die Kommunikation für Spiele, Bulletin Boards, Konferenzen und andere serverlose Anwendungen zu koordinieren.</span><span class="sxs-lookup"><span data-stu-id="6819c-121">P2P applications use the Peer-to-Peer Collaboration Infrastructure and the <xref:System.Net.PeerToPeer.Collaboration> namespace to coordinate communications for games, bulletin boards, conferencing, and other serverless presence applications.</span></span>

 <span data-ttu-id="6819c-123">In einer Active Directory-Domäne bieten Domänencontroller Kerberos-Authentifizierungsdienste an.</span><span class="sxs-lookup"><span data-stu-id="6819c-123">In an Stammspeicher des Peers wegfällt.</span><span class="sxs-lookup"><span data-stu-id="6819c-125">For Peer-to-Peer Networking, any node can act as a CA, removing the requirement of a root certificate in
