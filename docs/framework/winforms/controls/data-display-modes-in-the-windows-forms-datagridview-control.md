## <a name="virtual"></a><span data-ttu-id="72f26-129">Virtual</span><span class="sxs-lookup"><span data-stu-id="72f26-129">Virtual</span></span>  
 <span data-ttu-id="72f26-130">Mit dem virtuellen Modus können Sie eigene Datenverwaltungsvorgänge implementieren.</span><span class="sxs-lookup"><span data-stu-id="72f26-130">With virtual mode, you can implement your own data management operations.</span></span> <span data-ttu-id="72f26-131">Dies ist erforderlich, um die Werte von ungebundenen Spalten im gebundenen Modus verwalten, wenn das Steuerelement gebundenen Spalten sortiert werden.</span><span class="sxs-lookup"><span data-stu-id="72f26-131">This is necessary to maintain the values of unbound columns in bound mode when the control is sorted by bound columns.</span></span> <span data-ttu-id="72f26-132">Der Hauptverwendungszweck des virtuellen Modus befindet, ist jedoch zum Optimieren der Leistung bei der Interaktion mit großen Datenmengen.</span><span class="sxs-lookup"><span data-stu-id="72f26-132">The primary use of virtual mode, however, is to optimize performance when interacting with large amounts of data.</span></span>  
  
 <span data-ttu-id="72f26-133">Sie fügen die <xref:System.Windows.Forms.DataGridView> Steuerelement auf einen Cache, die Sie verwalten und Ihre Steuerelemente mit Code, wenn Daten mithilfe von Push übertragen und abgerufen werden.</span><span class="sxs-lookup"><span data-stu-id="72f26-133">You attach the <xref:System.Windows.Forms.DataGridView> control to a cache that you manage, and your code controls when data rows are pushed and pulled.</span></span> <span data-ttu-id="72f26-134">Um den Speicherbedarf gering zu halten, sollte der Cache ähnliche Größe auf die Anzahl der Zeilen angezeigt werden.</span><span class="sxs-lookup"><span data-stu-id="72f26-134">To keep the memory footprint small, the cache should be similar in size to the number of rows currently displayed.</span></span> <span data-ttu-id="72f26-135">Wenn der Benutzer neue Zeilen in der Ansicht einen Bildlauf durchführt, wird Ihr Code neue Daten aus dem Cache anfordert, und löscht optional alte Daten aus dem Arbeitsspeicher.</span><span class="sxs-lookup"><span data-stu-id="72f26-135">When the user scrolls new rows into view, your code requests new data from the cache and optionally flushes old data from memory.</span></span>
