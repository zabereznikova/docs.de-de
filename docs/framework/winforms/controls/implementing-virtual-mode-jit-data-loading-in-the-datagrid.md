## <a name="the-cache-class"></a><span data-ttu-id="29599-129">Die Cache-Klasse</span><span class="sxs-lookup"><span data-stu-id="29599-129">The Cache Class</span></span>  
 <span data-ttu-id="29599-130">Im folgenden Codebeispiel wird die `Cache`-Klasse definiert, die zwei Datenseiten verwaltet, die durch eine `IDataPageRetriever` Implementierung aufgefüllt werden.</span><span class="sxs-lookup"><span data-stu-id="29599-130">The following code example defines the `Cache` class, which manages two pages of data populated through an `IDataPageRetriever` implementation.</span></span> <span data-ttu-id="29599-131">Die `Cache`-Klasse definiert eine innere `DataPage` Struktur, die eine <xref:System.Data.DataTable> zum Speichern der Werte in einer einzelnen Cache Seite enthält und die Zeilen Indizes berechnet, die die obere und untere Grenze der Seite darstellen.</span><span class="sxs-lookup"><span data-stu-id="29599-131">The `Cache` class defines an inner `DataPage` structure, which contains a <xref:System.Data.DataTable> to store the values in a single cache page and which calculates the row indexes that represent the upper and lower boundaries of the page.</span></span>
