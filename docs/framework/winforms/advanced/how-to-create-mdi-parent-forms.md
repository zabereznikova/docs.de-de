 <span data-ttu-id="b3fb5-106">Die Grundlage einer MDI (Multiple Document Interface)-Anwendung ist das übergeordnete MDI-Formular.</span><span class="sxs-lookup"><span data-stu-id="b3fb5-106">The foundation of a Multiple-Document Interface (MDI) application is the MDI parent form.</span></span> <span data-ttu-id="b3fb5-107">Dies ist das Formular, das untergeordnete MDI-Fenster enthält. ### <a name="to-create-an-mdi-parent-form-at-design-time"></a><span data-ttu-id="b3fb5-109">So erstellen Sie ein übergeordnetes MDI-Formular zur Entwurfszeit</span><span class="sxs-lookup"><span data-stu-id="b3fb5-109">To create an MDI parent form at design time</span></span>  
  
1.  <span data-ttu-id="b3fb5-110">Erstellen Sie ein Windows-Anwendungsprojekt.</span><span class="sxs-lookup"><span data-stu-id="b3fb5-110">Create a Windows Application project.</span></span>  
  
2.  <span data-ttu-id="b3fb5-111">In der **Eigenschaften** legen die <xref:System.Windows.Forms.Form.IsMdiContainer%2A> Eigenschaft **"true"**.</span><span class="sxs-lookup"><span data-stu-id="b3fb5-111">In the **Properties** window, set the <xref:System.Windows.Forms.Form.IsMdiContainer%2A> property to **true**.</span></span>  
  
     <span data-ttu-id="b3fb5-112">Dies kennzeichnet das Formular als MDI-Container für untergeordnete Fenster.</span><span class="sxs-lookup"><span data-stu-id="b3fb5-112">This designates the form as an MDI container for child windows.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="b3fb5-113">Beim Festlegen der Eigenschaften im Fenster **Eigenschaften** können Sie bei Bedarf auch die `WindowState`-Eigenschaft auf **Maximiert** festlegen, da sich untergeordnete MDI-Fenster am leichtesten bearbeiten lassen, wenn das übergeordnete Formular maximiert ist.</span><span class="sxs-lookup"><span data-stu-id="b3fb5-113">While setting properties in the **Properties** window, you can also set the `WindowState` property to **Maximized**, if you like, as it is easiest to manipulate MDI child windows when the parent form is maximized.</span></span> <span data-ttu-id="b3fb5-114">Achten Sie außerdem darauf, dass durch den Rand des übergeordneten MDI-Formulars die Systemfarbe (die in der Windows-Systemsteuerung festgelegt wird) und nicht die Hintergrundfarbe, die Sie mithilfe der <xref:System.Windows.Forms.Control.BackColor%2A?displayProperty=nameWithType> -Eigenschaft festlegen, übernommen wird.</span><span class="sxs-lookup"><span data-stu-id="b3fb5-114">Additionally, be aware that the edge of the MDI parent form will pick up the system color (set in the Windows System Control Panel), rather than the back color you set using the <xref:System.Windows.Forms.Control.BackColor%2A?displayProperty=nameWithType> property.</span></span>  
  
3.  <span data-ttu-id="b3fb5-115">Ziehen Sie aus der **Toolbox** ein **MenuStrip**-Steuerelement in das Formular.</span><span class="sxs-lookup"><span data-stu-id="b3fb5-115">From the **Toolbox**, drag a **MenuStrip** control to the form.</span></span> <span data-ttu-id="b3fb5-116">Erstellen Sie ein Menüelement auf der obersten Ebene, wobei die **Text**-Eigenschaft auf **&Datei** festgelegt ist und die Untermenüs **&Neu** und **&Schließen** heißen.</span><span class="sxs-lookup"><span data-stu-id="b3fb5-116">Create a top-level menu item with the **Text** property set to **&File** with submenu items called **&New** and **&Close**.</span></span> <span data-ttu-id="b3fb5-117">Erstellen Sie darüber hinaus ein Menüelement auf der obersten Ebene mit dem Namen **&Fenster**.</span><span class="sxs-lookup"><span data-stu-id="b3fb5-117">Also create a top-level menu item called **&Window**.</span></span>  
  
     <span data-ttu-id="b3fb5-118">Durch das erste Menü werden Menüelemente zur Laufzeit erstellt und ausgeblendet, während über das zweite Menü die offenen untergeordneten MDI-Fenster nachverfolgt werden.</span><span class="sxs-lookup"><span data-stu-id="b3fb5-118">The first menu will create and hide menu items at run time, and the second menu will keep track of the open MDI child windows.</span></span> <span data-ttu-id="b3fb5-119">Jetzt haben Sie ein übergeordnetes MDI-Fenster erstellt.</span><span class="sxs-lookup"><span data-stu-id="b3fb5-119">At this point, you have created an MDI parent window.</span></span>  
  
4.  <span data-ttu-id="b3fb5-120">Drücken Sie **F5**, um die Anwendung auszuführen.</span><span class="sxs-lookup"><span data-stu-id="b3fb5-120">Press **F5** to run the application.</span></span> <span data-ttu-id="b3fb5-121">Informationen zum Erstellen von untergeordneten MDI-Fenstern, die innerhalb der übergeordneten MDI-Formulars verwendet werden, finden Sie unter [Gewusst wie: Erstellen von untergeordneten MDI-Formularen](../../../../docs/framework/winforms/advanced/how-to-create-mdi-child-forms.md).</span><span class="sxs-lookup"><span data-stu-id="b3fb5-121">For information about creating MDI child windows that operate within the MDI parent form, see [How to: Create MDI Child Forms](../../../../docs/framework/winforms/advanced/how-to-create-mdi-child-forms.md).</span></span>
