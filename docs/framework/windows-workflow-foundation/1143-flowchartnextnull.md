# <a name="1143---flowchartnextnull"></a>1143 - FlowchartNextNull

## <a name="properties"></a>Eigenschaften  
  
|||  
|-|-|  
|id|1143|  
|Keywords|WFActivities|  
|Ebene|Information|  
|Kanal|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>BESCHREIBUNG  

 Gibt an, dass der nächste Knoten in einem Flussdiagramm NULL ist. Die Ausführung des Flussdiagramms wird beendet.  
  
## <a name="message"></a>`Message`  

 Flussdiagramm '%1'/FlowStep - Nächster Knoten ist NULL. Die Ausführung des Flussdiagramms wird beendet.  
  
## <a name="details"></a>Details  
  
|Datenelementname|Datenelementtyp|BESCHREIBUNG|  
|--------------------|--------------------|-----------------|  
|FlowChart|xs:string|Der Anzeigename des FlowChart.|  
|AppDomain|xs:string|Die von AppDomain.CurrentDomain.FriendlyName zurückgegebene Zeichenfolge.|
