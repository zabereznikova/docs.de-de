## <a name="implementation-guidelines-and-conventions"></a><span data-ttu-id="95f35-109">Implementierungsrichtlinien und -konventionen</span><span class="sxs-lookup"><span data-stu-id="95f35-109">Implementation Guidelines and Conventions</span></span>
