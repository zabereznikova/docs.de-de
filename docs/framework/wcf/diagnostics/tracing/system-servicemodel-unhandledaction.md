# <a name="systemservicemodelunhandledaction"></a>System.ServiceModel.UnhandledAction

System.ServiceModel.UnhandledAction  
  
## <a name="description"></a>BESCHREIBUNG  

 Es ist kein Dienst zum Verarbeiten dieser Aktion vorhanden.  
  
 Alle Ablaufverfolgungen für Leistungsindikatoren können zum Debuggen von Problemen im Zusammenhang mit Leistungsindikatoren verwendet werden.
