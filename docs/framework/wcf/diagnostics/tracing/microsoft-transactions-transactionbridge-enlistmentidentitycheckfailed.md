# <a name="microsofttransactionstransactionbridgeenlistmentidentitycheckfailed"></a>Microsoft.Transactions.TransactionBridge.EnlistmentIdentityCheckFailed
Die Identität des Absenders der eingehenden Nachricht stimmt nicht mit der Identität überein, die vom Endpunktverweis der Eintragung abgeleitet wurde.  
  
## <a name="description"></a>Beschreibung  
 Wird nachverfolgt, wenn die Nachrichtensicherheit nicht validiert werden konnte.  
  
## <a name="troubleshooting"></a>Problembehandlung  
 Vergewissern Sie sich, dass die Transaktionssicherheit für alle an der Transaktion beteiligten Computer ordnungsgemäß konfiguriert wurde.
