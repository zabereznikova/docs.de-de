# <a name="microsofttransactionstransactionbridgeregistercoordinator"></a>Microsoft.Transactions.TransactionBridge.RegisterCoordinator

Der Protokolldienst für WS-Atomic-Transaktion wurde beim Koordinator für ein Steuerprotokoll registriert.  
  
## <a name="description"></a>BESCHREIBUNG  

 Wird nachverfolgt, wenn der lokale Transaktions-Manager beim übergeordneten Transaktions-Manager registriert wurde.
