# SslNoAccessiblePrivateKey
Id: 155  
  
 Schweregrad: Fehler  
  
 Kategorie: TransactionBridge  
  
## Beschreibung  
 Dieses Ereignis zeigt an, dass ein Identitätszertifikat mit dem bestimmten Betreffnamen und Fingerabdruck keinen zugreifbaren privaten Schlüssel besitzt.Das Ereignis listet den Prozessnamen und die Prozess\-ID auf.
