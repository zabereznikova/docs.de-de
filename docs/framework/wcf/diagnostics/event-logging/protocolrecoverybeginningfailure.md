# <a name="protocolrecoverybeginningfailure"></a>ProtocolRecoveryBeginningFailure
ID: 144  
  
 Schweregrad: Fehler  
  
 Kategorie: TransactionBridge  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis gibt an, dass das MSDTC WS-Atomic Transaction-Protokoll beim Start der Wiederherstellung fehlgeschlagen ist. Aus diesem Grund ist die WS-Atomic-Transaktionsfunktionalität deaktiviert. Das Ereignis führt die Protokoll-ID, den Protokollnamen, die Ausnahme, den Prozessnamen und Prozess-ID auf.
