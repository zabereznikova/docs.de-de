# <a name="protocolinitializationfailure"></a>ProtocolInitializationFailure

ID: 142  
  
 Schweregrad: Fehler  
  
 Kategorie: TransactionBridge  
  
## <a name="description"></a>BESCHREIBUNG  

 Dieses Ereignis gibt an, dass der WS-Atomic-Transaktionsprotokolldienst nicht initialisiert werden konnte. Aus diesem Grund ist die WS-Atomic-Transaktionsfunktionalität deaktiviert. Das Ereignis führt die Protokoll-ID, den Protokollnamen, die Ausnahme, den Prozessnamen und Prozess-ID auf. .
