# <a name="webhostfailedtoprocessrequest"></a>WebHostFailedToProcessRequest
ID: 111  
  
 Schweregrad: Fehler  
  
 Kategorie: ServiceModel  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis gibt an, dass WebHost eine Anforderung nicht verarbeitet hat. Das Ereignis listet die Absenderinformationen, die Ausnahme, den Prozessnamen und die Prozess-ID auf.
