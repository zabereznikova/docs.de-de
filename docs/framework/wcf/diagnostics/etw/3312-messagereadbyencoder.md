## <a name="description"></a>Beschreibung  
 Dieses Ereignis wird ausgegeben, wenn die Nachricht vom Encorder gelesen wurde.

 Vom Encoder wurde eine Nachricht mit einer Größe von '%1' Bytes gelesen.
