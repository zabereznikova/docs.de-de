# <a name="4016---messagequeueduplicatedsocketerror"></a>4016 - MessageQueueDuplicatedSocketError

## <a name="properties"></a>Eigenschaften  
  
|||  
|-|-|  
|id|4016|  
|Keywords|ActivationServices|  
|Ebene|Fehler|  
|Kanal|Microsoft-Windows-Application Server-Applications/Analytic|  
  
## <a name="description"></a>BESCHREIBUNG  

 Dieses Ereignis wird ausgegeben, wenn die Nachrichtenwarteschlange den Socket nicht duplizieren konnte.  
  
## <a name="message"></a>`Message`  

 Fehler beim Duplizieren des Sockets durch die Nachrichtenwarteschlange.
