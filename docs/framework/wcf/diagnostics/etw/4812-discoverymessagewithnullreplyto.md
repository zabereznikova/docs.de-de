# <a name="4812---discoverymessagewithnullreplyto"></a>4812 - DiscoveryMessageWithNullReplyTo

## <a name="properties"></a>Eigenschaften  
  
|||  
|-|-|  
|id|4812|  
|Keywords|Ermittlung|  
|Ebene|Warnung|  
|Kanal|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>BESCHREIBUNG  

 Dieses Ereignis wird ausgegeben, wenn die Ermittlungsmeldung durch den DiscoveryClient gelöscht wurde, da sie keine ReplyTo-Adresse enthielt.  
  
## <a name="message"></a>`Message`  

 Eine Erkennungsanforderungsmeldung mit messageId='%1' wurde gelöscht, da sie keine ReplyTo-Adresse enthielt.
