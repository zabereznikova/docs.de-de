|||  
|-|-|  
|id|4020|  
|Keywords|ActivationServices|  
|Ebene|Ausführlich|  
|Kanal|Microsoft-Windows-Application Server-Applications/Analytic|  
  
## <a name="description"></a>BESCHREIBUNG  

 Dieses Ereignis wird ausgegeben, wenn der TCP-Transportlistener mit dem Lauschen beginnt.  
  
## <a name="message"></a>`Message`  

 Der TCP-Transportlistener beginnt mit dem Lauschen für den URI "%1".
