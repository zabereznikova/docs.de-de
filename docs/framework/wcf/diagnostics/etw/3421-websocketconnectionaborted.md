|||  
|-|-|

|Keywords|HTTP|  
|Ebene|Fehler|

## <a name="description"></a>BESCHREIBUNG  

 Dieses Ereignis wird ausgegeben, wenn die WebSocket-Verbindung abgebrochen wird.  
  
## <a name="message"></a>`Message`  

 Die Verbindung mit WebSocketId: %1 wurde abgebrochen.
