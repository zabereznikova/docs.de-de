# <a name="3820---routingservicecreatingtransaction"></a>3820 - RoutingServiceCreatingTransaction

## <a name="properties"></a>Eigenschaften  
  
|||  
|-|-|  
|id|3820|  
|Keywords|RoutingServices|  
|Ebene|Information|  
|Kanal|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>BESCHREIBUNG  

 Dieses Ereignis wird ausgegeben, wenn der Routingdienst eine neue Transaktion zum Verarbeiten einer Nachricht erstellt hat.  
  
## <a name="message"></a>`Message`  

 Der Routingdienst hat eine neue Transaktion mit der ID '%1' zur Verarbeitung von Nachrichten erstellt.
