|||  
|-|-|  
|id|3321|  
|Keywords|Kanal|  
|Ebene|Ausführlich|

## <a name="description"></a>BESCHREIBUNG  

 Dieses Ereignis wird ausgegeben, wenn der Pool über keine verfügbaren Verbindungen verfügt.  
  
## <a name="message"></a>`Message`  

 Der Pool für "%1" besitzt keine verfügbaren Verbindungen und %2 ausgelastete Verbindungen.
