|||  
|-|-|

|Schlüsselwörter|HTTP|  
|Ebene|Ausführlich|

## <a name="description"></a>Beschreibung  
 Dieses Ereignis wird ausgegeben, wenn die WebSocket-Verbindung geschlossen wird.
