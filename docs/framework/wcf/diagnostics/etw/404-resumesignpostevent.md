|||  
|-|-|  
|id|404|  
|Keywords|Problembehandlung|  
|Ebene|Information|

## <a name="description"></a>BESCHREIBUNG  

 Dieses Ereignis kennzeichnet die Wiederaufnahme einer End-to-End-Aktivität. Enthält den Namen der Aktivität.  
  
## <a name="message"></a>`Message`
