# <a name="service-calls-duration"></a>Dienst: Dauer von Aufrufen
Indikatorname: Dauer von Aufrufen  
  
## <a name="description"></a>Beschreibung  
 Die durchschnittliche Dauer der Aufrufe dieses Dienstes. Die durchschnittliche Dauer wird anhand dieser Formel berechnet: (N1-N0)/(D1-D0).
