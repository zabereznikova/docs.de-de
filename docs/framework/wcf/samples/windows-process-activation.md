# <a name="windows-process-activation"></a>Windows-Prozessaktivierung
Dieser Abschnitt enthält Beispiele, in denen die Dienstaktivierung über Netzwerkprotokolle veranschaulicht wird.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 [NamedPipe-Aktivierung](../../../../docs/framework/wcf/samples/namedpipe-activation.md)  
 Veranschaulicht das Hosten eines Diensts, der den Windows-Prozessaktivierungsdienst (WAS, Windows Process Activation Service) zum Aktivieren eines Diensts verwendet, der über benannte Pipes kommuniziert.  
  
 [TCP-Aktivierung](../../../../docs/framework/wcf/samples/tcp-activation.md)  
 Veranschaulicht das Hosten eines Diensts, der den Windows-Prozessaktivierungsdienst zum Aktivieren eines Diensts verwendet, der über das net.tcp-Protokoll kommuniziert.

 [MSMQ-Aktivierung](../../../../docs/framework/wcf/samples/msmq-activation.md)  
 Veranschaulicht das Hosten eines Diensts, das Windows Process Activation Services (WAS) verwendet wird, zum Aktivieren eines Diensts, das über das net.msmq-Protokoll kommuniziert.
