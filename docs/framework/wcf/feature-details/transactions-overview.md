<span data-ttu-id="13ced-103">Transaktionen stellen eine Möglichkeit dar, mehrere Aktionen oder Vorgänge in eine einzelne unteilbare Ausführungseinheit zu gruppieren.</span><span class="sxs-lookup"><span data-stu-id="13ced-103">Transactions provide a way to group a set of actions or operations into a single indivisible unit of execution.</span></span> <span data-ttu-id="13ced-104">Eine Transaktion ist eine Auflistung von Vorgängen mit den folgenden Eigenschaften:</span><span class="sxs-lookup"><span data-stu-id="13ced-104">A transaction is a collection of operations with the following properties:</span></span>  
  
-   <span data-ttu-id="13ced-105">Unteilbarkeit.</span><span class="sxs-lookup"><span data-stu-id="13ced-105">Atomicity.</span></span> <span data-ttu-id="13ced-106">Dadurch wird sichergestellt, dass entweder alle unter einer bestimmten Transaktion ausgeführten Updates übernommen und dauerhaft gemacht werden oder abgebrochen und in ihren vorherigen Zustand zurückversetzt werden.</span><span class="sxs-lookup"><span data-stu-id="13ced-106">This ensures that either all of the updates completed under a specific transaction are committed and made durable or they are all aborted and rolled back to their previous state.</span></span>  
  
-   <span data-ttu-id="13ced-107">Konsistenz.</span><span class="sxs-lookup"><span data-stu-id="13ced-107">Consistency.</span></span> <span data-ttu-id="13ced-108">Dadurch wird sichergestellt, dass die unter einer bestimmten Transaktion vorgenommenen Änderungen eine Transformation von einem konsistenten Zustand zu einem anderen darstellen.</span><span class="sxs-lookup"><span data-stu-id="13ced-108">This guarantees that the changes made under a transaction represent a transformation from one consistent state to another.</span></span> <span data-ttu-id="13ced-109">Beispielsweise ändert eine Transaktion, bei der Geld von einem Girokonto auf ein Sparkonto überwiesen wird, die Geldmenge auf dem Bankkonto insgesamt nicht.</span><span class="sxs-lookup"><span data-stu-id="13ced-109">For example, a transaction that transfers money from a checking account to a savings account does not change the amount of money in the overall bank account.</span></span>  
  
-   <span data-ttu-id="13ced-110">Isolation</span><span class="sxs-lookup"><span data-stu-id="13ced-110">Isolation.</span></span> <span data-ttu-id="13ced-111">Dadurch wird verhindert, dass eine Transaktion ungespeicherte Änderungen beobachtet, die zu anderen gleichzeitigen Transaktionen gehören.</span><span class="sxs-lookup"><span data-stu-id="13ced-111">This prevents a transaction from observing uncommitted changes belonging to other concurrent transactions.</span></span> <span data-ttu-id="13ced-112">Isolation bietet eine Abstraktion von Parallelität und stellt gleichzeitig sicher, dass eine Transaktion keine unerwarteten Auswirkungen auf die Ausführung einer anderen Transaktion haben kann.</span><span class="sxs-lookup"><span data-stu-id="13ced-112">Isolation provides an abstraction of concurrency while ensuring one transaction cannot have an unexpected impact on the execution of another transaction.</span></span>  
  
-   <span data-ttu-id="13ced-113">Dauerhaftigkeit.</span><span class="sxs-lookup"><span data-stu-id="13ced-113">Durability.</span></span> <span data-ttu-id="13ced-114">Dies bedeutet, dass Updates verwalteter Ressourcen (z. B. eines Datensatzes in einer Datenbank) nach der Ausführung bei Fehlern dauerhaft sind.</span><span class="sxs-lookup"><span data-stu-id="13ced-114">This means that once committed, updates to managed resources (such as a database record) will be persistent in the face of failures.</span></span>

 <span data-ttu-id="13ced-116">WCF implementiert die Unterstützung für das WS-AtomicTransaction (WS-AT)-Protokoll, das WCF-Anwendungen, Transaktionen zu interoperablen Anwendungen ausführen können, z.  <span data-ttu-id="13ced-118">Sie können eine Anwendungskonfigurationsdatei verwenden, um Bindungen für das Aktivieren bzw.  <span data-ttu-id="13ced-121">Mithilfe von Transaktionsattributen im <xref:System.ServiceModel>-Namespace können Sie die folgenden Aktionen ausführen:</span><span class="sxs-lookup"><span data-stu-id="13ced-121">Transaction attributes in the <xref:System.ServiceModel> namespace allow you to do the following:</span></span>
