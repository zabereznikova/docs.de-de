TCP://localhost".</span><span class="sxs-lookup"><span data-stu-id="9cebd-114">The base URI prefix is constructed from the site binding and is used for all the applications under the site, for example, "net.msmq://localhost", "msmq.formatname://localhost", or "net.tcp://localhost".</span></span> <span data-ttu-id="9cebd-115">Anwendungs Adressen werden dann durch die Verwendung von anwendungsspezifischen Pfad Fragmenten (z. b.
