2. <span data-ttu-id="f4424-117">Konfigurieren Sie Ihren Ablaufverfolgungsschalter.</span><span class="sxs-lookup"><span data-stu-id="f4424-117">Configure your trace switch.</span></span> <span data-ttu-id="f4424-118">Weitere Informationen finden Sie unter [Konfigurieren von Ablauf Verfolgungs Schaltern](#configure).</span><span class="sxs-lookup"><span data-stu-id="f4424-118">For more information, see [Configuring trace switches](#configure).</span></span>

    System.Diagnostics.BooleanSwitch dataSwitch =
       new System.Diagnostics.BooleanSwitch("Data", "DataAccess module");

       new System.Diagnostics.TraceSwitch("General",
       "Entire application");  
    ```

 <span data-ttu-id="f4424-125">Ihre Schalter werden über die .config-Datei konfiguriert.</span><span class="sxs-lookup"><span data-stu-id="f4424-125">Your switches are configured using the .config file.</span></span> <span data-ttu-id="f4424-126">Für eine Webanwendung ist dies die "Web.config"-Datei, die dem Projekt zugeordnet ist.</span><span class="sxs-lookup"><span data-stu-id="f4424-126">For a Web application, this is the Web.config file associated with the project.</span></span> <span data-ttu-id="f4424-127">In einer Windows-Anwendung hat diese Datei den Namen (Anwendungsname) .exe.config. 1. <span data-ttu-id="f4424-147">Um Ablaufverfolgungsschalter zu verwenden, müssen Sie diese zuerst erstellen und sie, wie im Abschnitt [Erstellen und Initialisieren eines Ablaufverfolgungsschalters](#create) beschrieben, in Ihrem Code platzieren.</span><span class="sxs-lookup"><span data-stu-id="f4424-147">In order to use trace switches, you must first create them and place them in your code as described in the section [Creating and initializing a trace switch](#create).</span></span>  
  
2. <span data-ttu-id="f4424-148">Enthält Ihr Projekt keine Konfigurationsdatei („app.config“ oder „Web.config“), dann wählen Sie im Menü **Projekt** den Befehl **Neues Element hinzufügen** aus.</span><span class="sxs-lookup"><span data-stu-id="f4424-148">If your project does not contain a configuration file (app.config or Web.config), then from the **Project** menu, select **Add New Item**.</span></span>

         <span data-ttu-id="f4424-150">Die Anwendungskonfigurationsdatei wird erstellt und geöffnet.</span><span class="sxs-lookup"><span data-stu-id="f4424-150">The application configuration file is created and opened.</span></span> <span data-ttu-id="f4424-151">Diese Datei ist ein XML-Dokument mit dem Stammelement `<configuration>.`</span><span class="sxs-lookup"><span data-stu-id="f4424-151">This is an XML document whose root element is `<configuration>.`</span></span>  
  
    - <span data-ttu-id="f4424-152">**Visual C#:** Klicken Sie im Dialogfeld **Neues Element hinzufügen** auf **XML-Datei**.</span><span class="sxs-lookup"><span data-stu-id="f4424-152">**Visual C#:** In the **Add New Item** dialog box, choose **XML File**.</span></span> <span data-ttu-id="f4424-153">Benennen Sie diese Datei **app.config**.         <configuration>  
        </configuration>  
        ```  
  
         <span data-ttu-id="f4424-154">Nachdem das Projekt kompiliert ist, wird die Datei „app.config“ in den Projektausgabeordner kopiert und in „*Anwendungsname*.exe.config“ umbenannt.</span><span class="sxs-lookup"><span data-stu-id="f4424-154">When your project is compiled, the app.config file is copied to the project output folder and is renamed *applicationname*.exe.config.</span></span>

     <span data-ttu-id="f4424-157">In dieser Konfiguration sind beide Schalter deaktiviert.</span><span class="sxs-lookup"><span data-stu-id="f4424-157">In this configuration, both switches are off.</span></span>

5. <span data-ttu-id="f4424-159">Wenn Sie einen **TraceSwitch**, wie `TraceLevelSwitch` im vorhergehenden Beispiel, aktivieren möchten, ändern Sie den **Wert** in die entsprechende Ebeneneinstellung (1 bis 4).</span><span class="sxs-lookup"><span data-stu-id="f4424-159">If you need to turn on a **TraceSwitch**, such as `TraceLevelSwitch` shown in the previous example, change the **Value** to the appropriate level setting (1 to 4).</span></span>  
  
6. <span data-ttu-id="f4424-160">Fügen Sie der .config-Datei Kommentare hinzu , damit der Benutzer genau weiß, welche Werte geändert werden müssen, damit die Schalter geeignet konfiguriert sind.</span><span class="sxs-lookup"><span data-stu-id="f4424-160">Add comments to the .config file so the end user has a clear understanding of what values to change to configure the switches appropriately.</span></span>
