 <span data-ttu-id="393a6-111">Sie können keine Vererbungshierarchien erstellen, in denen ein abgeleiteter Typ von mehr als einem Typ erbt.</span><span class="sxs-lookup"><span data-stu-id="393a6-111">You cannot build inheritance hierarchies in which a derived type inherits from more than one type.</span></span> <span data-ttu-id="393a6-112">In einem konzeptionellen Modell mit einem `Book`-Entitätstyp könnten Sie z. B. die abgeleiteten Typen `FictionBook` und `NonFictionBook` definieren, die jeweils von `Book` erben.</span><span class="sxs-lookup"><span data-stu-id="393a6-112">For example, in a conceptual model with a `Book` entity type, you could define derived types `FictionBook` and `NonFictionBook` that each inherit from `Book`.</span></span> <span data-ttu-id="393a6-113">Sie könnten dann jedoch keinen Typ definieren, der vom Typ `FictionBook` und vom Typ `NonFictionBook` erbt.</span><span class="sxs-lookup"><span data-stu-id="393a6-113">However, you could not then define a type that inherits from both the `FictionBook` and `NonFictionBook` types.</span></span>
