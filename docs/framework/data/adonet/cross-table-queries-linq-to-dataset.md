<span data-ttu-id="a4dfb-103">Zusätzlich zum Abfragen einer einzelnen Tabelle können Sie in [!INCLUDE[linq_dataset](../../../../includes/linq-dataset-md.md)] auch Abfragen für mehrere Tabellen data-stu-id="a4dfb-104">This is done by using a *join*.</span></span> <span data-ttu-id="a4dfb-105">Bei einem Join werden Objekte in einer Datenquelle mit Objekten in einer anderen Datenquelle, die über ein gemeinsames Attribut (wie Produkt oder Kontakt-ID) verfügen, miteinander verknüpft.</span><span class="sxs-lookup"><span data-stu-id="a4dfb-105">A join is the association of objects in one data-stu-id="a4dfb-106">In object-oriented programming, relationships between objects are relatively easy to navigate because each object has a member that references another object.</span></span> <span data-ttu-id="a4dfb-107">In externen Datenbanktabellen ist die Navigation zwischen den Beziehungen etwas komplizierter.</span><span class="sxs-lookup"><span data-stu-id="a4dfb-107">In external database tables, however, navigating relationships is not as straightforward.</span></span> <span data-ttu-id="a4dfb-108">Datenbanktabellen enthalten keine integrierten Beziehungen.</span><span class="sxs-lookup"><span data-stu-id="a4dfb-108">Database tables do not contain built-in relationships.</span></span> <span data-ttu-id="a4dfb-109">In diesen Fällen kann die JOIN-Operation verwendet  <span data-ttu-id="a4dfb-111">Das [!INCLUDE[vbteclinqext](../../../../includes/vbteclinqext-md.md)]-Framework stellt zwei Joinoperatoren bereit: <xref:System.Linq.Enumerable.Join%2A> und <xref:System.Linq.Enumerable.GroupJoin%2A>.</span><span class="sxs-lookup"><span data-stu-id="a4dfb-111">The [!INCLUDE[vbteclinqext](../../../../includes/vbteclinqext-md.md)] framework provides two join operators, <xref:System.Linq.Enumerable.Join%2A> and <xref:System.Linq.Enumerable.GroupJoin%2A>.</span></span> <span data-ttu-id="a4dfb-112">Diese Operatoren führen *gleichheitsjoins*: d. h.  <span data-ttu-id="a4dfb-114">In der Terminologie relationaler Datenbanken implementiert <xref:System.Linq.Enumerable.Join%2A> einen inneren Join.</span><span class="sxs-lookup"><span data-stu-id="a4dfb-114">In relational database terms, <xref:System.Linq.Enumerable.Join%2A> implements an inner join.</span></span> <span data-ttu-id="a4dfb-115">Eine innerer Join ist eine Joinart, bei der nur diejenigen Objekte zurückgegeben werden, für die es im anderen Dataset ein passendes Gegenüber gibt.</span><span class="sxs-lookup"><span data-stu-id="a4dfb-115">An inner join is a type of join in which only those objects that have a match in the opposite data set are returned.</span></span>  
  
 <span data-ttu-id="a4dfb-116">Die <xref:System.Linq.Enumerable.GroupJoin%2A> Operatoren haben keine direkte Entsprechung in der Terminologie für relationale Datenbanken, die sie implementieren eine übergeordnete Menge innerer und linker äußerer Verknüpfungen.</span><span class="sxs-lookup"><span data-stu-id="a4dfb-116">The <xref:System.Linq.Enumerable.GroupJoin%2A> operators have no direct equivalent in relational database terms; they implement a superset of inner joins and left outer joins.</span></span> <span data-ttu-id="a4dfb-117">Ein linker äußerer Join ist ein Join, der jedes Element der ersten (linken) Auflistung zurückgibt, selbst wenn die zweite Auflistung keine zugehörigen Elemente enthält.</span><span class="sxs-lookup"><span data-stu-id="a4dfb-117">A left outer join is a join that returns each element of the first (left) collection, even if it has no correlated elements in the second collection.</span></span>  
  
 <span data-ttu-id="a4dfb-118">Weitere Informationen zu Joins finden Sie unter [Verknüpfungsoperationen](https://msdn.microsoft.com/library/442d176d-028c-4beb-8d22-407d4ef89107).</span><span class="sxs-lookup"><span data-stu-id="a4dfb-118">For more information about joins, see [Join Operations](https://msdn.microsoft.com/library/442d176d-028c-4beb-8d22-407d4ef89107).</span></span>
