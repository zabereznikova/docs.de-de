# <a name="closeenum-method"></a>CloseEnum-Methode
Closes the indicated enumeration and frees associated resources.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT CloseEnum(  
    HALINKENUM hEnum  
) PURE;  
```  
  
## <a name="parameters"></a>Parameter  
 `hEnum`  
 Handle of enumeration to be closed.
