# <a name="icordebugcodeisil-method"></a>ICorDebugCode::IsIL-Methode
Ruft einen Wert, der angibt, ob "ICorDebugCode" darstellt, der in Microsoft intermediate Language (MSIL) kompiliert wurde.  
  
## <a name="syntax"></a>Syntax  
  
```  
HRESULT IsIL (  
    [out] BOOL       *pbIL  
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `pbIL`  
 [out] `true` Wenn diese `ICorDebugCode` Code darstellt, der in MSIL kompilierte; andernfalls wurde `false`.
