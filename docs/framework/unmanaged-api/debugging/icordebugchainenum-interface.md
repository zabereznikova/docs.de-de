# <a name="icordebugchainenum-interface"></a>ICorDebugChainEnum-Schnittstelle

Implementiert ICorDebugEnum-Methoden und listet ICorDebugChain-Arrays auf.  
  
## <a name="methods"></a>Methoden  
  
|Methode|BESCHREIBUNG|  
|------------|-----------------|  
|[Next-Methode](icordebugchainenum-next-method.md)|Ruft die angegebene Anzahl von- `ICorDebugChain` Instanzen ab der aktuellen Position aus der-Enumeration ab.|  
  
## <a name="remarks"></a>Hinweise  
  
> [!NOTE]
> Diese Schnittstelle kann weder computerübergreifend noch prozessübergreifend remote aufgerufen werden.
