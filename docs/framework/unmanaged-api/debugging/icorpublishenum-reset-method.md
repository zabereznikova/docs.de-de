# <a name="icorpublishenumreset-method"></a>ICorPublishEnum::Reset-Methode

## <a name="syntax"></a>Syntax  
  
```  
HRESULT Reset ();  
```  
  
#### <a name="parameters"></a>Parameter  
 `Reset`nimmt keine Parameter an.
