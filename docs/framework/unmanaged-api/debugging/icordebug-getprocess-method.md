# <a name="icordebuggetprocess-method"></a>ICorDebug::GetProcess-Methode
Ruft einen Zeiger auf die Instanz "ICorDebugProcess" für den angegebenen Prozess.  
  
## <a name="syntax"></a>Syntax  
  
```  
HRESULT GetProcess (  
    [in] DWORD               dwProcessId,  
    [out] ICorDebugProcess   **ppProcess  
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `dwProcessId`  
 [in] Die ID des Prozesses.  
  
 `ppProcess`  
 [out] Ein Zeiger auf die Adresse von einem `ICorDebugProcess` Instanz für den angegebenen Prozess.
