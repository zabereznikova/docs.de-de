# <a name="icordebugappdomainenum-interface"></a>ICorDebugAppDomainEnum-Schnittstelle

Stellt die- `Next` Methode bereit, die eine angegebene Anzahl von Werten zurückgibt, `ICorDebugAppDomainEnum` beginnend an der nächsten Position in der-Enumeration. Bei dieser Schnittstelle handelt es sich um eine Unterklasse von "ICorDebugEnum".  
  
## <a name="methods"></a>Methoden  
  
|Methode|BESCHREIBUNG|  
|------------|-----------------|  
|[Next-Methode](icordebugappdomainenum-next-method.md)|Ruft die angegebene Anzahl von Anwendungs Domänen ab der aktuellen Cursorposition aus der Auflistung ab.|  
  
## <a name="remarks"></a>Hinweise  
  
> [!NOTE]
> Diese Schnittstelle kann weder computerübergreifend noch prozessübergreifend remote aufgerufen werden.
