# <a name="icordebugmodulegetassembly-method"></a>ICorDebugModule::GetAssembly-Methode
Ruft die enthaltende Assembly für dieses Modul ab.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT GetAssembly(  
    [out] ICorDebugAssembly **ppAssembly  
);  
```  
  
## <a name="parameters"></a>Parameter  
 `ppAssembly`  
 vorgenommen Ein Zeiger auf ein ICorDebug-Objekt, das die Assembly darstellt, die dieses Modul enthält.
