# <a name="icorprofilercallback2handlecreated-method"></a>ICorProfilerCallback2::HandleCreated-Methode
Benachrichtigt den Profiler, dass eine Garbage Collection-Handle erstellt wurde.  
  
## <a name="syntax"></a>Syntax  
  
```  
HRESULT HandleCreated(  
    [in] GCHandleID handleId,  
    [in] ObjectID initialObjectId);  
```  
  
#### <a name="parameters"></a>Parameter  
 `handleId`  
 [in] Die ID des Handles für die Garbagecollection.  
  
 `initialObjectId`  
 [in] Die ID des Objekts, für die Garbage Collection-Handle erstellt wurde.
