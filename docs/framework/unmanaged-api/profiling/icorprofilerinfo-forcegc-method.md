# <a name="icorprofilerinfoforcegc-method"></a>ICorProfilerInfo::ForceGC-Methode

Erzwingt, dass Garbage Collection im Common Language Runtime (CLR) auftritt.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT ForceGC();  
```
