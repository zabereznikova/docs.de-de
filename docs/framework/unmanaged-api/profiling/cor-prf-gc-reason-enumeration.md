# <a name="corprfgcreason-enumeration"></a>COR_PRF_GC_REASON-Enumeration

```

    COR_PRF_GC_INDUCED = 1,  
    COR_PRF_GC_OTHER = 0  
} COR_PRF_GC_REASON;
```  
