# <a name="icorprofilerinfo10resumeruntime-method"></a>ICorProfilerInfo10:: resumeruntime-Methode
  
Setzt die Laufzeit fort, ohne eine GC auszuführen.   
  
## <a name="syntax"></a>Syntax  
  
```cpp
HRESULT ResumeRuntime();
```
