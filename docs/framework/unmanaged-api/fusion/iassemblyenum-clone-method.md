# <a name="iassemblyenumclone-method"></a>IAssemblyEnum::Clone-Methode

Erstellt eine flache Kopie dieses [IAssemblyEnum](iassemblyenum-interface.md) -Objekts.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT Clone (  
    [out] IAssemblyEnum   **ppEnum  
);  
```  
  
## <a name="parameters"></a>Parameter  

 `ppEnum`  
 vorgenommen Ein Zeiger auf die Kopie.
