# <a name="iassemblycacheitemabortitem-method"></a>IAssemblyCacheItem::AbortItem-Methode
Ermöglicht es, die Assembly im globalen Assemblycache Bereinigungsvorgänge durchzuführen, bevor er freigegeben wird.  
  
## <a name="syntax"></a>Syntax  
  
```  
HRESULT AbortItem ();  
```
