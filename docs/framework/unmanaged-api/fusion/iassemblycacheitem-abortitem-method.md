# <a name="iassemblycacheitemabortitem-method"></a>IAssemblyCacheItem::AbortItem-Methode

Ermöglicht es der Assembly im globalen Assemblycache, Cleanupvorgänge auszuführen, bevor Sie freigegeben wird.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT AbortItem ();  
```
