# <a name="imetadataimportgetnestedclassprops-method"></a>IMetaDataImport::GetNestedClassProps-Methode

Ruft das TypeDef-Token für das <xref:System.Type> übergeordnete Element des angegebenen eingefügten Typs ab.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT GetNestedClassProps (  
   [in]   mdTypeDef      tdNestedClass,  
   [out]  mdTypeDef      *ptdEnclosingClass  
);  
```  
  
## <a name="parameters"></a>Parameter  

 `tdNestedClass`  
 in Ein TypeDef-Token, das den darstellt <xref:System.Type> , für den das übergeordnete Klassen Token zurückgegeben werden soll  
  
 `ptdEnclosingClass`  
 vorgenommen Ein Zeiger auf das TypeDef-Token für das <xref:System.Type> , das `tdNestedClass` in eingebettet ist.
