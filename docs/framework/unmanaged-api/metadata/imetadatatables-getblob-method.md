# <a name="imetadatatablesgetblob-method"></a>IMetaDataTables::GetBlob-Methode
Ruft einen Zeiger auf das binary large Object (BLOB) am angegebenen Spaltenindex.  
  
## <a name="syntax"></a>Syntax  
  
```  
HRESULT GetBlob (  
    [in]  ULONG          ixBlob,  
    [out] ULONG          *pcbData,  
    [out] const void     **ppData  
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `ixBlob`  
 [in] Die Speicheradresse, von dem abzurufenden `ppData`.  
  
 `pcbData`  
 [out] Ein Zeiger auf die Größe in Bytes, der `ppData`.  
  
 `ppData`  
 [out] Ein Zeiger auf einen Zeiger auf die binären Daten abgerufen werden.
