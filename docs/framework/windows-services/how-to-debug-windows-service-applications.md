## <a name="debugging-tips-for-windows-services"></a><span data-ttu-id="02523-149">Tipps zum Debuggen für Windows-Dienste</span><span class="sxs-lookup"><span data-stu-id="02523-149">Debugging Tips for Windows Services</span></span>  

 <span data-ttu-id="02523-150">Durch das Anfügen an den Prozess des Dienstes können Sie die meisten, aber nicht alle, Codes für diesen Dienst debuggen.</span><span class="sxs-lookup"><span data-stu-id="02523-150">Attaching to the service's process allows you to debug most, but not all, the code for that service.</span></span> <span data-ttu-id="02523-151">Z. B. weil der Dienst bereits gestartet wurde, können Sie den Code in der <xref:System.ServiceProcess.ServiceBase.OnStart%2A>-Methode des Dienstes oder den Code in der `Main`-Methode nicht debuggen, die verwendet wird, um den Dienst auf diese Weise zu lassen.</span><span class="sxs-lookup"><span data-stu-id="02523-151">For example, because the service has already been started, you cannot debug the code in the service's <xref:System.ServiceProcess.ServiceBase.OnStart%2A> method or the code in the `Main` method that is used to load the service this way.</span></span> <span data-ttu-id="02523-152">Dies kann dadurch umgangen werden, dass ein zweiter temporärer Dienst in der Dienstanweisung erstellt wird, der lediglich als Hilfsmittel für das Debuggen verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="02523-152">One way to work around this limitation is to create a temporary second service in your service application that exists only to aid in debugging.</span></span> <span data-ttu-id="02523-153">Beide Dienste können installiert werden.  <span data-ttu-id="02523-155">Versuchen Sie, Aufrufe an die <xref:System.Threading.Thread.Sleep%2A>-Methode zur Verzögerung der Aktion hinzuzufügen, bis Sie sie an den Prozess anfügen können.</span><span class="sxs-lookup"><span data-stu-id="02523-155">Try adding calls to the <xref:System.Threading.Thread.Sleep%2A> method to delay action until you’re able to attach to the process.</span></span>

#### <a name="how-to-run-a-windows-service-as-a-console-application"></a><span data-ttu-id="02523-158">Vorgehensweise: Ausführen eines Windows-Dienstes als Konsolenanwendung</span><span class="sxs-lookup"><span data-stu-id="02523-158">How to: Run a Windows Service as a console application</span></span>

    ```csharp  
    internal void TestStartupAndStop(string[] args)  
    {  
        this.OnStart(args);  
        Console.ReadLine();  
        this.OnStop();  
    }  
    ```  
  
2. <span data-ttu-id="02523-160">Schreiben Sie die `Main`-Methode wie folgt um:</span><span class="sxs-lookup"><span data-stu-id="02523-160">Rewrite the `Main` method as follows:</span></span>

    ```csharp  
    static void Main(string[] args)  
    {  
        if (Environment.UserInteractive)  
        {  
            MyNewService service1 = new MyNewService(args);  
            service1.TestStartupAndStop(args);  
        }  
        else  
        {  
            // Put the body of your old Main method here.  
        }  
    }
    ```  
  
3. <span data-ttu-id="02523-161">Legen Sie in der Registerkarte **Anwendung** der Projekteigenschaften den **Ausgabetyp** auf **Konsolenanwendung** fest.</span><span class="sxs-lookup"><span data-stu-id="02523-161">In the **Application** tab of the project's properties, set the **Output type** to **Console Application**.</span></span>  
  
4. <span data-ttu-id="02523-162">Wählen Sie **Debugging starten** (F5) aus.</span><span class="sxs-lookup"><span data-stu-id="02523-162">Choose **Start Debugging** (F5).</span></span>  
  
5. <span data-ttu-id="02523-163">Wenn Sie das Programm als Windows-Dienst ausführen, installieren Sie es, und starten Sie sie wie gewohnt für einen Windows-Dienst.</span><span class="sxs-lookup"><span data-stu-id="02523-163">To run the program as a Windows Service again, install it and start it as usual for a Windows Service.</span></span> <span data-ttu-id="02523-164">Es ist nicht notwendig, diese Änderungen rückgängig zu machen.</span><span class="sxs-lookup"><span data-stu-id="02523-164">It's not necessary to reverse these changes.</span></span>  
  
 <span data-ttu-id="02523-165">In einigen Fällen, z.
