<span data-ttu-id="3edb3-115">Methoden können über bedingten Code verfügen, z. b. if-Anweisungen, binäre Ausdrücke ( `==` , `!=` , `<` , `>` ), Null-Überprüfungen usw. Sehen Sie sich beispielsweise den folgenden Code an:</span><span class="sxs-lookup"><span data-stu-id="3edb3-115">Methods can have conditional code, such as if statements, binary expressions (`==`, `!=`, `<`, `>`), null checks, etc. For example, consider the following code:</span></span>

```csharp
public void M(int i, int j)
{
    if (i != 0)
    {
        return;
    }

    if (j != 0)
    {
        return;
    }

    // Below condition will always evaluate to 'false' as 'i' and 'j' are both '0' here.
    if (i != j)
    {
        // Code in this 'if' branch is dead code.
        // It can either be removed or refactored.
        ...
    }
}
```
