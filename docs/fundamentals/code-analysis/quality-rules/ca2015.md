## <a name="how-to-fix-violations"></a><span data-ttu-id="634d0-117">Behandeln von Verstößen</span><span class="sxs-lookup"><span data-stu-id="634d0-117">How to fix violations</span></span>

```csharp

    {
        if (disposing)
        {
            _handle.Dispose();

    ~DerivedClass() => Dispose(false);
```
