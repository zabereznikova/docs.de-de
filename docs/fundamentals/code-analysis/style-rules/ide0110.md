## <a name="example"></a><span data-ttu-id="f62d9-119">Beispiel</span><span class="sxs-lookup"><span data-stu-id="f62d9-119">Example</span></span>

```csharp

{
    case int _:
        Console.WriteLine("Value was an int");
        break;
    case string _:
        Console.WriteLine("Value was a string");
        break;
}

// Fixed code

{
    case int:
        Console.WriteLine("Value was an int");
        break;
    case string:
        Console.WriteLine("Value was a string");
        break;
}
```
