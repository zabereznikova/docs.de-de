# <a name="remove-unused-parameter-ide0060"></a>Nicht verwendeten Parameter entfernen (IDE0060)

|Eigenschaft|Wert|
|-|-|
| **Regel-ID** | IDE0060 |
| **Titel** | Nicht verwendeten Parameter entfernen |
| **Kategorie** | Style |
| **Unterkategorie** | Nicht benötigte Coderegeln |
| **Gültige Sprachen** | C# und Visual Basic |

## <a name="overview"></a>Übersicht

Diese Regel bezieht sich auf das Kennzeichnen von nicht verwendeten Parametern. Der Optionswert bestimmt, ob nicht verwendete Parameter nur für nicht öffentliche Methoden oder sowohl für öffentliche als auch für nicht öffentliche Methoden gekennzeichnet werden sollen.

## <a name="dotnet_code_quality_unused_parameters"></a>dotnet_code_quality_unused_parameters

|Eigenschaft|Wert|
|-|-|
| **Options Name** | dotnet_code_quality_unused_parameters
| **Optionswerte** | `all` – Methoden mit Barrierefreiheit markieren, die nicht genutzte Parameter enthalten<br /><br />`non_public` – Nur nicht öffentliche Methoden markieren, die nicht genutzte Parameter enthalten |
| **Standard Optionswert** | `all` |

### <a name="example"></a>Beispiel

```csharp
// dotnet_code_quality_unused_parameters = all
public int GetNum1(int unusedParam) { return 1; }
internal int GetNum2(int unusedParam) { return 1; }
private int GetNum3(int unusedParam) { return 1; }

// dotnet_code_quality_unused_parameters = non_public
internal int GetNum4(int unusedParam) { return 1; }
private int GetNum5(int unusedParam) { return 1; }
```

```vb
' dotnet_code_quality_unused_parameters = all
Public Function GetNum1(unused As Integer)
    Return 1
End Function

Friend Function GetNum2(unused As Integer)
    Return 1
End Function

Private Function GetNum3(unused As Integer)
    Return 1
End Function

' dotnet_code_quality_unused_parameters = non_public
Friend Function GetNum4(arg1 As Integer)
    Return 1
End Function

Private Function GetNum5(arg1 As Integer)
    Return 1
End Function
```
