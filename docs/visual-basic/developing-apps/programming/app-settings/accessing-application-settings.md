# <a name="accessing-application-settings-visual-basic"></a>Zugreifen auf Anwendungseinstellungen (Visual Basic)
Dieser Abschnitt enthält Themen, in denen das `My.Settings`-Objekt und die Aufgaben beschrieben sind, die Sie damit ausführen können.  
  
## <a name="mysettings"></a>My.Settings  
 Die Eigenschaften des `My.Settings`-Objekts ermöglichen den Zugriff auf die Einstellungen Ihrer Anwendung. Verwenden Sie zum Hinzufügen oder Entfernen der Einstellungen den Bereich **Einstellungen** des **Projekt-Designers**.  
  
 Mit den Methoden des `My.Settings`-Objekts können Sie die aktuellen Benutzereinstellungen speichern oder die Benutzereinstellungen auf die zuletzt gespeicherten Werte wiederherstellen.
