 <span data-ttu-id="1aa2e-180">Bei einer Konvertierung von einem *Verweistyp* wird nur der Zeiger auf den Wert kopiert.</span><span class="sxs-lookup"><span data-stu-id="1aa2e-180">A conversion from a *reference type* copies only the pointer to the value.</span></span> <span data-ttu-id="1aa2e-181">Der Wert selbst wird weder kopiert noch auf irgendeine Weise geändert.</span><span class="sxs-lookup"><span data-stu-id="1aa2e-181">The value itself is neither copied nor changed in any way.</span></span> <span data-ttu-id="1aa2e-182">Das einzige, was geändert werden kann, ist der Datentyp der Variablen, die den Zeiger enthält.</span><span class="sxs-lookup"><span data-stu-id="1aa2e-182">The only thing that can change is the data type of the variable holding the pointer.</span></span> <span data-ttu-id="1aa2e-183">Im folgenden Beispiel wird der Datentyp von der abgeleiteten Klasse in seine Basisklasse konvertiert, aber das Objekt, auf das beide Variablen nun zeigen, ist unverändert.</span><span class="sxs-lookup"><span data-stu-id="1aa2e-183">In the following example, the data type is converted from the derived class to its base class, but the object that both variables now point to is unchanged.</span></span>

```vb  
' Assume class cSquare inherits from class cShape.  
Dim shape As cShape  
Dim square As cSquare = New cSquare

' conversion from a derived class to its base class.  
shape = square  
```
