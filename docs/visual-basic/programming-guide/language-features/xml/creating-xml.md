# Creating XML in Visual Basic
[!INCLUDE[vs2017banner](../../../../visual-basic/includes/vs2017banner.md)]

Mit [!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] können Sie *XML\-Literale* direkt im Code verwenden.  Die XML\-Literalsyntax stellt [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq-md.md)]\-Objekte dar und ähnelt der XML 1.0\-Syntax.  Dies vereinfacht die programmgesteuerte Erstellung von XML\-Elementen, \-Dokumenten und \-Fragmenten, da der Code dieselbe Struktur hat wie der fertige XML\-Code.
