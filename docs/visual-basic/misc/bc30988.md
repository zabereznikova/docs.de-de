<span data-ttu-id="a2196-103">Wenn Sie eine Instanz einer Klasse deklarieren, muss auf das Schlüsselwort `New` ein Typname oder `With`folgen.</span><span class="sxs-lookup"><span data-stu-id="a2196-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="a2196-104">Die folgenden Anweisungen deklarieren z. B. jeweils, dass `client` eine Instanz der `Customer` -Klasse ist.</span><span class="sxs-lookup"><span data-stu-id="a2196-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="a2196-105">Der Typname `Customer` folgt auf `New`.</span><span class="sxs-lookup"><span data-stu-id="a2196-105">The type name `Customer` follows `New`.</span></span>  
  
```vb

' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="a2196-106">Ab Visual Basic 2008 können Sie ein Objekt als eine Instanz eines anonymen Typs deklarieren. in diesem Fall geben Sie keinen Datentyp an.</span><span class="sxs-lookup"><span data-stu-id="a2196-106">Beginning with Visual Basic 2008, you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="a2196-107">In Deklarationen anonymer Typen folgt das Schlüsselwort `With` auf `New`.</span><span class="sxs-lookup"><span data-stu-id="a2196-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>

