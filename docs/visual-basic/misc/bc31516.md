# <a name="type-39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-not-declared-39public39"></a>Typ &#39; &lt;Typename&gt; &#39; kann nicht als Attribut verwendet werden, da er nicht deklariert ist &#39;öffentliche&#39;
Die Attribute müssen öffentlich sein.  
  
 **Fehler-ID:** BC31516  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Fügen Sie der Attributklasse den `Public` -Modifizierer hinzu.
