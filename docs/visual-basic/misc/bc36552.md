## <a name="example"></a><span data-ttu-id="40ce1-110">Beispiel</span><span class="sxs-lookup"><span data-stu-id="40ce1-110">Example</span></span>  
 <span data-ttu-id="40ce1-111">Der erste Parameter im folgenden Beispiel gibt an, dass die `Print` -Methode den `String` -Datentyp erweitert.</span><span class="sxs-lookup"><span data-stu-id="40ce1-111">The first parameter in the following example indicates that the `Print` method extends the `String` data type.</span></span>

```  
    Console.WriteLine(str)  
End Sub  
```
