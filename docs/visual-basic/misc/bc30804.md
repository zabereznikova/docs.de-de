# <a name="39variant39-is-no-longer-a-supported-type-use-the-39object39-type-instead"></a>&#39;Variant&#39; ist nicht mehr ein unterstützten Typ handelt. Verwenden Sie die &#39;Objekt&#39; stattdessen den Typ
`Object` wird durch den `Variant`-Datentyp ersetzt.  
  
 **Fehler-ID:** BC30804  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Deklarieren Sie das Element als `Object`.
