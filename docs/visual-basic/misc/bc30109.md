# <a name="39lttypenamegt39-is-a-class-type-and-cannot-be-used-as-an-expression"></a>&#39; &lt;Typename&gt;&#39; ist ein Klassentyp und kann nicht als Ausdruck verwendet werden
Ein Typname tritt auf, wo ein Ausdruck erforderlich ist. Ein Ausdruck muss aus einer Kombination aus Variablen, Konstanten, Literalen, Eigenschaften und `Function` -Prozeduraufrufen bestehen.  
  
 **Fehler-ID:** BC30109  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie den Typnamen, und erstellen Sie den Ausdruck mit gültigen Elementen.
