# <a name="object-initializer-syntax-cannot-be-used-to-initialize-an-instance-of-type-39object39"></a>Die Syntax des Objektinitialisierers kann nicht verwendet werden, zum Initialisieren einer Instanz des Typs &#39; Object &#39;
Eine Instanz von `Object` kann nicht mithilfe der Objektinitialisierersyntax initialisiert werden. Eine Instanz von `Object` enth√§lt keine Eigenschaften oder Felder, denen ein Wert zugewiesen werden kann, und die Objektinitialisierersyntax erfordert mindestens eine entsprechende Eigenschaft oder ein entsprechendes Feld.  
  
```  
' Not valid.  
' Dim obj1 = New Object With {}  
' Dim obj2 = New Object With {.ToString = <some value>}  
```  
  
 **Fehler-ID:** BC30994  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Deklarieren Sie Instanzen des Typs `Object` ohne Verwendung einer Initialisiererliste:  
  
    ```  
    Dim obj3 as Object  
    Dim obj4 as New Object()  
    ```
