# <a name="value-valuename1-cannot-be-converted-to-valuename2"></a>Wert '\<wertname1 >' kann nicht in konvertiert werden kann '\<valuename2 >'
Es wurde eine Typkonvertierung zwischen zwei inkompatiblen Typen versucht.  
  
 **Fehler-ID:** BC30742  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Überprüfen Sie die umzuwandelnden Typen.
