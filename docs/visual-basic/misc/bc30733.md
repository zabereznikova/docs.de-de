# <a name="39ltmembernamegt39-is-already-declared-by-39ltprocedurenamegt39-which-was-generated-for-this-ltcontainernamegt"></a>&#39; &lt;Membername&gt;&#39; wurde bereits deklariert. von &#39;&lt; Prozedurname&gt;&#39; das dafür generiert wurde &lt;Containername&gt;
Es wurde ein Member deklariert, der den gleichen Namen wie ein Member aufweist, der in der Common Language Runtime deklariert ist.  
  
 **Fehler-ID:** BC30733  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Ändern Sie den Namen des Members, um den Konflikt zu vermeiden.
