## <a name="example"></a><span data-ttu-id="be852-111">Beispiel</span><span class="sxs-lookup"><span data-stu-id="be852-111">Example</span></span>  
 <span data-ttu-id="be852-112">Im folgenden Beispiel wird das `AttributeUsage` -Attribut verwendet, um anzugeben, auf welche Programmelemente die neue Version von `ExtensionAttribute` angewendet werden kann.</span><span class="sxs-lookup"><span data-stu-id="be852-112">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="be852-113">Das Beispiel gibt drei Member der `AttributeTargets` -Enumeration an: `Assembly`, `Class`und `Method`.</span><span class="sxs-lookup"><span data-stu-id="be852-113">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="be852-114">Die Auslassung eines dieser Elemente verursacht diesen Fehler.</span><span class="sxs-lookup"><span data-stu-id="be852-114">The omission of any one of these elements will cause this error.</span></span>

        AttributeTargets.Class Or AttributeTargets.Method)>

        Inherits System.Attribute

    End Class
