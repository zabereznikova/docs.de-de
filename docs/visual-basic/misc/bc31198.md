<span data-ttu-id="22a5d-103">Eine Deklaration der XML-Literale wird in einem Ausdruck in einen Speicherort verwendet, die für die Visual Basic-Compiler nicht eindeutig ist.</span><span class="sxs-lookup"><span data-stu-id="22a5d-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="22a5d-104">Visual Basic-Compiler kann nicht bestimmt werden, also, ob das kleiner-als ein Zeichen (<) als Vergleichsoperator oder den Anfang eines XML-Literals vorgesehen ist.</span><span class="sxs-lookup"><span data-stu-id="22a5d-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="22a5d-105">Der folgende Code veranschaulicht dies:</span><span class="sxs-lookup"><span data-stu-id="22a5d-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.

                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 <span data-ttu-id="22a5d-106">**Fehler-ID:** BC31198</span><span class="sxs-lookup"><span data-stu-id="22a5d-106">**Error ID:** BC31198</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="22a5d-107">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="22a5d-107">To correct this error</span></span>
