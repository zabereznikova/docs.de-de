# Der Arrayinitialisierer hat zu wenig Dimensionen.
Sie haben im Initialisierer für das Array nicht genügend Dimensionen angegeben.  
  
 **Fehler\-ID:** BC30565  
  
### So beheben Sie diesen Fehler  
  
-   Überprüfen Sie den Arrayinitialisierer, um zu bestimmen, wie viele Dimensionen erforderlich sind.  
  
-   Geben Sie das fehlende Argument oder die fehlenden Argumente an.
