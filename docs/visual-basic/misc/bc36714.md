# <a name="expanded-properties-cannot-be-initialized"></a><span data-ttu-id="e42bd-102">Erweiterte Eigenschaften können nicht initialisiert werden.</span><span class="sxs-lookup"><span data-stu-id="e42bd-102">Expanded properties cannot be initialized</span></span>
<span data-ttu-id="e42bd-103">Eine automatisch implementierte Eigenschaft kann als Teil der Deklaration initialisiert werden. Dies gilt nicht für eine erweiterte Eigenschaft.</span><span class="sxs-lookup"><span data-stu-id="e42bd-103">An auto-implemented property can be initialized as part of its declaration, but an expanded property cannot be.</span></span>  
  
 <span data-ttu-id="e42bd-104">**Fehler-ID:** BC36714</span><span class="sxs-lookup"><span data-stu-id="e42bd-104">**Error ID:** BC36714</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e42bd-105">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="e42bd-105">To correct this error</span></span>  
  
- <span data-ttu-id="e42bd-106">Verwenden Sie eine automatisch implementierte Eigenschaft, oder entfernen Sie die Initialisierung aus der Eigenschaftendeklaration.</span><span class="sxs-lookup"><span data-stu-id="e42bd-106">Either use an auto-implemented property or remove the initialization from the property declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e42bd-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="e42bd-107">Example</span></span>  
 <span data-ttu-id="e42bd-108">Die folgenden Beispiele zeigen sowohl automatisch implementierte als auch erweiterte Eigenschaften.</span><span class="sxs-lookup"><span data-stu-id="e42bd-108">The following examples show both auto-implemented and expanded properties.</span></span> <span data-ttu-id="e42bd-109">Automatisch implementierte Eigenschaften können mithilfe der Zuweisung oder einer `New` -Klausel initialisiert werden. Dies gilt nicht für erweiterte Eigenschaften.</span><span class="sxs-lookup"><span data-stu-id="e42bd-109">Auto-implemented properties can be initialized by using assignment or a `New` clause, but expanded properties cannot be.</span></span>  
  
```vb  
Class AutoImplementedExample  
    ' An auto-implemented property can be assigned an initial value.  
    Property IDNum As Integer = 33542  
    ' An auto-implemented property can be initialized with New.  
    Property Name As New String("Don Hall")  
End Class  
  
Class ExpandedExample  
    ' Attempting to expand an initialized auto-implemented property  
    ' causes this error.  
    'Property IDNum As Integer = 33542  
    '    Get  
    '    End Get  
    '    Set(ByVal value As Integer)  
    '    End Set  
    'End Property  
  
    ' Instead, you can assign the initial value to the backing field.  
    Private _IDNum As Integer = 33542  
    Property IDNum As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
End Class  
```
