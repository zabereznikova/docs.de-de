# <a name="catch-cannot-catch-type-lttypenamegt-because-it-is-not-systemexception-or-a-class-that-inherits-from-systemexception"></a>"Catch" können nicht Typ Auffangen "&lt;Typename&gt;' da er nicht"System.Exception"oder eine Klasse, die von"System.Exception"erbt
`Catch` kann nur Ausnahmen abfangen, und Sie haben versucht, etwas abzufangen, das nicht von einer Ausnahme abgeleitet ist.  
  
 **Fehler-ID:** BC30392  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Entfernen Sie die `Catch` -Anweisung, oder ändern Sie das Ziel von `Catch` in eine tatsächliche Ausnahme.
