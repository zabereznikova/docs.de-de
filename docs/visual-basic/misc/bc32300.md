# Das Objekt ist aufgrund eines Compilerfehlers oder L&#246;schvorgangs nicht mehr vorhanden.
Innerhalb des [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]\-Compilers ist beim Zugriff auf das Codeelement ein Fehler aufgetreten.  
  
 **Fehler\-ID:** BC32300  
  
### So beheben Sie diesen Fehler  
  
1.  Kompilieren Sie das Programm erneut, um festzustellen, ob der Fehler erneut auftritt.  
  
2.  Wenn der Fehler erneut auftritt, installieren Sie den [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]\-Compiler neu.  
  
3.  Wenn der Fehler nach der Neuinstallation weiterhin besteht, tragen Sie Informationen zu den Umst√§nden zusammen, und benachrichtigen Sie den Produktsupport von Microsoft.
