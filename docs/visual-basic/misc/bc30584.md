# &#39;&lt;Typ&gt;&#39; kann nicht mehrmals geerbt werden.
Sie haben eine `Inherits`\-Anweisung dupliziert.  
  
 **Fehler\-ID:** BC30584  
  
### So beheben Sie diesen Fehler  
  
-   Entfernen Sie die redundante `Inherits`\-Anweisung.
