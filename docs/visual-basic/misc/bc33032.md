# Konvertierungsoperatoren k&#246;nnen keine Konvertierung von &quot;Object&quot; durchf&#252;hren.
Ein Konvertierungsoperator ist mit einem Parameter vom [Object Data Type](../../visual-basic/language-reference/data-types/object-data-type.md) deklariert.  
  
 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] geht zur Kompilierzeit davon aus, dass eine vordefinierte Konvertierung von jedem Verweistyp in jeden Typ in der Vererbungshierarchie vorhanden ist, d. h. in jeden Typ, von dem der Verweistyp abgeleitet oder der vom Verweistyp abgeleitet werden kann. In [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort-md.md)] ist der universelle Datentyp `Object`, daher wird jeder Typ von `Object` abgeleitet.  
  
 Da der Compiler diese Konvertierung als bereits definiert ansieht, lässt er ein erneutes Definieren dieser Konvertierung nicht zu.  
  
 **Fehler\-ID:** BC33032  
  
### So beheben Sie diesen Fehler  
  
-   Entfernen Sie diese Operatordefinition vollständig. Sie ist bereits vordefiniert.
