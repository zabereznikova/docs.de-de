# <a name="39for39-loop-control-variable-already-in-use-by-an-enclosing-39for39-loop"></a>&#39; Für &#39; Loop-Steuerelementvariable bereits in Verwendung von einer einschließenden &#39; Für &#39; Schleife
Bei geschachtelten `For` -Schleifen müssen jeweils eindeutige Steuerungsvariablen verwendet werden.  
  
 **Fehler-ID:** BC30975  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Ändern Sie einen der Namen der Steuerungsvariablen.
