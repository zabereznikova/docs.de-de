# <a name="ltderivedtypenamegt-cannot-inherit-from-lttypegt-ltconstructedbasetypenamegt-because-it-expands-the-access-of-type-ltinternaltypenamegt-to-ltregiongt-ltregionnamegt"></a><span data-ttu-id="36dd5-102">"&lt;Derivedtypename&gt;' kann nicht von erben &lt;Typ&gt; "&lt;konstruierter_basistypname&gt;'da den Zugriff des Typs erweitert"&lt;Internaltypename&gt;"um &lt;Region&gt; "&lt;Regionname&gt;"</span><span class="sxs-lookup"><span data-stu-id="36dd5-102">'&lt;derivedtypename&gt;' cannot inherit from &lt;type&gt; '&lt;constructedbasetypename&gt;' because it expands the access of type '&lt;internaltypename&gt;' to &lt;region&gt; '&lt;regionname&gt;'</span></span>
<span data-ttu-id="36dd5-103">Eine abgeleitete Klasse oder Schnittstelle versucht, die Zugriffsebene eines internen Typs zu erweitern, indem sie diesen als Typargument für eine Basisklasse oder Schnittstelle verwendet.</span><span class="sxs-lookup"><span data-stu-id="36dd5-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="36dd5-104">Dieser Fehler kann durch folgenden Code generiert werden.</span><span class="sxs-lookup"><span data-stu-id="36dd5-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="36dd5-105">Code außerhalb der `containingClass` darf nicht auf die `internalStructure`zugreifen.</span><span class="sxs-lookup"><span data-stu-id="36dd5-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="36dd5-106">Allerdings kann jeder Code in derselben Assembly auf die `derivedClass` zugreifen.</span><span class="sxs-lookup"><span data-stu-id="36dd5-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="36dd5-107">Daher kann die `derivedClass` die `baseClass` nicht erben, wenn sie die `internalStructure` als Typargument verwendet, da auf diese Weise die `internalStructure` in der gesamten definierenden Coderegion verfügbar gemacht würde.</span><span class="sxs-lookup"><span data-stu-id="36dd5-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="36dd5-108">**Fehler-ID:** BC30921</span><span class="sxs-lookup"><span data-stu-id="36dd5-108">**Error ID:** BC30921</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="36dd5-109">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="36dd5-109">To correct this error</span></span>  
  
-   <span data-ttu-id="36dd5-110">Passen Sie die Zugriffsebenen der Klassen oder Schnittstellen so an, dass der abgeleitete Typ die Zugriffsebene des internen Typs nicht erweitert.</span><span class="sxs-lookup"><span data-stu-id="36dd5-110">Adjust the access levels of the classes or interfaces so that the derived type does not expand the access level of the internal type.</span></span>  
  
     <span data-ttu-id="36dd5-111">- oder -</span><span class="sxs-lookup"><span data-stu-id="36dd5-111">-or-</span></span>  
  
-   <span data-ttu-id="36dd5-112">Wenn Sie die Zugriffsebenen nicht anpassen können, verwenden Sie beim Erstellen der Basisklasse oder Schnittstelle den internen Typ nicht als Typargument.</span><span class="sxs-lookup"><span data-stu-id="36dd5-112">If you cannot adjust the access levels, then do not use the internal type as a type argument when constructing the base class or interface.</span></span>
