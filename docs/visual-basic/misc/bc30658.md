# <a name="property-39ltpropertynamegt39-with-no-parameters-cannot-be-found"></a>Eigenschaft &#39; &lt;Propertyname&gt; &#39; ohne Parameter kann nicht gefunden werden
Das verwendete Attribut verfügt über eine Eigenschaft, die einen oder mehrere Parameter erfordert.  
  
 **Fehler-ID:** BC30658  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Geben Sie Parameter für die Eigenschaft an.
