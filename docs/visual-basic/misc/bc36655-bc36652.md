# <a name="data-types-of-the-type-parameters-in-extension-method-methodname-defined-in-typename-cannot-be-inferred-from-these-arguments-because-more-than-one-type-is-possible"></a><span data-ttu-id="12c86-102">Die Datentypen der Typparameter in der Erweiterungsmethode '\<Methodname >' in definierten '\<Typname >' können nicht von diesen Argumenten abgeleitet werden, da mehrere Typen möglich sind.</span><span class="sxs-lookup"><span data-stu-id="12c86-102">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in '\<typename>' cannot be inferred from these arguments because more than one type is possible</span></span>

<span data-ttu-id="12c86-103">Die Datentypen der Typparameter in der Erweiterungsmethode '\<Methodname >' definiert, die '\<Typname >' können nicht von diesen Argumenten abgeleitet werden, da mehrere Typen möglich sind.</span><span class="sxs-lookup"><span data-stu-id="12c86-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in '\<typename>' cannot be inferred from these arguments because more than one type is possible.</span></span> <span data-ttu-id="12c86-104">Sie können diesen Fehler möglicherweise beheben, indem Sie die Datentypen explizit angeben.</span><span class="sxs-lookup"><span data-stu-id="12c86-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="12c86-105">In einem Aufruf einer generischen Erweiterungsmethode wurde versucht, über den Typrückschluss den Typ (oder die Typen) des bzw. der Typparameter zu bestimmen.</span><span class="sxs-lookup"><span data-stu-id="12c86-105">An attempt has been made to use type inference to determine the type (or types) of the type parameter (or parameters) in a call to a generic extension method.</span></span> <span data-ttu-id="12c86-106">Der Compiler findet mehrere mögliche Datentypen für einen oder mehrere Typparameter und meldet diesen Fehler.</span><span class="sxs-lookup"><span data-stu-id="12c86-106">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>

> [!NOTE]
> <span data-ttu-id="12c86-107">Wenn die Angabe von Argumenten keine Option ist (z. B. für Abfrageoperatoren in Abfrageausdrücken), wird nur der erste Satz der Fehlermeldung angezeigt.</span><span class="sxs-lookup"><span data-stu-id="12c86-107">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

<span data-ttu-id="12c86-108">Im folgenden Code wird der Fehler veranschaulicht.</span><span class="sxs-lookup"><span data-stu-id="12c86-108">The following code demonstrates the error.</span></span>

```vb
Option Strict Off
Imports System.Runtime.CompilerServices
Module Module1
    Sub Main()

        Dim caller As New Class1
        '' Not valid.
        'caller.targetExtension(1, "2")

    End Sub

    <Extension()> _
    Sub targetExtension(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)
    End Sub

    Class Class1
    End Class

End Module
```

<span data-ttu-id="12c86-109">**Fehler-ID:** BC36655 (innerhalb von LINQ-Abfragen) und BC36652 (außerhalb von Abfragen)</span><span class="sxs-lookup"><span data-stu-id="12c86-109">**Error ID:** BC36655 (within LINQ queries) and BC36652 (outside queries)</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="12c86-110">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="12c86-110">To correct this error</span></span>

<span data-ttu-id="12c86-111">Wenn der Fehler außerhalb einer Abfrage angezeigt wird, versuchen Sie es mit der expliziten Angabe des Datentyps des Typparameters bzw. der Typparameter:</span><span class="sxs-lookup"><span data-stu-id="12c86-111">If the error appears outside of a query, try specifying the data type of the type parameter or parameters explicitly:</span></span>

```vb
caller.targetExtension(Of Integer)(1, "2")
caller.targetExtension(Of String)(1, "2")
```
