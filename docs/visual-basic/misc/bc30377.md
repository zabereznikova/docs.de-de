# <a name="method-39ltmethodnamegt39-is-already-declared-in-interface-39ltinterfacenamegt39"></a>Methode &#39; &lt;Methodname&gt; &#39; ist bereits in der Schnittstelle deklariert &#39; &lt;Schnittstellenname&gt;&#39;
Sie haben eine Schnittstellenmethode ein zweites Mal deklariert, wodurch diese doppelt angelegt wurde.  
  
 **Fehler-ID:** BC30377  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie die doppelte Deklaration.
