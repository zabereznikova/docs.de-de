# <a name="39continue39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Weiterhin &#39; Anweisungen sind im Direktfenster nicht gültig
`Continue` -Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler-ID:** BC30780  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie die `Continue` -Anweisung aus dem Debugcode.
