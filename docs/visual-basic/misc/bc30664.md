# <a name="conversions-from-lttypename1gt-to-lttypename2gt-must-be-explicit"></a>Konvertierungen von "&lt;Typname1&gt;'to'&lt;Typname2&gt;" muss explizit sein
Diese Konvertierung kann nicht implizit vorgenommen werden.  
  
 **Fehler-ID:** BC30664  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Verwenden Sie `CType`oder eine der speziellen Konvertierungsfunktionen, etwa `CInt` oder `CDec`.
