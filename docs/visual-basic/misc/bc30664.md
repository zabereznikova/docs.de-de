# <a name="conversions-from-39lttypename1gt39-to-39lttypename2gt39-must-be-explicit"></a>Konvertierungen von &#39; &lt;Typname1&gt; &#39; auf &#39; &lt;Typname2&gt; &#39; muss explizit sein.
Diese Konvertierung kann nicht implizit vorgenommen werden.  
  
 **Fehler-ID:** BC30664  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Verwenden Sie `CType`oder eine der speziellen Konvertierungsfunktionen, etwa `CInt` oder `CDec`.
