# <a name="39sub-new39-cannot-implement-interface-members"></a>&#39; Sub New &#39; kann keine Schnittstellenmember implementieren
`Sub New` ist ein Konstruktor und kann keine Member implementieren.  
  
 **Fehler-ID:** BC31042  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie `Implements` -Anweisungen aus `Sub New` -Prozeduren.
