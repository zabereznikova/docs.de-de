# <a name="nesting-of-parentheses-in-lambda-expression39s-parameter-list-exceeds-maximum-allowable-depth"></a>Schachteln von Klammern in Lambda-Ausdruck&#39;Parameterliste s überschreitet die maximal zulässige Tiefe.
Für geschachtelte Klammern in der Parameterliste eines lambda-Ausdrucks darf eine Tiefe von 128 nicht überschritten werden.  
  
 **Fehler-ID:** BC36664
