# <a name="39removehandler39-declaration-must-end-with-a-matching-39end-removehandler39"></a>&#39; RemoveHandler &#39; Deklaration muss mit einem übereinstimmenden &#39; enden. End RemoveHandler &#39;
Eine `RemoveHandler` -Deklaration muss mit einer `End RemoveHandler` -Anweisung enden.  
  
 **Fehler-ID:** BC31116  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Überprüfen Sie, ob die `RemoveHandler` -Deklaration mit einer `End RemoveHandler` -Anweisung endet.
