# <a name="indirect-constraint-ltconstraint1gt-obtained-from-the-type-parameter-constraint-lttypeparameter1gt-conflicts-with-the-indirect-constraint-ltconstraint2gt-obtained-from-the-type-parameter-constraint-lttypeparameter2gt"></a><span data-ttu-id="e2630-102">Die indirekte Einschränkung "&lt;einschränkung1&gt;"abgerufen haben, aus der typparametereinschränkung"&lt;typparameter1&gt;"steht in Konflikt mit der indirekten Einschränkung"&lt;einschränkung2&gt;" abgerufen aus der typparametereinschränkung "&lt;typparameter2&gt;"</span><span class="sxs-lookup"><span data-stu-id="e2630-102">Indirect constraint '&lt;constraint1&gt;' obtained from the type parameter constraint '&lt;typeparameter1&gt;' conflicts with the indirect constraint '&lt;constraint2&gt;' obtained from the type parameter constraint '&lt;typeparameter2&gt;'</span></span>

 <span data-ttu-id="e2630-104">Dieser Fehler kann durch die folgende Anweisung generiert werden.</span><span class="sxs-lookup"><span data-stu-id="e2630-104">The following statement can generate this error.</span></span>

  
 <span data-ttu-id="e2630-105">Die indirekten Einschränkungen `Structure` und `Class` verursachen einen Konflikt für Typparameter `t1`, weil die `Structure` -Einschränkung erfordert, dass das entsprechende Typargument ein Werttyp ist, während `Class` erfordert, dass es ein Verweistyp ist.</span><span class="sxs-lookup"><span data-stu-id="e2630-105">The indirect constraints `Structure` and `Class` cause a conflict for type parameter `t1`, because the `Structure` constraint requires that the corresponding type argument be a value type, while `Class` requires that it be a reference type.</span></span>
