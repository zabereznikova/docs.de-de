# Konstanten m&#252;ssen einen Wert erhalten.
Sie haben eine Konstante deklariert, ohne ihr einen Wert zuzuweisen.  
  
 **Fehler\-ID:** BC30438  
  
### So beheben Sie diesen Fehler  
  
-   Weisen Sie der Konstante einen Wert zu.
