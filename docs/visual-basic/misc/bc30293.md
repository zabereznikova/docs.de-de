# <a name="lterrorgt-39ltstructurename1gt39-contains-39ltstructurename2gt39"></a>&lt;Fehler beim&gt;: &#39;&lt; strukturname1&gt;&#39; enthält &#39;&lt; strukturname2&gt;&#39;
Eine zirkuläre Strukturdefinition entsteht, wenn in der Definition die Struktur selbst als Member genannt wird.  
  
 **Fehler-ID:** BC30293  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Ändern Sie den Namen des Strukturmembers.
