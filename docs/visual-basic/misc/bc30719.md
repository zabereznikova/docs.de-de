# <a name="39case39-statements-are-not-valid-in-the-immediate-window"></a>'Case'-Anweisungen sind im Direktfenster ungültig
`Case`Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler-ID:** BC30719  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie `Case` -Anweisungen aus dem **Direktfenster** .
