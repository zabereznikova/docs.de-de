# <a name="unable-to-access-member"></a>Auf den Member kann nicht zugegriffen werden.
Es war nicht möglich, auf den angegebenen Member zuzugreifen.  
  
 **Fehler-ID:** BC30749  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Vergewissern Sie sich, dass der Member definiert ist und Sie den Membernamen richtig geschrieben haben.  
  
2.  Versuchen Sie, auf einen der im Modul deklarierten Member zuzugreifen. In einigen Fällen können in der Debugumgebung Member nicht gefunden werden, weil die Module, in denen sie deklariert sind, noch nicht geladen wurden.
