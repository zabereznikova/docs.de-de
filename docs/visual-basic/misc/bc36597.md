# <a name="goto-to-label-39ltlabelnamegt39-cannot-be-used-within-lambda-expressions"></a>GoTo zu Bezeichnung &#39; &lt;Bezeichnungsname&gt; &#39; kann nicht in Lambda-Ausdrücken verwendet werden
Der Kern eines Lambda-Ausdrucks muss ein Ausdruck sein, der einen einzelnen Wert berechnet und zurückgibt. Es sind keine Anweisungen zulässig, wozu auch `GoTo` -Anweisungen gehören.  
  
 **Fehler-ID:** BC36597
