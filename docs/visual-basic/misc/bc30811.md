# <a name="39redim39-statements-can-no-longer-be-used-to-declare-array-variables"></a>&#39;ReDim&#39; Anweisungen können nicht mehr zum Deklarieren eines Arrays verwendet werden
`ReDim` kann nur verwendet werden, um die Größe eines vorhandenen Arrays zu ändern.  
  
 **Fehler-ID:** BC30811  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Geben Sie die Größe des Arrays bei der Deklaration an; Beispiel:  
  
    ```  
    Dim X(20) As Integer  
    ```
