# <a name="interface-39ltinterfacenamegt39-can-be-implemented-only-once-by-this-type"></a>Schnittstelle &#39; &lt;Schnittstellenname&gt; &#39; nur einmal von diesem Typ implementiert werden können
Die angegebene Schnittstelle wurde bereits von diesem Typ implementiert und kann nicht erneut implementiert werden.  
  
 **Fehler-ID:** BC31033  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Implementieren Sie diese Schnittstelle nur einmal mit diesem Typ.
