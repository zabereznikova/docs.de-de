# <a name="39raiseevent39-declaration-must-end-with-a-matching-39end-raiseevent39"></a>&#39; RaiseEvent &#39; Deklaration muss mit einem übereinstimmenden &#39; enden. End RaiseEvent &#39;
Eine `RaiseEvent` -Deklaration muss mit einer `End RaiseEvent` -Anweisung enden.  
  
 **Fehler-ID:** BC31117  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Überprüfen Sie, ob die `RaiseEvent` -Deklaration mit einer `End RaiseEvent` -Anweisung endet.
