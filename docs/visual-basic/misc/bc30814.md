# <a name="39gosub39-statements-are-no-longer-supported"></a>&#39; GoSub &#39; Anweisungen werden nicht mehr unterstützt.
`GoSub` kann nicht zum Aufrufen einer Prozedur verwendet werden.  
  
 **Fehler-ID:** BC30814  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Rufen Sie Prozeduren direkt ohne `GoSub`auf. Beispiel:  
  
    ```  
    CalculateInterest(Amount, Rate, Time)  
    ```
