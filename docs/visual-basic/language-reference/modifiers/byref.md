# <a name="byref-visual-basic"></a>ByRef (Visual Basic)
Gibt an, dass ein Argument so übergeben wird, dass die aufgerufene Prozedur den Wert einer Variablen, die das Argument im aufrufenden Code zugrunde liegt, ändern kann.  
  
## <a name="remarks"></a>Hinweise  
 Der `ByRef`-Modifizierer kann in folgenden Kontexten verwendet werden:  
  
 [Declare-Anweisung](../../../visual-basic/language-reference/statements/declare-statement.md)  
  
 [Function-Anweisung](../../../visual-basic/language-reference/statements/function-statement.md)  
  
 [Sub-Anweisung](../../../visual-basic/language-reference/statements/sub-statement.md)
