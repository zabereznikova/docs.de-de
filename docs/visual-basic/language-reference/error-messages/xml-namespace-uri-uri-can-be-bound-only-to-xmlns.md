# <a name="xml-namespace-uri-39httpwwww3orgxml1998namespace39-can-be-bound-only-to-39xmlns39"></a>XML-Namespace-URI &#39; http://www.w3.org/XML/1998/namespace &#39; kann nur für gebunden werden &#39; Xmlns &#39;
Dem URI http://www.w3.org/XML/1998/namespace wird in der Deklaration eines XML-Namespace verwendet. Dieser URI wird von einem reservierten Namespace und kann nicht in eine XML-Namespacedeklaration enthalten sein.  
  
 **Fehler-ID:** BC31183  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie die XML-Namespace-Deklaration, oder Ersetzen Sie dem URI http://www.w3.org/XML/1998/namespace durch einen gültigen Namespace-URI.
