Der `Is` Vergleichs Operator bestimmt, ob zwei Objektvariablen auf dieselbe Instanz verweisen. Dieser Vergleich ist f√ºr Werttypen nicht definiert.  
  
 **Fehler-ID:** BC30020  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Verwenden Sie den entsprechenden arithmetischen Vergleichs Operator oder den- `Like` Operator, um zwei Werttypen zu vergleichen.
