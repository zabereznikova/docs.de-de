# <a name="mycomputerfilesystemspecialdirectories-object"></a>My.Computer.FileSystem.SpecialDirectories-Objekt
Stellt Eigenschaften für den Zugriff auf häufig verwiesene Verzeichnisse bereit.  
  
## <a name="remarks"></a>Hinweise  
 Informationen zu den Methoden und Eigenschaften der `My.Computer.FileSystem.SpecialDirectories` Objekt, finden Sie unter <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>.  
  
 Weitere Informationen finden Sie unter [wie: Abrufen des Inhalts des Verzeichnisses "Eigene Dateien"](../../../visual-basic/developing-apps/programming/drives-directories-files/how-to-retrieve-the-contents-of-the-my-documents-directory.md).  
  
## <a name="requirements"></a>Anforderungen  
 **Namespace:**<xref:Microsoft.VisualBasic.MyServices>  
  
 **Klasse:** <xref:Microsoft.VisualBasic.MyServices.SpecialDirectoriesProxy> (ermöglicht den Zugriff auf <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>)
