<span data-ttu-id="b27ac-104">Wenn Sie ein Framework in einer App oder Bibliothek als Ziel verwenden, geben Sie mehrere APIs an, die Sie für die App oder Bibliothek verfügbar machen möchten.</span><span class="sxs-lookup"><span data-stu-id="b27ac-104">When you target a framework in an app or library, you're specifying the set of APIs that you'd like to make available to the app or library.</span></span> <span data-ttu-id="b27ac-105">Das Zielframework wird mithilfe von Zielframeworkmonikern (Target Framework Monikers, TFMs) in Ihrer Projektdatei angegeben.</span><span class="sxs-lookup"><span data-stu-id="b27ac-105">You specify the target framework in your project file using a target framework moniker (TFM).</span></span>

<span data-ttu-id="b27ac-106">Eine App oder Bibliothek kann eine Version des [.NET Standards](net-standard.md) als Ziel verwenden.</span><span class="sxs-lookup"><span data-stu-id="b27ac-106">An app or library can target a version of [.NET Standard](net-standard.md).</span></span> <span data-ttu-id="b27ac-107">.NET Standard-Versionen stellen standardisierte APIs in allen .NET-Implementierungen dar.</span><span class="sxs-lookup"><span data-stu-id="b27ac-107">.NET Standard versions represent standardized sets of APIs across all .NET implementations.</span></span> <span data-ttu-id="b27ac-108">Eine Bibliothek kann z.B. <span data-ttu-id="b27ac-111">Für einige Zielframeworks (z. B. das .NET Framework) werden die APIs von den Assemblys definiert, die das Framework in einem System installiert. ASP.NET).</span><span class="sxs-lookup"><span data-stu-id="b27ac-111">For some target frameworks, such as .NET Framework, the APIs are defined by the assemblies that the framework installs on a system and may include application framework APIs (for example, ASP.NET).</span></span>

<span data-ttu-id="b27ac-112">Für paketbasierte Zielframeworks (z. B. .NET 5, .NET Core und .NET Standard) werden die APIs von den NuGet-Paketen definiert, die in der App oder der Bibliothek enthalten sind.</span><span class="sxs-lookup"><span data-stu-id="b27ac-112">For package-based target frameworks (for example, .NET 5, .NET Core, and .NET Standard), the APIs are defined by the NuGet packages included in the app or library.</span></span>
