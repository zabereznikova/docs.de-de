 <span data-ttu-id="883b1-109">Wenn das <xref:System.Xml.Schema.XmlSchemaDatatype>-Objekt einen Listentyp darstellt, konvertiert das Objekt den Wert der Eingabezeichenfolge in eine Liste aus einem oder mehreren Objekten.</span><span class="sxs-lookup"><span data-stu-id="883b1-109">If the <xref:System.Xml.Schema.XmlSchemaDatatype> object represents a list type the object converts the input string value to a list of one or more objects.</span></span> <span data-ttu-id="883b1-110">Wenn das <xref:System.Xml.Schema.XmlSchemaDatatype>-Objekt einen Union-Typ darstellt, wird versucht, den Eingabewert als einen Membertyp der Union zu interpretieren.</span><span class="sxs-lookup"><span data-stu-id="883b1-110">If the <xref:System.Xml.Schema.XmlSchemaDatatype> represents a union type then an attempt is made to parse the input value as a member type of the union.</span></span> <span data-ttu-id="883b1-111">Wenn ein Versuch fehlschlägt, wird der Konvertierungsversuch mit dem nächsten Member der Union wiederholt.  <span data-ttu-id="883b1-123">In gleicher Weise wird den Instanzen von `xs:gMonth`, `xs:gDay`, `xs:gYear`, `xs:gYearMonth` und `xs:gMonthDay` ein <xref:System.DateTime>-Objekt zugeordnet.</span><span class="sxs-lookup"><span data-stu-id="883b1-123">Similarly, instances of `xs:gMonth`, `xs:gDay`, `xs:gYear`, `xs:gYearMonth` and `xs:gMonthDay` are also mapped to a <xref:System.DateTime> object.</span></span> <span data-ttu-id="883b1-124">Nicht verwendete Eigenschaften des <xref:System.DateTime>-Objekts werden mit den Werten von <xref:System.DateTime.MinValue> initialisiert.</span><span class="sxs-lookup"><span data-stu-id="883b1-124">Unused properties on the <xref:System.DateTime> object are initialized to those from <xref:System.DateTime.MinValue>.</span></span>
