## <a name="creating-and-deploying-microservices-on-containers"></a><span data-ttu-id="26be6-127">Erstellen und Bereitstellen von Microservices in Containern</span><span class="sxs-lookup"><span data-stu-id="26be6-127">Creating and deploying microservices on containers</span></span>
