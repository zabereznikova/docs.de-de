 <span data-ttu-id="dcfd9-107">Beispielsweise ist eine Eigenschaft mit dem Namen `HorizontalAlignment` besser lesbar als `AlignmentHorizontal` .</span><span class="sxs-lookup"><span data-stu-id="dcfd9-107">For example, a property named `HorizontalAlignment` is more English-readable than `AlignmentHorizontal`.</span></span>

 <span data-ttu-id="dcfd9-108">✔️ bevorzugen die Lesbarkeit gegenüber der Übersichtlichkeit.</span><span class="sxs-lookup"><span data-stu-id="dcfd9-108">✔️ DO favor readability over brevity.</span></span>

 <span data-ttu-id="dcfd9-109">Der Eigenschaftsname `CanScrollHorizontally` ist besser als `ScrollableX` (ein undurchsichtiger Verweis auf die X-Achse).</span><span class="sxs-lookup"><span data-stu-id="dcfd9-109">The property name `CanScrollHorizontally` is better than `ScrollableX` (an obscure reference to the X-axis).</span></span>

 <span data-ttu-id="dcfd9-110">❌Verwenden Sie keine Unterstriche, Bindestriche oder andere nicht alphanumerische Zeichen.</span><span class="sxs-lookup"><span data-stu-id="dcfd9-110">❌ DO NOT use underscores, hyphens, or any other nonalphanumeric characters.</span></span>

 <span data-ttu-id="dcfd9-111">❌Verwenden Sie keine ungarische Notation.</span><span class="sxs-lookup"><span data-stu-id="dcfd9-111">❌ DO NOT use Hungarian notation.</span></span>

 <span data-ttu-id="dcfd9-112">❌Vermeiden Sie die Verwendung von bezeichlern, die mit Schlüsselwörtern häufig verwendeter Programmiersprachen in Konflikt stehen</span><span class="sxs-lookup"><span data-stu-id="dcfd9-112">❌ AVOID using identifiers that conflict with keywords of widely used programming languages.</span></span>

 <span data-ttu-id="dcfd9-113">Gemäß Regel 4 der Common Language Specification (CLS) müssen alle kompatiblen Sprachen einen Mechanismus bereitstellen, der den Zugriff auf benannte Elemente ermöglicht, die ein Schlüsselwort dieser Sprache als Bezeichner verwenden.</span><span class="sxs-lookup"><span data-stu-id="dcfd9-113">According to Rule 4 of the Common Language Specification (CLS), all compliant languages must provide a mechanism that allows access to named items that use a keyword of that language as an identifier.</span></span> <span data-ttu-id="dcfd9-114">C# verwendet in diesem Fall z. b. das @-Zeichen als Escapezeichen.</span><span class="sxs-lookup"><span data-stu-id="dcfd9-114">C#, for example, uses the @ sign as an escape mechanism in this case.</span></span> <span data-ttu-id="dcfd9-115">Es ist jedoch immer noch eine gute Idee, gängige Schlüsselwörter zu vermeiden, da es weitaus schwieriger ist, eine Methode mit der Escapesequenz zu verwenden, als Sie ohne Sie.</span><span class="sxs-lookup"><span data-stu-id="dcfd9-115">However, it is still a good idea to avoid common keywords because it is much more difficult to use a method with the escape sequence than one without it.</span></span>

 <span data-ttu-id="dcfd9-118">Verwenden Sie z `GetWindow` . b. anstelle von `GetWin` .</span><span class="sxs-lookup"><span data-stu-id="dcfd9-118">For example, use `GetWindow` rather than `GetWin`.</span></span>

 <span data-ttu-id="dcfd9-119">❌Verwenden Sie keine Akronyme, die nicht weitgehend akzeptiert werden, und auch dann nicht, wenn dies der Fall ist.</span><span class="sxs-lookup"><span data-stu-id="dcfd9-119">❌ DO NOT use any acronyms that are not widely accepted, and even if they are, only when necessary.</span></span>
