  <span data-ttu-id="98875-114">Wenn der *Genauigkeitsspezifizierer* die Anzahl von Dezimalstellen in der Ergebniszeichenfolge steuert, gibt die Ergebniszeichenfolge eine Zahl an, die auf ein darstellbares Ergebnis gerundet wird, das am ehesten dem unendlich präzisen Ergebnis entspricht.</span><span class="sxs-lookup"><span data-stu-id="98875-114">When *precision specifier* controls the number of fractional digits in the result string, the result string reflects a number that is rounded to a representable result nearest to the infinitely precise result.</span></span> <span data-ttu-id="98875-115">Wenn zwei gleich nahe darstellbare Ergebnisse vorhanden sind:</span><span class="sxs-lookup"><span data-stu-id="98875-115">If there are two equally near representable results:</span></span>
