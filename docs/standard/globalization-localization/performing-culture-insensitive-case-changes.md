 <span data-ttu-id="86728-114">Aus Gründen der Übersichtlichkeit des Codes wird empfohlen, immer die Überladungen der `String.ToUpper`-Methode und der `String.ToLower`-Methode zu verwenden, bei denen ein `culture`-Parameter explizit angegeben werden kann.</span><span class="sxs-lookup"><span data-stu-id="86728-114">For code clarity, it is recommended that you always use overloads of the `String.ToUpper` and `String.ToLower` methods that allow you to specify a `culture` parameter explicitly.</span></span> <span data-ttu-id="86728-115">Mit dem folgenden Code wird z. B. nach einem Bezeichner gesucht.</span><span class="sxs-lookup"><span data-stu-id="86728-115">For example, the following code performs an identifier lookup.</span></span> <span data-ttu-id="86728-116">In der Standardeinstellung ist die `key.ToLower`-Operation kulturabhängig, jedoch wird dieses Verhalten beim Lesen des Codes nicht deutlich.</span><span class="sxs-lookup"><span data-stu-id="86728-116">The `key.ToLower` operation is culture-sensitive by default, but this behavior is not clear from reading the code.</span></span>  
  
### <a name="example"></a><span data-ttu-id="86728-117">Beispiel</span><span class="sxs-lookup"><span data-stu-id="86728-117">Example</span></span>  
  
```vb

   Return internalHashtable(key.ToLower())  
End Function  
```

static object LookupKey(string key)
{  
    return internalHashtable[key.ToLower()];  
}  
```

```  
  
```csharp  
static object LookupKey(string key)
{  
    return internalHashtable[key.ToLower(CultureInfo.InvariantCulture)];

```
