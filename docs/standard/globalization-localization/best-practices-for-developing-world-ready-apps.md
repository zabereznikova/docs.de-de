4. <span data-ttu-id="df63e-116">Aktivieren Sie für die Anwendung das Lesen und Schreiben von Daten für eine Vielzahl von Codierungen, indem Sie die Codierungsklassen im <xref:System.Text>-Namespace verwenden.</span><span class="sxs-lookup"><span data-stu-id="df63e-116">Enable your application to read and write data to and from a variety of encodings by using the encoding classes in the <xref:System.Text> namespace.</span></span> <span data-ttu-id="df63e-117">Gehen Sie nicht von ASCII-Daten aus.</span><span class="sxs-lookup"><span data-stu-id="df63e-117">Do not assume ASCII data.</span></span> <span data-ttu-id="df63e-118">Gehen Sie davon aus, dass internationale Zeichen überall dort zur Verfügung gestellt werden, wo der Benutzer Text eingeben kann.</span><span class="sxs-lookup"><span data-stu-id="df63e-118">Assume that international characters will be supplied anywhere a user can enter text.</span></span> <span data-ttu-id="df63e-119">Beispielsweise sollte die Anwendung internationale Zeichen in Servernamen, Verzeichnissen, Dateinamen, Benutzernamen und URLs akzeptieren.</span><span class="sxs-lookup"><span data-stu-id="df63e-119">For example, the application should accept international characters in server names, directories, file names, user names, and URLs.</span></span>

5. <span data-ttu-id="df63e-120">Wenn Sie die <xref:System.Text.UTF8Encoding>-Klasse verwenden, sollten Sie aus Sicherheitsgründen die von dieser Klasse bereitgestellte Fehlererkennungsfunktion verwenden.</span><span class="sxs-lookup"><span data-stu-id="df63e-120">When using the <xref:System.Text.UTF8Encoding> class, for security reasons, use the error detection feature offered by this class.</span></span> <span data-ttu-id="df63e-121">Erstellen Sie zum Aktivieren der Fehlererkennungsfunktion eine Instanz der Klasse mit dem Konstruktor, der einen `throwOnInvalidBytes`-Parameter akzeptiert, und legen Sie den Wert dieses Parameters auf `true` fest.</span><span class="sxs-lookup"><span data-stu-id="df63e-121">To turn on the error detection feature, create an instance of the class using the constructor that takes a `throwOnInvalidBytes` parameter and set the value of this parameter to `true`.</span></span>

6. <span data-ttu-id="df63e-122">Behandeln Sie Zeichenfolgen möglichst als ganze Zeichenfolgen und nicht als eine Reihe einzelner Zeichen.</span><span class="sxs-lookup"><span data-stu-id="df63e-122">Whenever possible, handle strings as entire strings instead of as a series of individual characters.</span></span> <span data-ttu-id="df63e-123">Dies ist besonders beim Sortieren von und Suchen nach untergeordneten Zeichenfolgen von Bedeutung.</span><span class="sxs-lookup"><span data-stu-id="df63e-123">This is especially important when sorting or searching for substrings.</span></span> <span data-ttu-id="df63e-124">Dadurch werden Probleme vermieden, die beim Analysieren von kombinierten Zeichen auftreten können.</span><span class="sxs-lookup"><span data-stu-id="df63e-124">This will prevent problems associated with parsing combined characters.</span></span> <span data-ttu-id="df63e-125">Sie können auch mit Texteinheiten anstatt mit einzelnen Zeichen arbeiten, indem Sie die Klasse <xref:System.Globalization.StringInfo?displayProperty=nameWithType> verwenden.</span><span class="sxs-lookup"><span data-stu-id="df63e-125">You can also work with units of text rather than single characters by using the <xref:System.Globalization.StringInfo?displayProperty=nameWithType> class.</span></span>

7. <span data-ttu-id="df63e-126">Zeigen Sie Text mithilfe der Klassen im <xref:System.Drawing>-Namespace an.</span><span class="sxs-lookup"><span data-stu-id="df63e-126">Display text using the classes provided by the <xref:System.Drawing> namespace.</span></span>

8. <span data-ttu-id="df63e-127">Damit die betriebssystemübergreifende Konsistenz gewährleistet ist, dürfen Benutzereinstellungen <xref:System.Globalization.CultureInfo> nicht überschreiben.</span><span class="sxs-lookup"><span data-stu-id="df63e-127">For consistency across operating systems, do not allow user settings to override <xref:System.Globalization.CultureInfo>.</span></span> <span data-ttu-id="df63e-128">Verwenden Sie den `CultureInfo`-Konstruktor, der einen `useUserOverride`-Parameter akzeptiert, und legen Sie diesen auf `false` fest.</span><span class="sxs-lookup"><span data-stu-id="df63e-128">Use the `CultureInfo` constructor that accepts a `useUserOverride` parameter and set it to `false`.</span></span>

9. <span data-ttu-id="df63e-129">Testen Sie die Funktionsfähigkeit der Anwendung in internationalen Betriebssystemen und mit internationalen Daten.</span><span class="sxs-lookup"><span data-stu-id="df63e-129">Test your application functionality on international operating system versions, using international data.</span></span>

10. <span data-ttu-id="df63e-130">Wenn eine Sicherheitsentscheidung auf dem Ergebnis eines Zeichenfolgenvergleichs oder der Änderung der Groß-/Kleinschreibung beruht, sollten Sie eine kulturunabhängige Zeichenfolgenoperation verwenden.</span><span class="sxs-lookup"><span data-stu-id="df63e-130">If a security decision is based on the result of a string comparison or case change operation, use a culture-insensitive string operation.</span></span> <span data-ttu-id="df63e-131">Auf diese Weise stellen Sie sicher, dass das Ergebnis nicht vom Wert von `CultureInfo.CurrentCulture` beeinflusst wird.</span><span class="sxs-lookup"><span data-stu-id="df63e-131">This practice ensures that the result is not affected by the value of `CultureInfo.CurrentCulture`.</span></span> <span data-ttu-id="df63e-132">Im Abschnitt [Zeichenfolgenvergleiche mit der aktuellen Kultur](../base-types/best-practices-strings.md#string-comparisons-that-use-the-current-culture) im Artikel [Empfohlene Vorgehensweisen für die Verwendung von Zeichenfolgen in .NET](../base-types/best-practices-strings.md) finden Sie ein Beispiel, das veranschaulicht, wie kulturabhängige Zeichenfolgenvergleiche zu inkonsistenten Ergebnissen führen können.</span><span class="sxs-lookup"><span data-stu-id="df63e-132">See the ["String Comparisons that Use the Current Culture"](../base-types/best-practices-strings.md#string-comparisons-that-use-the-current-culture) section of [Best Practices for Using Strings](../base-types/best-practices-strings.md) for an example that demonstrates how culture-sensitive string comparisons can produce inconsistent results.</span></span>

## <a name="localization-best-practices"></a><span data-ttu-id="df63e-133">Bewährte Methoden für die Lokalisierung</span><span class="sxs-lookup"><span data-stu-id="df63e-133">Localization best practices</span></span>

9. <span data-ttu-id="df63e-148">Verwenden Sie [Visual Studio](https://visualstudio.microsoft.com/vs/?utm_medium=microsoft&utm_source=docs.microsoft.com&utm_campaign=inline+link) zum Erstellen von Windows Forms-Dialogfeldern, sodass diese mit dem [Windows Forms Resource Editor (Winres.exe)](../../framework/tools/winres-exe-windows-forms-resource-editor.md) lokalisiert werden können.</span><span class="sxs-lookup"><span data-stu-id="df63e-148">Use [Visual Studio](https://visualstudio.microsoft.com/vs/?utm_medium=microsoft&utm_source=docs.microsoft.com&utm_campaign=inline+link) to create Windows Forms dialog boxes so they can be localized using the [Windows Forms Resource Editor (Winres.exe)](../../framework/tools/winres-exe-windows-forms-resource-editor.md).</span></span> <span data-ttu-id="df63e-149">Windows Forms-Dialogfelder dürfen nicht manuell codiert werden.</span><span class="sxs-lookup"><span data-stu-id="df63e-149">Do not code Windows Forms dialog boxes by hand.</span></span>

10. <span data-ttu-id="df63e-150">Es empfiehlt sich, eine professionelle Lokalisierung (Übersetzung) anfertigen zu lassen.</span><span class="sxs-lookup"><span data-stu-id="df63e-150">Arrange for professional localization (translation).</span></span>
