# <a name="compiler-warning-level-1-cs0183"></a><span data-ttu-id="dc03f-102">Compilerwarnung (Stufe 1) CS0183</span><span class="sxs-lookup"><span data-stu-id="dc03f-102">Compiler Warning (level 1) CS0183</span></span>
<span data-ttu-id="dc03f-103">Der angegebene Ausdruck ist immer vom bereitgestellten Typ ("Typ")</span><span class="sxs-lookup"><span data-stu-id="dc03f-103">The given expression is always of the provided ('type') type</span></span>  
  
 <span data-ttu-id="dc03f-104">Wenn eine Bedingungsanweisung immer **true**ergibt, benötigen Sie keine Bedingungsanweisung.</span><span class="sxs-lookup"><span data-stu-id="dc03f-104">If a conditional statement always evaluates to **true**, then you do not need a conditional statement.</span></span> <span data-ttu-id="dc03f-105">Diese Warnung wird angezeigt, wenn Sie versuchen, einen Typ mit dem **is** -Operator auszuwerten.</span><span class="sxs-lookup"><span data-stu-id="dc03f-105">This warning occurs when you try to evaluate a type using the **is** operator.</span></span> <span data-ttu-id="dc03f-106">Wenn es sich bei der Auswertung um einen Werttyp handelt, ist die Überprüfung nicht erforderlich.</span><span class="sxs-lookup"><span data-stu-id="dc03f-106">If the evaluation is a value type, then the check is unnecessary.</span></span>  
  
 <span data-ttu-id="dc03f-107">Im folgenden Beispiel wird CS0183 generiert:</span><span class="sxs-lookup"><span data-stu-id="dc03f-107">The following sample generates CS0183:</span></span>  
  
```csharp  
// CS0183.cs  
// compile with: /W:1  
using System;  
public class Test  
{  
   public static void F(Int32 i32, String str)  
   {  
      if (str is Object)          // OK  
         Console.WriteLine( "str is an object" );  
      else  
         Console.WriteLine( "str is not an object" );  
  
      if (i32 is Object)   // CS0183  
         Console.WriteLine( "i32 is an object" );  
      else  
         Console.WriteLine( "i32 is not an object" ); // never reached  
   }  
  
   public static void Main()  
   {  
  
      F(0, "CS0183");  
      F(120, null);   
   }  
}  
```
