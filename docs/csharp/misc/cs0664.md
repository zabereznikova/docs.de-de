# <a name="compiler-error-cs0664"></a><span data-ttu-id="07ef1-102">Compilerfehler CS0664</span><span class="sxs-lookup"><span data-stu-id="07ef1-102">Compiler Error CS0664</span></span>
<span data-ttu-id="07ef1-103">Literale vom Typ 'double' können nicht implizit in den 'Typ'-Typ konvertiert werden. Verwenden Sie ein 'Suffix'-Suffix, um ein Literal mit diesem Typ zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="07ef1-103">Literal of type double cannot be implicitly converted to type 'type'; use an 'suffix' suffix to create a literal of this type</span></span>  
  
 <span data-ttu-id="07ef1-104">Eine Zuordnung konnte nicht abgeschlossen werden. Verwenden Sie ein Suffix, um die Anweisung zu korrigieren.</span><span class="sxs-lookup"><span data-stu-id="07ef1-104">An assignment could not be completed; use a suffix to correct the instruction.</span></span> <span data-ttu-id="07ef1-105">Die Dokumentation für die einzelnen Typen identifiziert das entsprechende Suffix für den Typ.</span><span class="sxs-lookup"><span data-stu-id="07ef1-105">The documentation for each type identifies the corresponding suffix for the type.</span></span> <span data-ttu-id="07ef1-106">Weitere Informationen zu Konvertierungen finden Sie unter [Umwandlung und Typkonvertierungen](../../csharp/programming-guide/types/casting-and-type-conversions.md).</span><span class="sxs-lookup"><span data-stu-id="07ef1-106">For more information on conversions, see [Casting and Type Conversions](../../csharp/programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="07ef1-107">Im folgenden Beispiel wird CS0664 generiert:</span><span class="sxs-lookup"><span data-stu-id="07ef1-107">The following sample generates CS0664:</span></span>  
  
```csharp  
// CS0664.cs  
class Example  
{  
    static void Main()  
    {  
        decimal d1 = 1.0;   // CS0664, because 1.0 is interpreted  
                            // as a double.  
  
        // Try the following line instead.  
        decimal d2 = 1.0M;  // The M tells the compiler that 1.0 is a  
                            // decimal.  
        Console.WriteLine(d2);  
    }  
}  
```
