# <a name="compiler-error-cs0127"></a>Compilerfehler CS0127
Da 'Funktion' 'void' zurückgibt, darf auf ein Rückgabeschlüsselwort kein Objektausdruck folgen.  
  
 Eine Methode mit dem Rückgabetyp [void](../language-reference/keywords/void.md) kann keinen Wert zurückgeben. Weitere Informationen finden Sie unter [Methoden](../programming-guide/classes-and-structs/methods.md).  
  
 Im folgenden Beispiel wird CS0127 generiert:  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
