# Compilerfehler CS0155
Der aufgefangene oder ausgelöste Typ muss von System.Exception abgeleitet werden.  
  
 Es wurde versucht, einen Datentyp, der nicht von **System.Exception** abgeleitet wurde, an einen [catch](../../csharp/language-reference/keywords/try-catch.md)\-Block zu übergeben. An einen **catch**\-Block können nur Datentypen übergeben werden, die von **System.Exception** abgeleitet wurden. Weitere Informationen finden Sie unter [Ausnahmebehandlungsanweisungen](../../csharp/language-reference/keywords/exception-handling-statements.md) und [Ausnahmen und Ausnahmebehandlung](../../csharp/programming-guide/exceptions/exceptions-and-exception-handling.md).  
  
 Im folgenden Beispiel wird CS0155 generiert:  
  
```  
// CS0155.cs using System; namespace MyNamespace { public class MyClass2 // try the following line instead // public class MyClass2 : Exception { } public class MyClass { public static void Main() { try { } catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception { } } } }  
```
