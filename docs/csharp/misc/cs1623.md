# <a name="compiler-error-cs1623"></a><span data-ttu-id="ecbd6-102">Compilerfehler CS1623</span><span class="sxs-lookup"><span data-stu-id="ecbd6-102">Compiler Error CS1623</span></span>
<span data-ttu-id="ecbd6-103">Iteratoren können keine ref oder out-Parameter haben.</span><span class="sxs-lookup"><span data-stu-id="ecbd6-103">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="ecbd6-104">Dieser Fehler tritt auf, wenn eine Iteratormethode verwendet eine `in`, `ref`, oder `out` Parameter.</span><span class="sxs-lookup"><span data-stu-id="ecbd6-104">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="ecbd6-105">Um diesen Fehler zu vermeiden, entfernen Sie die `in`, `ref`, oder `out` -Schlüsselwort aus der Signatur der Methode.</span><span class="sxs-lookup"><span data-stu-id="ecbd6-105">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ecbd6-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="ecbd6-106">Example</span></span>  
 <span data-ttu-id="ecbd6-107">Im folgenden Beispiel wird CS1623 generiert:</span><span class="sxs-lookup"><span data-stu-id="ecbd6-107">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
