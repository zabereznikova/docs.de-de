 <span data-ttu-id="711bf-107">Im folgenden Beispiel wird CS0505 generiert:</span><span class="sxs-lookup"><span data-stu-id="711bf-107">The following sample generates CS0505:</span></span>  
  
```  
// CS0505.cs

{  
   public int i;  
}

   public override int i() { return 0; }   // CS0505  
}  
```
