# <a name="compiler-error-cs0170"></a>Compilerfehler CS0170
Verwendung des möglicherweise nicht zugewiesenen Felds 'Feld'.  
  
 Ein Feld in einer Struktur wurde ohne vorherige Initialisierung verwendet. Ermitteln Sie zunächst, welches Feld nicht initialisiert wurde, und initialisieren Sie es anschließend, bevor Sie versuchen, darauf zuzugreifen, um dieses Problem zu beheben. Weitere Informationen zum Initialisieren von Strukturen finden Sie unter [Strukturen](../programming-guide/classes-and-structs/structs.md) und [Verwenden von Strukturen](../programming-guide/classes-and-structs/using-structs.md).  
  
 Im folgenden Beispiel wird CS0170 generiert:  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```
