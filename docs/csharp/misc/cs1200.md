# <a name="compiler-warning-level-1-cs1200"></a>Compilerwarnung (Stufe 1) CS1200
Das Feature „Ungültiges Feature“ ist veraltet. Verwenden Sie stattdessen 'gültige Funktion'.  
  
 Die Funktion, die Sie verwenden möchten, ist veraltet. Aktualisieren Sie den Code, um stattdessen die gültige Funktion zu verwenden.
