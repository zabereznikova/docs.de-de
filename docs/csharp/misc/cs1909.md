## <a name="example"></a><span data-ttu-id="b1a20-105">Beispiel</span><span class="sxs-lookup"><span data-stu-id="b1a20-105">Example</span></span>

```  
// CS1909.cs

public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909

```
