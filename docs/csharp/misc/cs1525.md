# <a name="compiler-error-cs1525"></a><span data-ttu-id="66fd8-102">Compilerfehler CS1525</span><span class="sxs-lookup"><span data-stu-id="66fd8-102">Compiler Error CS1525</span></span>
<span data-ttu-id="66fd8-103">Ungültiger Ausdruck "character".</span><span class="sxs-lookup"><span data-stu-id="66fd8-103">Invalid expression term 'character'</span></span>  
  
 <span data-ttu-id="66fd8-104">Der Compiler hat ein ungültiges Zeichen in einem Ausdruck erkannt.</span><span class="sxs-lookup"><span data-stu-id="66fd8-104">The compiler detected an invalid character in an expression.</span></span>  
  
 <span data-ttu-id="66fd8-105">Im folgenden Beispiel wird CS1525 generiert:</span><span class="sxs-lookup"><span data-stu-id="66fd8-105">The following sample generates CS1525:</span></span>  
  
```  
// CS1525.cs  
class x  
{  
   public static void Main()  
   {  
      int i = 0;  
      i = i +   // OK - identifier  
      'c' +     // OK - character  
      (5) +     // OK - parenthesis  
      [ +       // CS1525, operator not a valid expression element  
      throw +   // CS1525, keyword not allowed in expression  
      void;     // CS1525, void not allowed in expression  
   }  
}  
```  
  
 <span data-ttu-id="66fd8-106">Eine leere Bezeichnung kann auch CS1525 generieren, wie im folgenden Beispiel gezeigt:</span><span class="sxs-lookup"><span data-stu-id="66fd8-106">An empty label can also generate CS1525, as in the following sample:</span></span>  
  
```  
// CS1525b.cs  
using System;  
public class MyClass  
{  
   public static void Main()  
   {  
      goto FoundIt;  
      FoundIt:      // CS1525  
      // Uncomment the following line to resolve:  
      // System.Console.Write("Hello");  
   }  
}  
```
