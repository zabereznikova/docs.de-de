<span data-ttu-id="22852-104">Der Compiler hat eine Überschreibung der erkannt. die <xref:System.Object.Equals%2A?displayProperty=nameWithType> -Methode, aber keine Überschreibung, der die <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> Methode.</span><span class="sxs-lookup"><span data-stu-id="22852-104">The compiler detected an override of the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method but no override of the <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="22852-105">Eine Überschreibung der <xref:System.Object.Equals%2A> bedeutet, dass Sie auch außer Kraft setzen möchten <xref:System.Object.GetHashCode%2A>.</span><span class="sxs-lookup"><span data-stu-id="22852-105">An override of <xref:System.Object.Equals%2A> implies that you also want to override <xref:System.Object.GetHashCode%2A>.</span></span>

{  
   public override bool Equals(object o) { return true; }   // CS0659
