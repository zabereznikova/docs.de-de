 <span data-ttu-id="b4c73-104">Es ist nicht möglich, eine [partielle](../../csharp/language-reference/keywords/partial-method.md) Methode zu deklarieren, sofern diese nicht in einer partiellen Klasse oder Struktur gekapselt ist.</span><span class="sxs-lookup"><span data-stu-id="b4c73-104">It is not possible to declare a [partial](../../csharp/language-reference/keywords/partial-method.md) method unless it is encapsulated inside a partial class or partial struct.</span></span>
