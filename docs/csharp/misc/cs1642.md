## <a name="example"></a><span data-ttu-id="cfb90-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="cfb90-106">Example</span></span>  
 <span data-ttu-id="cfb90-107">Im folgenden Beispiel wird CS1642 generiert:</span><span class="sxs-lookup"><span data-stu-id="cfb90-107">The following sample generates CS1642.</span></span>  
  
```csharp  
// CS1642.cs  
// compile with: /unsafe /target:library  
unsafe class C  
{  
   fixed int a[10];   // CS1642  
}  
  
unsafe struct D  
{  
    fixed int a[10];  
}  
  
unsafe class E  
{  
   public int[] a = null;  
}  
```
