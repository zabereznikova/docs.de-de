## <a name="example"></a><span data-ttu-id="d7afc-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="d7afc-108">Example</span></span>  
 <span data-ttu-id="d7afc-109">Im folgenden Beispiel wird CS1930 generiert, da der Bezeichner `num` für die Bereichsvariable in der `from` -Klausel und für die durch die `let` -Klausel eingeführten Bereichsvariable verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="d7afc-109">The following example generates CS1930 because the identifier `num` is used for the range variable in the `from` clause and for the range variable introduced by the `let` clause.</span></span>  
  
```  
// cs1930.cs  
using System.Linq;  
class Program  
{  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        var query = from num in nums  
                    let num = 3 // CS1930  
                    select num;   
    }  
}  
```
