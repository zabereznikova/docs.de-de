# <a name="compiler-error-cs0447"></a><span data-ttu-id="9787a-102">Compilerfehler CS0447</span><span class="sxs-lookup"><span data-stu-id="9787a-102">Compiler Error CS0447</span></span>
<span data-ttu-id="9787a-103">Attribute können nicht für Typargumente verwendet werden, sondern nur für Typparameter.</span><span class="sxs-lookup"><span data-stu-id="9787a-103">Attributes cannot be used on type arguments, only on type parameters</span></span>  
  
 <span data-ttu-id="9787a-104">Dieser Fehler tritt auf, wenn Sie ein Attribut auf ein Typargument anwenden, das in einer Aufrufanweisung auftritt.</span><span class="sxs-lookup"><span data-stu-id="9787a-104">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="9787a-105">Es ist zulässig, ein Attribut auf einen Typparameter in einer Klassen- oder Methodendeklarationsanweisung wie der folgenden anzuwenden:</span><span class="sxs-lookup"><span data-stu-id="9787a-105">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>  
  
```

```  
  
 <span data-ttu-id="9787a-106">Dieser Fehler wird durch die folgende Codezeile generiert.</span><span class="sxs-lookup"><span data-stu-id="9787a-106">The following line of code will generate this error.</span></span> <span data-ttu-id="9787a-107">Es wird davon ausgegangen, dass die in der vorherigen Codezeile definierte Klasse `C`eine statische Methode namens `MyStaticMethod`enthält.</span><span class="sxs-lookup"><span data-stu-id="9787a-107">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>  
  
```  
C<[some attribute] T>.MyStaticMethod();  
```  
  
## <a name="example"></a><span data-ttu-id="9787a-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="9787a-108">Example</span></span>  
 <span data-ttu-id="9787a-109">Der folgende Code generiert den Fehler CS0447.</span><span class="sxs-lookup"><span data-stu-id="9787a-109">The following code generates error CS0447.</span></span>  
  
```  
// CS0447.cs  
using System;

{  
    public interface I<A>   
    {  
        void Meth<B>();  
    }  
    public class B : I<int>   
    {  
        void I<[Test] int>.Meth<X>() { }  // CS0447  
    }  
}  
```
