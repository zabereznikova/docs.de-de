# <a name="compiler-error-cs1577"></a>Compilerfehler CS1577
Fehler beim Generieren der Assembly – Grund  
  
 Nach der erfolgreichen Kompilierung des Quellcodes durch den C#-Compiler ist beim Generieren der Assembly ein Fehler aufgetreten. Finden Sie in der Fehlerdokumentation für das ALink-Dienstprogramm ([Al.exe Tool Errors and Warnings](http://msdn.microsoft.com/library/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)) Weitere Informationen.
