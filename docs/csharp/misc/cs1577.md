# <a name="compiler-error-cs1577"></a>Compilerfehler CS1577
Fehler beim Generieren der Assembly – Grund  
  
 Nach der erfolgreichen Kompilierung des Quellcodes durch den C#-Compiler ist beim Generieren der Assembly ein Fehler aufgetreten. Weitere Informationen finden Sie in der Fehlerdokumentation für das ALink-Dienstprogramm ([Fehler und Warnungen des Al.exe-Tools](../../framework/tools/al-exe-assembly-linker.md#errors-and-warnings)).
