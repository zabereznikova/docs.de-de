<span data-ttu-id="aeb09-104">Der Compiler gibt eine Warnung der Stufe 3 aus, wenn Sie eine Variable deklarieren, aber nicht verwenden.</span><span class="sxs-lookup"><span data-stu-id="aeb09-104">The compiler issues a level-three warning when you declare a variable, but do not use it.</span></span>
  
<span data-ttu-id="aeb09-105">Im folgende Beispiel wird eine Warnung CS0168 generiert:</span><span class="sxs-lookup"><span data-stu-id="aeb09-105">The following sample generates a CS0168 warning:</span></span>  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
