# <a name="compiler-error-cs0601"></a><span data-ttu-id="f93d8-103">Compilerfehler CS0601</span><span class="sxs-lookup"><span data-stu-id="f93d8-103">Compiler Error CS0601</span></span>

<span data-ttu-id="f93d8-104">Das DllImport-Attribut muss für eine Methode angegeben werden, die als "static" und "extern" markiert ist.</span><span class="sxs-lookup"><span data-stu-id="f93d8-104">The DllImport attribute must be specified on a method marked 'static' and 'extern'</span></span>  
  
 <span data-ttu-id="f93d8-105">Das `DllImport` -Attribut wurde für eine Methode verwendet, die nicht über die richtigen Zugriffsschlüsselwörter verfügt.</span><span class="sxs-lookup"><span data-stu-id="f93d8-105">The `DllImport` attribute was used on a method that did not have the correct access keywords.</span></span>  
  
 <span data-ttu-id="f93d8-106">Im folgenden Beispiel wird CS0601 generiert:</span><span class="sxs-lookup"><span data-stu-id="f93d8-106">The following sample generates CS0601:</span></span>  
  
```csharp  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
