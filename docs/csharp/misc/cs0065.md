# <a name="compiler-error-cs0065"></a><span data-ttu-id="13fb2-102">Compilerfehler CS0065</span><span class="sxs-lookup"><span data-stu-id="13fb2-102">Compiler Error CS0065</span></span>

<span data-ttu-id="13fb2-103">„event“: Die Ereigniseigenschaft muss sowohl add- als auch remove-Accessoren besitzen.</span><span class="sxs-lookup"><span data-stu-id="13fb2-103">'event' : event property must have both add and remove accessors</span></span>

<span data-ttu-id="13fb2-104">Ein Ereignis, das kein Feld ist, muss beide Zugriffsmethoden unterstützen.</span><span class="sxs-lookup"><span data-stu-id="13fb2-104">An event that is not a field must have both access methods.</span></span>

<span data-ttu-id="13fb2-105">Im folgenden Beispiel wird CS0065 generiert:</span><span class="sxs-lookup"><span data-stu-id="13fb2-105">The following sample generates CS0065:</span></span>

```csharp
// CS0065.cs
using System;
public delegate void EventHandler(object sender, int e);
public class MyClass
{
   public event EventHandler Click   // CS0065,
   {
      // to fix, uncomment the add and remove definitions
      /*
      add
      {
         Click += value;
      }
      remove
      {
         Click -= value;
      }
      */
   }

   public static void Main()
   {
   }
}
```
