# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="35422-102">Compilerwarnung (Stufe 1) CS0684</span><span class="sxs-lookup"><span data-stu-id="35422-102">Compiler Warning (level 1) CS0684</span></span>
<span data-ttu-id="35422-103">Die mit „CoClassAttribute“ markierte interface-Schnittstelle wurde nicht mit „ComImportAttribute“ markiert.</span><span class="sxs-lookup"><span data-stu-id="35422-103">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="35422-104">Wenn Sie **CoClassAttribute** in einer Schnittstelle angeben, müssen Sie auch **ComImportAttribute**angeben.</span><span class="sxs-lookup"><span data-stu-id="35422-104">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="35422-105">Im folgenden Beispiel wird CS0684 generiert:</span><span class="sxs-lookup"><span data-stu-id="35422-105">The following sample generates CS0684:</span></span>  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
