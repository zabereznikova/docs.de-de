## <a name="example"></a><span data-ttu-id="d5eee-105">Beispiel</span><span class="sxs-lookup"><span data-stu-id="d5eee-105">Example</span></span>  
 <span data-ttu-id="d5eee-106">Im folgenden Beispiel wird CS0448 generiert:</span><span class="sxs-lookup"><span data-stu-id="d5eee-106">The following sample generates CS0448.</span></span>  
  
```csharp

{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK

}  
```

   public static S? operator ++(S s) { return new S(); }   // CS0448

   public static T? operator ++(T? t) { return new T(); }
