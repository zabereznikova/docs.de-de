# <a name="compiler-error-cs0043"></a>Compilerfehler CS0043
Die PDB-Datei "Datei" hat ein falsches oder veraltetes Format. Löschen Sie sie, und erstellen Sie sie neu.  
  
 Löschen Sie die PDB-Datei für diese Kompilierung, und kompilieren Sie neu.
