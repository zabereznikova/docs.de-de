# <a name="compiler-error-cs0175"></a><span data-ttu-id="4b9e2-102">Compilerfehler CS0175</span><span class="sxs-lookup"><span data-stu-id="4b9e2-102">Compiler Error CS0175</span></span>
<span data-ttu-id="4b9e2-103">Die Verwendung des base-Schlüsselworts ist in diesem Kontext nicht gültig.</span><span class="sxs-lookup"><span data-stu-id="4b9e2-103">Use of keyword 'base' is not valid in this context</span></span>  
  
 <span data-ttu-id="4b9e2-104">Zum Angeben eines bestimmten Members der Basisklasse muss das [base](../language-reference/keywords/base.md) -Schlüsselwort muss verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="4b9e2-104">The [base](../language-reference/keywords/base.md) keyword must be used to specify a particular member of the base class.</span></span> <span data-ttu-id="4b9e2-105">Weitere Informationen finden Sie unter [Konstruktoren](../programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="4b9e2-105">For more information, see [Constructors](../programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="4b9e2-106">Im folgenden Beispiel wird CS0175 generiert:</span><span class="sxs-lookup"><span data-stu-id="4b9e2-106">The following sample generates CS0175:</span></span>  
  
```csharp  
// CS0175.cs  
using System;  
class BaseClass  
{  
    public int TestInt = 0;  
}  
  
class MyClass : BaseClass  
{  
    public static void Main()  
    {  
        MyClass aClass = new MyClass();  
        aClass.BaseTest();  
    }  
  
    public void BaseTest()  
    {  
        Console.WriteLine(base); // CS0175  
        // Try the following line instead:  
        // Console.WriteLine(base.TestInt);  
        base = 9;   // CS0175  
  
        // Try the following line instead:  
        // base.TestInt = 9;  
    }  
}  
```
