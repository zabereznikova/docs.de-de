# Compilerfehler CS2015
'Datei' ist eine Binärdatei und keine Textdatei.  
  
 Eine Datei wurde an den Compiler übergeben, bei der es sich um eine binäre Datei handelt. Der Compiler erwartet eine Quellcodedatei.
