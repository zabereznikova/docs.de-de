## <a name="example"></a><span data-ttu-id="44c91-109">Beispiel</span><span class="sxs-lookup"><span data-stu-id="44c91-109">Example</span></span>  
 <span data-ttu-id="44c91-110">Der folgende Code generiert CS0737:</span><span class="sxs-lookup"><span data-stu-id="44c91-110">The following code generates CS0737:</span></span>  
  
```csharp  
// cs0737.cs  
interface ITest  
{  
    // Default access of private with no modifier.  
    int Return42();  
    // Try the following line instead.  
    // public int Return42();  
}  
  
struct Struct1 : ITest // CS0737  
{  
    int Return42() { return (42); }  
}  
  
public class Test  
{  
    public static int Main(string[] args)  
    {  
        Struct1 s1 = new Struct1();  
  
        return (1);  
    }  
  
}  
```
