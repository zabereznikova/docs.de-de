# <a name="compiler-error-cs0009"></a>Compilerfehler CS0009
Die Metadatendatei 'datei' konnte nicht geöffnet werden – 'beschreibung'  
  
 Die mit der Compileroption [/reference](../language-reference/compiler-options/reference-compiler-option.md) angegebene Datei enthält keine gültigen Metadaten.
