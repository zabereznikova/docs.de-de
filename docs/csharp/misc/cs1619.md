# <a name="compiler-error-cs1619"></a>Compilerfehler CS1619
Temporäre Datei "Dateiname" kann nicht erstellt werden: "Grund"  
  
 Der Compiler konnte aus dem genannten Grund (z. B. Datenträger ist voll) die temporäre Datei nicht erstellen.
