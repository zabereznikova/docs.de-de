 <span data-ttu-id="fa600-104">Eine Bezeichnung wurde in einer [switch](../../csharp/language-reference/keywords/switch.md) -Anweisung wiederholt.</span><span class="sxs-lookup"><span data-stu-id="fa600-104">A label was repeated in a [switch](../../csharp/language-reference/keywords/switch.md) statement.</span></span> <span data-ttu-id="fa600-105">Weitere Informationen finden Sie unter [switch](../../csharp/language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="fa600-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="fa600-106">Im folgenden Beispiel wird CS0152 generiert:</span><span class="sxs-lookup"><span data-stu-id="fa600-106">The following sample generates CS0152:</span></span>  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
