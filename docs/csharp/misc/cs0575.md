 Eine [Struktur](../../csharp/language-reference/keywords/struct.md) kann keinen Destruktor enthalten.  
  
 Im folgenden Beispiel wird CS0575 generiert:  
  
```csharp  
// CS0575.cs

{  
   public struct iii  
   {  
      ~iii()   // CS0575  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
