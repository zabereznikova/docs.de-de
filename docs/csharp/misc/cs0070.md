 <span data-ttu-id="4e3e2-104">Au√üerhalb der Klasse, in der es definiert ist, kann ein [Ereignis](../language-reference/keywords/event.md) Verweise nur addieren oder subtrahieren.</span><span class="sxs-lookup"><span data-stu-id="4e3e2-104">Outside of the class it is defined in, an [event](../language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="4e3e2-105">Weitere Informationen finden Sie unter [Ereignisse](../programming-guide/events/index.md)definiert sind.</span><span class="sxs-lookup"><span data-stu-id="4e3e2-105">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="4e3e2-106">Im folgenden Beispiel wird CS0070 generiert:</span><span class="sxs-lookup"><span data-stu-id="4e3e2-106">The following sample generates CS0070:</span></span>  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
