## <a name="example"></a><span data-ttu-id="f7afb-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="f7afb-106">Example</span></span>

```csharp  
   public static implicit operator int(iii x)  
   {  
      return 0;  
   }

   {  
      return null;  
   }  
  
   public static int operator ++(int aa)   // CS0559

   // public static iii operator ++(iii aa)

   }

```

 <span data-ttu-id="f7afb-109">Im folgenden Beispiel wird CS0559 generiert.</span><span class="sxs-lookup"><span data-stu-id="f7afb-109">The following sample generates CS0559.</span></span>  
  
```csharp  
// CS0559_b.cs

public struct S  
{  
   public static S operator ++(S? s) { return new S(); }   // CS0559

public struct T  
{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
}  
```
