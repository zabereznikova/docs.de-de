## <a name="example"></a><span data-ttu-id="85265-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="85265-106">Example</span></span>

```csharp
// CS0136.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         {  
            char i = 'a';   // CS0136, hides int i  
         }  
         i++;  
      }  
   }

```
