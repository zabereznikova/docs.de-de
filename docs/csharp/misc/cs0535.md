## <a name="example"></a><span data-ttu-id="24f23-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="24f23-106">Example</span></span>  
 <span data-ttu-id="24f23-107">Im folgenden Beispiel wird CS0535 generiert:</span><span class="sxs-lookup"><span data-stu-id="24f23-107">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
