 <span data-ttu-id="f7d0e-106">Im folgenden Beispiel wird CS1641 generiert:</span><span class="sxs-lookup"><span data-stu-id="f7d0e-106">The following sample generates CS1641:</span></span>  
  
```csharp

unsafe struct S {  
   fixed int [] a;  // CS1641

   fixed int b [10];

}  
```
