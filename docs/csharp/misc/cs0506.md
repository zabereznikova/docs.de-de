# <a name="compiler-error-cs0506"></a><span data-ttu-id="ceaa4-102">Compilerfehler CS0506</span><span class="sxs-lookup"><span data-stu-id="ceaa4-102">Compiler Error CS0506</span></span>
<span data-ttu-id="ceaa4-103">"Funktion1": Der geerbte Member "Funktion2" kann nicht überschrieben werden, da er nicht als "virtual" , "abstract" oder "override" markiert ist.</span><span class="sxs-lookup"><span data-stu-id="ceaa4-103">'function1' : cannot override inherited member 'function2' because it is not marked "virtual", "abstract", or "override"</span></span>  
  
 <span data-ttu-id="ceaa4-104">Eine Methode wurde überschrieben, die nicht explizit als **virtual**, **abstract**oder `override`markiert ist.</span><span class="sxs-lookup"><span data-stu-id="ceaa4-104">A method was overridden that was not explicitly marked as **virtual**, **abstract**, or `override`.</span></span>  
  
 <span data-ttu-id="ceaa4-105">Im folgenden Beispiel wird CS0506 generiert:</span><span class="sxs-lookup"><span data-stu-id="ceaa4-105">The following sample generates CS0506:</span></span>  
  
```csharp  
// CS0506.cs  
namespace MyNameSpace  
{  
   abstract public class ClassX  
   {  
      public int i = 0;  
  
      public int f()  
      {  
         return 0;  
      }  
      // Try the following definition for f() instead:  
      // abstract public int f();  
   }  
  
   public class ClassY : ClassX  
   {  
      public override int f()   // CS0506  
      {  
         return 0;  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
