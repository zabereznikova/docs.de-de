# <a name="compiler-error-cs2020"></a>Compilerfehler CS2020
Nur die erste Gruppe von Eingabedateien kann ein anderes Ziel als 'module' erstellen.  
  
 Bei einer Kompilierung mit mehreren Ausgabedateien muss die erste Ausgabedatei mit [/target:exe](../../csharp/language-reference/compiler-options/target-exe-compiler-option.md), [/target:winexe](../../csharp/language-reference/compiler-options/target-winexe-compiler-option.md)oder [/target:library](../../csharp/language-reference/compiler-options/target-library-compiler-option.md)erstellt werden.
