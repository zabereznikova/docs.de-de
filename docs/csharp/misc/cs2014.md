# Compilerwarnung (Stufe 1) CS2014
Die Compileroption "alte Option" ist veraltet, verwenden Sie stattdessen "neue Option".  
  
 Die Form der Compileroption ist veraltet. Weitere Informationen zu Compileroptionen finden Sie unter [C\#\-Compileroptionen](../../csharp/language-reference/compiler-options/index.md).
