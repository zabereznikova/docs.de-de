# <a name="compiler-error-cs0572"></a><span data-ttu-id="89c2b-102">Compilerfehler CS0572</span><span class="sxs-lookup"><span data-stu-id="89c2b-102">Compiler Error CS0572</span></span>
<span data-ttu-id="89c2b-103">'Typ': Auf einen Typ kann nicht durch einen Ausdruck verwiesen werden. Verwenden Sie stattdessen 'Pfad_zu_Typ'.</span><span class="sxs-lookup"><span data-stu-id="89c2b-103">'type' : cannot reference a type through an expression; try 'path_to_type' instead</span></span>  
  
 <span data-ttu-id="89c2b-104">Es wurde versucht, über einen Bezeichner auf einen Member einer Klasse zuzugreifen, der in dieser Situation nicht zulässig ist.</span><span class="sxs-lookup"><span data-stu-id="89c2b-104">An attempt was made to access a member of a class through an identifier, which is not permitted in this situation.</span></span>  
  
 <span data-ttu-id="89c2b-105">Im folgenden Beispiel wird CS0572 generiert:</span><span class="sxs-lookup"><span data-stu-id="89c2b-105">The following sample generates CS0572:</span></span>  
  
```csharp  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
