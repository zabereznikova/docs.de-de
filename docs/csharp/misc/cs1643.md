 <span data-ttu-id="43c24-107">Im folgenden Beispiel wird CS1643 generiert:</span><span class="sxs-lookup"><span data-stu-id="43c24-107">The following sample generates CS1643:</span></span>

```csharp  
// CS1643.cs

{  
    static void Main()  
    {  
        MyDelegate d = delegate  
        {                 // CS1643  
            int i = 0;  
            if (i == 0)  
                return 1;  
        };  
    }  
}  
```
