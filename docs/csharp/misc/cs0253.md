 <span data-ttu-id="325b5-106">Im folgenden Beispiel wird CS0253 generiert:</span><span class="sxs-lookup"><span data-stu-id="325b5-106">The following sample generates CS0253:</span></span>  
  
```csharp 
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
