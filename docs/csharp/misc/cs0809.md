# <a name="compiler-warning-level-1-cs0809"></a>Compilerwarnung (Stufe 1) CS0809

 In der Regel sollte ein als veraltet gekennzeichneter Member keinen Member 端berschreiben, der nicht als veraltet gekennzeichnet ist. Diese Warnung wird in [!INCLUDE[vs_orcas_long](~/includes/vs-orcas-long-md.md)] , jedoch nicht in [!INCLUDE[vsprvslong](~/includes/vsprvslong-md.md)]generiert.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Entfernen Sie das `Obsolete` -Attribut aus dem 端berschreibenden Member, oder f端gen Sie es zum Basisklassenmember hinzu.  
  
## <a name="example"></a>Beispiel  
  
```  
// CS0809.cs  
public class Base  
{  
    public virtual void Test1()

    }

    {  
    }
```
