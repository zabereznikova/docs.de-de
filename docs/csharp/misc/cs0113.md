# <a name="compiler-error-cs0113"></a><span data-ttu-id="23744-102">Compilerfehler CS0113</span><span class="sxs-lookup"><span data-stu-id="23744-102">Compiler Error CS0113</span></span>
<span data-ttu-id="23744-103">Ein Member "Funktion", der als "override" markiert ist, kann nicht als "new" oder "virtual" markiert werden.</span><span class="sxs-lookup"><span data-stu-id="23744-103">A member 'function' marked as override cannot be marked as new or virtual</span></span>  
  
 <span data-ttu-id="23744-104">Die zum Markieren einer Methode verwendeten Schlüsselwörter [new](../../csharp/language-reference/keywords/new-modifier.md) und [override](../../csharp/language-reference/keywords/override.md) schließen sich gegenseitig aus.</span><span class="sxs-lookup"><span data-stu-id="23744-104">It is mutually exclusive to mark a method with the [new](../../csharp/language-reference/keywords/new-modifier.md) and [override](../../csharp/language-reference/keywords/override.md) keywords.</span></span>  
  
 <span data-ttu-id="23744-105">Im folgenden Beispiel wird CS0113 generiert:</span><span class="sxs-lookup"><span data-stu-id="23744-105">The following sample generates CS0113:</span></span>  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
