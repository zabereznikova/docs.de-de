# <a name="compiler-error-cs0156"></a><span data-ttu-id="d1fcf-102">Compilerfehler CS0156</span><span class="sxs-lookup"><span data-stu-id="d1fcf-102">Compiler Error CS0156</span></span>
<span data-ttu-id="d1fcf-103">In einer finally-Klausel, die in der nächsten einschließenden catch-Klausel geschachtelt ist, ist keine throw-Anweisung ohne Argumente zulässig.</span><span class="sxs-lookup"><span data-stu-id="d1fcf-103">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>  
  
 <span data-ttu-id="d1fcf-104">Eine [throw](../../csharp/language-reference/keywords/throw.md) -Anweisung ohne Parameter kann nur in einer **catch** -Klausel vorkommen, die keine Parameter annimmt.</span><span class="sxs-lookup"><span data-stu-id="d1fcf-104">A [throw](../../csharp/language-reference/keywords/throw.md) statement with no parameters can only appear in a **catch** clause that takes no parameters.</span></span>  
  
 <span data-ttu-id="d1fcf-105">Weitere Informationen finden Sie unter [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) und [Ausnahmen und Ausnahmebehandlung](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="d1fcf-105">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="d1fcf-106">Im folgenden Beispiel wird CS0156 generiert:</span><span class="sxs-lookup"><span data-stu-id="d1fcf-106">The following sample generates CS0156:</span></span>  
  
```csharp  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```
