## <a name="example"></a><span data-ttu-id="26ec5-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="26ec5-106">Example</span></span>  
 <span data-ttu-id="26ec5-107">Im folgenden Beispiel wird CS0550 generiert:</span><span class="sxs-lookup"><span data-stu-id="26ec5-107">The following sample generates CS0550.</span></span>  
  
```csharp  
// CS0550.cs  
namespace x  
{  
   interface ii  
   {  
      int i  
      {  
         get;  
         // add the following accessor to resolve this CS0550  
         // set;  
      }  
   }  
  
   public class a : ii  
   {  
      int ii.i  
      {  
         get  
         {  
            return 0;  
         }  
         set {}   // CS0550  no set in interface  
      }  
  
      public static void Main() {}  
   }  
}  
```
