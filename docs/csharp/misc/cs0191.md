## <a name="example"></a><span data-ttu-id="3391f-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="3391f-107">Example</span></span>  
 <span data-ttu-id="3391f-108">Im folgenden Beispiel wird CS0191 generiert:</span><span class="sxs-lookup"><span data-stu-id="3391f-108">The following sample generates CS0191.</span></span>  
  
```csharp  
// CS0191.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  // OK to assign to readonly field in declaration  
  
    MyClass()  
    {  
        TestInt = 11; // OK to assign to readonly field in constructor  
    }  
  
    public void TestReadOnly()  
    {  
        TestInt = 19;                  // CS0191  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
