# <a name="compiler-warning-level-2-cs0251"></a><span data-ttu-id="466cb-102">Compilerwarnung (Stufe 2) CS0251</span><span class="sxs-lookup"><span data-stu-id="466cb-102">Compiler Warning (level 2) CS0251</span></span>
<span data-ttu-id="466cb-103">Indizierung eines Arrays mit einem negativen Index (Arrayindizes starten immer mit Null).</span><span class="sxs-lookup"><span data-stu-id="466cb-103">Indexing an array with a negative index (array indices always start at zero)</span></span>  
  
 <span data-ttu-id="466cb-104">Verwenden Sie keine negative Zahl zum Indizieren eines Arrays.</span><span class="sxs-lookup"><span data-stu-id="466cb-104">Do not use a negative number to index into an array.</span></span>  
  
 <span data-ttu-id="466cb-105">Im folgenden Beispiel wird CS0251 generiert:</span><span class="sxs-lookup"><span data-stu-id="466cb-105">The following sample generates CS0251:</span></span>  
  
```  
// CS0251.cs  
// compile with: /W:2  
class MyClass  
{  
   public static void Main()  
   {  
      int[] myarray = new int[] {1,2,3};     
      try  
      {  
         myarray[-1]++;   // CS0251  
         // try the following line instead  
         // myarray[1]++;  
      }  
      catch (System.IndexOutOfRangeException e)  
      {  
         System.Console.WriteLine("{0}", e);  
      }  
   }  
}  
```
