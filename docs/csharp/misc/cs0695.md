# <a name="compiler-error-cs0695"></a><span data-ttu-id="7b0a2-102">Compilerfehler CS0695</span><span class="sxs-lookup"><span data-stu-id="7b0a2-102">Compiler Error CS0695</span></span>
<span data-ttu-id="7b0a2-103">"Generischer Typ" kann nicht gleichzeitig "generische Schnittstelle" und "generische Schnittstelle" implementieren, da diese für einige Typparameterersetzungen zusammengeführt werden können.</span><span class="sxs-lookup"><span data-stu-id="7b0a2-103">'generic type' cannot implement both 'generic interface' and 'generic interface' because they may unify for some type parameter substitutions</span></span>  
  
 <span data-ttu-id="7b0a2-104">Dieser Fehler tritt auf, wenn eine generische Klasse mehrere Parametrisierungen derselben generischen Schnittstelle implementiert und eine Ersetzung von Typparametern vorhanden ist, durch die die beiden Schnittstellen als identisch festgelegt werden.</span><span class="sxs-lookup"><span data-stu-id="7b0a2-104">This error occurs when a generic class implements more than one parameterization of the same generic interface, and there exists a type parameter substitution which would make the two interfaces identical.</span></span> <span data-ttu-id="7b0a2-105">Implementieren Sie nur eine der Schnittstellen, um diesen Fehler zu vermeiden, oder ändern Sie die Typparameter, sodass kein Konflikt mehr vorliegt.</span><span class="sxs-lookup"><span data-stu-id="7b0a2-105">To avoid this error, implement only one of the interfaces, or change the type parameters to avoid the conflict.</span></span>  
  
 <span data-ttu-id="7b0a2-106">Im folgenden Beispiel wird CS0695 generiert:</span><span class="sxs-lookup"><span data-stu-id="7b0a2-106">The following sample generates CS0695:</span></span>  
  
```csharp  
// CS0695.cs  
// compile with: /target:library  
  
interface I<T>  
{  
}  
  
class G<T1, T2> : I<T1>, I<T2>  // CS0695  
{  
}  
```
