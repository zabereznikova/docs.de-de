# <a name="compiler-error-cs0744"></a><span data-ttu-id="88c81-102">Compilerfehler CS0744</span><span class="sxs-lookup"><span data-stu-id="88c81-102">Compiler Error CS0744</span></span>
<span data-ttu-id="88c81-103">Kontextbezogenes Schlüsselwort "equals" erwartet.</span><span class="sxs-lookup"><span data-stu-id="88c81-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="88c81-104">Das Muster für eine `join` -Klausel ist `join`...`in`...`on`...`equals`, wie im folgenden Beispiel gezeigt:</span><span class="sxs-lookup"><span data-stu-id="88c81-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="88c81-105">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="88c81-105">To correct this error</span></span>  
  
1. <span data-ttu-id="88c81-106">Fügen Sie der `equals` -Klausel das `join` Schlüsselwort hinzu.</span><span class="sxs-lookup"><span data-stu-id="88c81-106">Add the `equals` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="88c81-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="88c81-107">Example</span></span>  
 <span data-ttu-id="88c81-108">Mit dem folgenden Code wird CS0744 generiert:</span><span class="sxs-lookup"><span data-stu-id="88c81-108">The following code generates CS0744:</span></span>  
  
```csharp  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
