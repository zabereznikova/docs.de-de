# <a name="compiler-error-cs1678"></a>Compilerfehler CS1678
Parameter 'Nummer' ist als Typ 'Typ1' deklariert, muss jedoch 'Typ2' sein.  
  
 Dieser Fehler tritt auf, wenn der Parametertyp in einer anonymen Methode nicht mit der Deklaration des Delegaten Ã¼bereinstimmt, in den Sie die Methode umwandeln.  
  
 Im folgenden Beispiel wird CS1678 generiert:  
  
```csharp  
// CS1678  
delegate void D(int i);  
class Errors   
{  
   static void Main()   
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```
