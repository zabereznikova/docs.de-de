 <span data-ttu-id="52e35-104">Eine Bezeichnung in einem inneren Gültigkeitsbereich verdeckt eine Bezeichnung mit demselben Namen in einem äußeren Gültigkeitsbereich.</span><span class="sxs-lookup"><span data-stu-id="52e35-104">A label in an inner scope hides a label with the same name in an outer scope.</span></span> <span data-ttu-id="52e35-105">Weitere Informationen finden Sie unter [goto](../language-reference/keywords/goto.md).</span><span class="sxs-lookup"><span data-stu-id="52e35-105">For more information, see [goto](../language-reference/keywords/goto.md).</span></span>

      public static void Main()

         }  
      }  
   }  
}
