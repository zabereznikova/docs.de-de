# <a name="compiler-warning-level-1-cs0672"></a><span data-ttu-id="bd261-102">Compilerwarnung (Stufe 1) CS0672</span><span class="sxs-lookup"><span data-stu-id="bd261-102">Compiler Warning (level 1) CS0672</span></span>
<span data-ttu-id="bd261-103">Member "Member1" überschreibt den veralteten Member "Member2".</span><span class="sxs-lookup"><span data-stu-id="bd261-103">Member 'member1' overrides obsolete member 'member2.</span></span> <span data-ttu-id="bd261-104">Fügen Sie "Member1" das Obsolete-Attribut hinzu.</span><span class="sxs-lookup"><span data-stu-id="bd261-104">Add the Obsolete attribute to 'member1'</span></span>  
  
 <span data-ttu-id="bd261-105">Der Compiler hat `override` für eine Methode gefunden, die als `obsolete`markiert ist.</span><span class="sxs-lookup"><span data-stu-id="bd261-105">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="bd261-106">Die überschreibende Methode selbst wurde jedoch nicht als "obsolete" markiert.</span><span class="sxs-lookup"><span data-stu-id="bd261-106">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="bd261-107">Die überschreibende Methode löst beim Aufrufen immer den Fehler [CS0612](../../csharp/misc/cs0612.md)aus.</span><span class="sxs-lookup"><span data-stu-id="bd261-107">The overriding method will still generate [CS0612](../../csharp/misc/cs0612.md), if called.</span></span>  
  
 <span data-ttu-id="bd261-108">Überprüfen Sie die Methodendeklarationen, und geben Sie explizit an, ob eine Methode (und alle Überschreibungen) als `obsolete`markiert werden soll.</span><span class="sxs-lookup"><span data-stu-id="bd261-108">Review your method declarations and explicitly indicate whether a method (and all of its overrides) should be marked `obsolete`.</span></span>  
  
 <span data-ttu-id="bd261-109">Im folgenden Beispiel wird CS0672 generiert:</span><span class="sxs-lookup"><span data-stu-id="bd261-109">The following sample generates CS0672:</span></span>  
  
```  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```
