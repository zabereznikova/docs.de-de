# <a name="compiler-error-cs0077"></a><span data-ttu-id="07433-102">Compilerfehler CS0077</span><span class="sxs-lookup"><span data-stu-id="07433-102">Compiler Error CS0077</span></span>
<span data-ttu-id="07433-103">Der as-Operator muss mit einem Referenztyp oder einem Typ, der NULL-Werte zulässt, verwendet werden. ("int" ist ein Werttyp, der keine NULL-Werte zulässt.)</span><span class="sxs-lookup"><span data-stu-id="07433-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="07433-104">An den [as](../language-reference/operators/type-testing-and-cast.md#as-operator) -Operator wurde ein [Werttyp](../language-reference/keywords/value-types.md)übergeben.</span><span class="sxs-lookup"><span data-stu-id="07433-104">The [as](../language-reference/operators/type-testing-and-cast.md#as-operator) operator was passed a [value type](../language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="07433-105">Da `as` [null](../language-reference/keywords/null.md)zurückgeben kann, kann nur ein [Verweistyp](../language-reference/keywords/reference-types.md) oder ein [Werte zulässt-Werttyp](../language-reference/builtin-types/nullable-value-types.md)an ihn übermittelt werden.</span><span class="sxs-lookup"><span data-stu-id="07433-105">Because `as` can return [null](../language-reference/keywords/null.md), it can only be passed a [reference type](../language-reference/keywords/reference-types.md) or a [nullable value type](../language-reference/builtin-types/nullable-value-types.md).</span></span>
  
 <span data-ttu-id="07433-106">Im folgenden Beispiel wird CS0077 generiert:</span><span class="sxs-lookup"><span data-stu-id="07433-106">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
