 <span data-ttu-id="bc1fd-105">Im folgenden Beispiel wird CS1032 generiert:</span><span class="sxs-lookup"><span data-stu-id="bc1fd-105">The following sample generates CS1032:</span></span>

   {  
      #define a   // CS1032, put before namespace
