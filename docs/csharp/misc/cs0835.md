## <a name="example"></a><span data-ttu-id="0537b-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="0537b-108">Example</span></span>  
 <span data-ttu-id="0537b-109">Im folgenden Beispiel wird CS0835 generiert:</span><span class="sxs-lookup"><span data-stu-id="0537b-109">The following example generates CS0835:</span></span>

```csharp  
using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;

    }  
}  
```
