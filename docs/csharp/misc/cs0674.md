 <span data-ttu-id="0f808-105">Der C#-Compiler lässt keine für die Verwendung von <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; verwenden Sie [Params](../../csharp/language-reference/keywords/params.md) stattdessen.</span><span class="sxs-lookup"><span data-stu-id="0f808-105">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../../csharp/language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="0f808-106">Im folgenden Beispiel wird CS0674 generiert:</span><span class="sxs-lookup"><span data-stu-id="0f808-106">The following sample generates CS0674:</span></span>  
  
```csharp

{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead

         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
