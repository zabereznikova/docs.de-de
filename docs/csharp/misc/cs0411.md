## <a name="example"></a><span data-ttu-id="7c617-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="7c617-107">Example</span></span>  
 <span data-ttu-id="7c617-108">Im folgenden Beispiel wird CS0411 generiert:</span><span class="sxs-lookup"><span data-stu-id="7c617-108">The following sample generates CS0411:</span></span>

{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411

    }
