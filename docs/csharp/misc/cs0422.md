# <a name="compiler-warning-level-4-cs0422"></a>Compilerwarnung (Stufe 4) CS0422
Die Option '/incremental' wird nicht mehr unterstützt.  
  
 Inkrementelle Kompilierung (**INCR** oder **/ incremental**) wird nicht unterstützt C# 2.0.
