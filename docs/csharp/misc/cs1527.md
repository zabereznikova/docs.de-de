# <a name="compiler-error-cs1527"></a><span data-ttu-id="36ef9-102">Compilerfehler CS1527</span><span class="sxs-lookup"><span data-stu-id="36ef9-102">Compiler Error CS1527</span></span>
<span data-ttu-id="36ef9-103">In einem Namespace definierte Elementen nicht explizit deklariert werden als privat, geschützt, geschützt intern oder privat geschützt.</span><span class="sxs-lookup"><span data-stu-id="36ef9-103">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span> 
  
 <span data-ttu-id="36ef9-104">Typdeklarationen in einem Namespace können [öffentlichen](../../csharp/language-reference/keywords/public.md) oder [internen](../../csharp/language-reference/keywords/internal.md) Zugriff haben.</span><span class="sxs-lookup"><span data-stu-id="36ef9-104">Type declarations in a namespace can have either [public](../../csharp/language-reference/keywords/public.md) or [internal](../../csharp/language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="36ef9-105">Wenn keine Zugriffsart angegeben ist, wird standardmäßig der **interne** Zugriff verwendet.</span><span class="sxs-lookup"><span data-stu-id="36ef9-105">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="36ef9-106">Im folgenden Beispiel wird CS1527 generiert:</span><span class="sxs-lookup"><span data-stu-id="36ef9-106">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="36ef9-107">Im folgenden Beispiel wird CS1527 generiert, da sich alle Typdeklarationen implizit im globalen Namespace befinden, wenn kein Namespace explizit im Programmcode deklariert wird.</span><span class="sxs-lookup"><span data-stu-id="36ef9-107">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4{}  
private struct S1{}  
```
