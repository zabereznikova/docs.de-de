# Compilerfehler CS0585
Interner Compilerfehler: Stufe 'stufe'  
  
 Versuchen Sie festzustellen, ob der Compiler einen Fehler verursacht, weil er unerwartete Syntax nicht analysieren kann. Wenn dies nicht der Fall ist, versuchen Sie Folgendes: [Sprechen Sie mit uns](/visual-studio/ide/talk-to-us).
