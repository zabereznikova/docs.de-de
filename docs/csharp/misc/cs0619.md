# <a name="compiler-error-cs0619"></a><span data-ttu-id="55caf-102">Compilerfehler CS0619</span><span class="sxs-lookup"><span data-stu-id="55caf-102">Compiler Error CS0619</span></span>
<span data-ttu-id="55caf-103">Member ist veraltet: "Text"</span><span class="sxs-lookup"><span data-stu-id="55caf-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="55caf-104">Ein Klassenmember wurde mit markiert die [veraltet](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) Attribut, z. B., dass ein Fehler ausgegeben wird, wenn die Klassenmember verwiesen wird.</span><span class="sxs-lookup"><span data-stu-id="55caf-104">A class member was marked with the [Obsolete](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="55caf-105">Im folgenden Beispiel wird der Fehler CS0619 generiert:</span><span class="sxs-lookup"><span data-stu-id="55caf-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
