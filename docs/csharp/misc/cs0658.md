# <a name="compiler-warning-level-1-cs0658"></a>Compilerwarnung (Stufe 1) CS0658
"Attributmodifizierer" ist kein bekannter Attributpfad. Alle Attribute in diesem Block werden ignoriert.  
  
 Es wurde ein ungültiger Attributmodifizierer angegeben. Finden Sie unter [Attributziele](http://msdn.microsoft.com/library/59a261f0-1cfb-4aa5-b610-6b735389882c) für Weitere Informationen.  
  
 Im folgenden Beispiel wird CS0658 generiert:  
  
```csharp  
// CS0658.cs  
using System;  
public class TestAttribute : Attribute{}  
[badAttributeLocation: Test]   // CS0658, badAttributeLocation is invalid  
class ClassTest  
{  
   public static void Main()  
   {  
   }  
}  
```
