# <a name="compiler-error-cs0243"></a><span data-ttu-id="cf4d4-102">Compilerfehler CS0243</span><span class="sxs-lookup"><span data-stu-id="cf4d4-102">Compiler Error CS0243</span></span>

<span data-ttu-id="cf4d4-103">Das Conditional-Attribut ist für 'Methode' nicht gültig, da es eine Überschreibungsmethode ist.</span><span class="sxs-lookup"><span data-stu-id="cf4d4-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
 <span data-ttu-id="cf4d4-104">Das [Conditional](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) -Attribut für für Methoden nicht zulässig, die mit dem [override](../../csharp/language-reference/keywords/override.md) -Schlüsselwort gekennzeichnet sind.</span><span class="sxs-lookup"><span data-stu-id="cf4d4-104">The [Conditional](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="cf4d4-105">Weitere Informationen finden Sie unter [wann Override "und" neue Schlüsselwörter](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="cf4d4-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="cf4d4-106">Der Compiler bindet niemals Überschreibungsmethoden. Er bindet nur die Basismethode und die Common Language Runtime ruft dann die Überschreibung entsprechend auf.</span><span class="sxs-lookup"><span data-stu-id="cf4d4-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="cf4d4-107">Im folgenden Beispiel wird CS0243 generiert:</span><span class="sxs-lookup"><span data-stu-id="cf4d4-107">The following sample generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
