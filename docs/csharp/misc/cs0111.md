# <a name="compiler-error-cs0111"></a>Compilerfehler CS0111

Der Typ "Klasse" definiert bereits einen Member namens "Member" mit den gleichen Parametertypen.  
  
 CS0111 tritt auf, wenn eine Klasse zwei Memberdeklarationen enth√§lt, die denselben Namen und dieselben Parametertypen haben. Weitere Informationen finden Sie unter [Methoden](../programming-guide/classes-and-structs/methods.md).  
  
## <a name="example"></a>Beispiel  

 Im folgenden Beispiel wird CS0111 generiert:  
  
```csharp  
// CS0111.cs  
class A  
{  
   void Test() { }  
   public static void Test(){}   // CS0111  
  
   public static void Main() {}  
}  
```
