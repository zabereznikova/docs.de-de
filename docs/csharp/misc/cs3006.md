# <a name="compiler-warning-level-1-cs3006"></a>Compilerwarnung (Stufe 1) CS3006
Die überladene Methode 'Methode' weicht nur hinsichtlich des Verweises oder der Ausgabe ab, oder des Arrayrangs, und ist nicht CLS-kompatibel  
  
 Eine Methode kann nicht auf Basis der Parameter [ref](../../csharp/language-reference/keywords/ref.md) oder [out](../../csharp/language-reference/keywords/out.md) überladen werden und weiterhin der Common Language Specification (CLS) entsprechen. Weitere Informationen zu den CLS-Konformitätsregeln finden Sie unter [Writing CLS-Compliant Code (Schreiben von CLS-kompatiblem Code)](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3)und [Language Independence and Language-Independent Components (Sprachunabhängigkeit und sprachunabhängige Komponenten)](https://msdn.microsoft.com/library/12a7a7h3).  
  
## <a name="example"></a>Beispiel  
 Im folgenden Beispiel wird CS3006 generiert: Kommentieren Sie das Attribut auf Assemblyebene aus, oder entfernen Sie eine der Methodendefinitionen, um diese Warnung zu beheben.  
  
```  
// CS3006.cs  
  
using System;  
  
[assembly: CLSCompliant(true)]  
public class MyClass  
{  
    public void f(int i)  
    {  
    }  
  
    public void f(ref int i)   // CS3006  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
