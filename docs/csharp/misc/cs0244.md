 <span data-ttu-id="a0f6d-106">Im folgenden Beispiel wird CS0244 generiert:</span><span class="sxs-lookup"><span data-stu-id="a0f6d-106">The following sample generates CS0244:</span></span>  
  
```csharp

class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer

      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
