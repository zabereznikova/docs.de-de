- <span data-ttu-id="51beb-126">Beim Schreiben von LINQ-Abfragen empfiehlt sich diese Faustregel: Verwenden Sie die Abfragesyntax, wann immer es möglich ist, und verwenden Sie die Methodensyntax nur, wenn es nötig ist.</span><span class="sxs-lookup"><span data-stu-id="51beb-126">As a rule when you write LINQ queries, we recommend that you use query syntax whenever possible and method syntax whenever necessary.</span></span> <span data-ttu-id="51beb-127">Zwischen den beiden Formen gibt es keine semantischen oder leistungsbezogenen Unterschiede.</span><span class="sxs-lookup"><span data-stu-id="51beb-127">There is no semantic or performance difference between the two different forms.</span></span> <span data-ttu-id="51beb-128">Abfrageausdrücke sind oft besser lesbar als die entsprechenden in der Methodensyntax geschriebenen Ausdrücke.</span><span class="sxs-lookup"><span data-stu-id="51beb-128">Query expressions are often more readable than equivalent expressions written in method syntax.</span></span>
