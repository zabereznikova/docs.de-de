# <a name="compiler-error-cs0006"></a>Compilerfehler CS0006

Metadata file 'dll_name' could not be found (Die Metadatendatei „dll_name“ konnte nicht gefunden werden)
  
 Das Programm wurde kompiliert und hat den Namen der Datei explizit übergeben, die die Metadaten enthalten hat. Allerdings konnte die DLL-Datei nicht gefunden werden. Weitere Informationen finden Sie unter [-reference (C#-Compileroptionen)](../compiler-options/reference-compiler-option.md).
