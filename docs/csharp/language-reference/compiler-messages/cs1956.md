 <span data-ttu-id="26544-106">Diese Warnung kann generiert werden, wenn sich zwei Schnittstellenmethoden lediglich dadurch unterscheiden, ob ein bestimmter Parameter mit `ref` oder `out` gekennzeichnet ist.</span><span class="sxs-lookup"><span data-stu-id="26544-106">This warning can be generated when two interface methods are differentiated only by whether a particular parameter is marked with `ref` or with `out`.</span></span> <span data-ttu-id="26544-107">Es empfiehlt sich, Ihren Code zu ändern, um diese Warnung zu vermeiden, da es nicht offensichtlich oder garantiert ist, welche Methode zur Laufzeit aufgerufen wird.</span><span class="sxs-lookup"><span data-stu-id="26544-107">It is best to change your code to avoid this warning because it is not obvious or guaranteed which method is called at runtime.</span></span>

### <a name="to-avoid-this-warning"></a><span data-ttu-id="26544-110">So vermeiden Sie diese Warnung</span><span class="sxs-lookup"><span data-stu-id="26544-110">To avoid this warning</span></span>

## <a name="example"></a><span data-ttu-id="26544-113">Beispiel</span><span class="sxs-lookup"><span data-stu-id="26544-113">Example</span></span>  
 <span data-ttu-id="26544-114">Im folgende Code wird CS1956 generiert, da sich die zwei `Test`-Methoden in `Base` nur durch den Modifizierer `ref`/`out` auf dem ersten Parameter unterscheiden.</span><span class="sxs-lookup"><span data-stu-id="26544-114">The following code generates CS1956 because the two `Test` methods in `Base` differ only by the `ref`/`out` modifier on the first parameter.</span></span>  
  
```csharp  
// cs1956.cs  
class Base<T, S>  
{  
    // This is the method that should be called.  
    public virtual int Test(out T x) // CS1956  
    {  
        x = default(T);  
        return 0;  
    }  
  
    // This is the "last" method and is the one that ends up being called  
    public virtual int Test(ref S x)  
    {  
        return 1;  
    }  
}  
  
interface IFace  
{  
    int Test(out int x);  
}  
  
class Derived : Base<int, int>, IFace  
{  
    static int Main()  
    {  
        IFace x = new Derived();  
        int y;  
        return x.Test(out y);  
    }  
}  
```
