# <a name="compiler-error-cs1936"></a><span data-ttu-id="02809-102">Compilerfehler CS1936</span><span class="sxs-lookup"><span data-stu-id="02809-102">Compiler Error CS1936</span></span>

<span data-ttu-id="02809-103">Es konnte keine Implementierung des Abfragemusters für den Quelltyp "Typ" gefunden werden.</span><span class="sxs-lookup"><span data-stu-id="02809-103">Could not find an implementation of the query pattern for source type 'type'.</span></span>  <span data-ttu-id="02809-104">"Methode" wurde nicht gefunden.</span><span class="sxs-lookup"><span data-stu-id="02809-104">'method' not found.</span></span>  
  
<span data-ttu-id="02809-105">Der Typ muss die Standardabfrageoperator-Methoden implementieren, die Sie in der Abfrage aufrufen, um einen Quelltyp abzufragen.</span><span class="sxs-lookup"><span data-stu-id="02809-105">In order to query a source type, that type must implement the standard query operator methods that you are invoking in the query.</span></span> <span data-ttu-id="02809-106">Die Implementierung kann entweder in Form von Klassenmembern oder von Erweiterungsmethoden vorliegen, die mit der entsprechenden `using`-Anweisung in den Gültigkeitsbereich eingebunden werden.</span><span class="sxs-lookup"><span data-stu-id="02809-106">The implementation can be either in the form of class members or extension methods that are brought into scope with the appropriate `using` directive.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="02809-107">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="02809-107">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="02809-110">Beispiel</span><span class="sxs-lookup"><span data-stu-id="02809-110">Example</span></span>

<span data-ttu-id="02809-111">Im folgenden Beispiel wird CS1936 erzeugt:</span><span class="sxs-lookup"><span data-stu-id="02809-111">The following example produces CS1936:</span></span>  

```csharp
// cs1936.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```

<span data-ttu-id="02809-112">Dieser Fehler tritt normalerweise auf, wenn Sie versehentlich versuchen, ein Objekt eines Typs abzufragen, statt einer Sammlung dieser Objekte.</span><span class="sxs-lookup"><span data-stu-id="02809-112">This error typically occurs when you accidentally try to query an object of some type instead of a collection of those objects.</span></span>
