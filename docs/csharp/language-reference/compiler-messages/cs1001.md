# <a name="compiler-error-cs1001"></a><span data-ttu-id="d29f4-102">Compilerfehler CS1001</span><span class="sxs-lookup"><span data-stu-id="d29f4-102">Compiler Error CS1001</span></span>
<span data-ttu-id="d29f4-103">Bezeichner erwartet.</span><span class="sxs-lookup"><span data-stu-id="d29f4-103">Identifier expected</span></span>  
  
 <span data-ttu-id="d29f4-104">Sie haben keinen Bezeichner angegeben.</span><span class="sxs-lookup"><span data-stu-id="d29f4-104">You did not supply an identifier.</span></span> <span data-ttu-id="d29f4-105">Ein Bezeichner kann der Name einer Klasse, Struktur, Methode, Variable, eines Namespace usw. sein, den Sie angeben.</span><span class="sxs-lookup"><span data-stu-id="d29f4-105">An identifier is the name of a class, struct, namespace, method, variable and so on that you provide.</span></span>  
  
 <span data-ttu-id="d29f4-106">In folgendem Beispiel wird eine einfache Klasse deklariert, ohne ihr allerdings einen Namen zu geben:</span><span class="sxs-lookup"><span data-stu-id="d29f4-106">The following example declares a simple class but does not give the class a name:</span></span>  
  
```csharp  
//cs1001.cs  
public class              //CS1001  
    {  
        public int Num {get; set;}  
        void MethodA(){}  
    }  
```  
  
 <span data-ttu-id="d29f4-107">Im folgenden Beispiel wird der Fehler CS1001 generiert, weil Sie Member angeben müssen, wenn Sie eine Enumeration deklarieren:</span><span class="sxs-lookup"><span data-stu-id="d29f4-107">The following sample generates CS1001 because, when declaring an enum, you must specify members:</span></span>  
  
```csharp  
// CS1001.cs  
public class clx  
{  
   enum Colors : int  
   {  
      'a', 'b' // CS1001, 'a' is not a valid int identifier  
       // The following line shows examples of valid identifiers:  
       // Blue, Red, Orange  
   };  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="d29f4-108">Parameternamen sind auch dann erforderlich, wenn der Compiler sie z.B. nicht in einer Schnittstellendefinition verwendet.</span><span class="sxs-lookup"><span data-stu-id="d29f4-108">Parameter names are required even if the compiler doesn't use them, for example, in an interface definition.</span></span> <span data-ttu-id="d29f4-109">Diese Parameter sind erforderlich, damit die Programmierer, die die Schnittstelle verwenden, Informationen darüber erhalten, was die Parameter bedeuten.</span><span class="sxs-lookup"><span data-stu-id="d29f4-109">These parameters are required so that programmers who are consuming the interface know something about the what the parameters mean.</span></span>  
  
```csharp  
// CS1001-2.cs  
// compile with: /target:library  
interface IMyTest  
{  
   void TestFunc1(int, int);  // CS1001  
   // Use the following line instead:  
   // void TestFunc1(int a, int b);  
}  
  
class CMyTest : IMyTest  
{  
   void IMyTest.TestFunc1(int a, int b)  
   {  
   }  
}  
```
