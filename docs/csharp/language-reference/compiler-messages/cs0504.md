 <span data-ttu-id="fdef6-104">Wenn eine Variable [const (konstant)](../../../csharp/language-reference/keywords/const.md) ist, ist sie auch [static (statisch)](../../../csharp/language-reference/keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="fdef6-104">If a variable is [const](../../../csharp/language-reference/keywords/const.md), it is also [static](../../../csharp/language-reference/keywords/static.md).</span></span> <span data-ttu-id="fdef6-105">Wenn Sie eine **konstante** und eine **statische** Variable benötigt, deklarieren Sie die Variable als **konstant**.
