# Auswahlanweisungen (C#-Referenz)
Eine Auswahlanweisung bewirkt, dass die Programmsteuerung in Abhängigkeit davon, ob eine bestimmte Bedingung `true` ist, an einen bestimmten Ablauf übergeben wird.  
  
 In Auswahlanweisungen werden die folgenden Schlüsselwörter verwendet:  
  
-   [if](../../../csharp/language-reference/keywords/if-else.md)  
  
-   [else](../../../csharp/language-reference/keywords/if-else.md)  
  
-   [switch](../../../csharp/language-reference/keywords/switch.md)  
  
-   [case](../../../csharp/language-reference/keywords/switch.md)  
  
-   [default](../../../csharp/language-reference/keywords/switch.md)
