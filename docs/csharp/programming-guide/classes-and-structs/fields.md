<span data-ttu-id="e5935-103">Ein *Feld* ist eine Variable eines beliebigen Typs, die direkt in einer [class](../../../csharp/language-reference/keywords/class.md) oder [struct](../../../csharp/language-reference/keywords/struct.md) deklariert ist.</span><span class="sxs-lookup"><span data-stu-id="e5935-103">A *field* is a variable of any type that is declared directly in a [class](../../../csharp/language-reference/keywords/class.md) or [struct](../../../csharp/language-reference/keywords/struct.md).</span></span> <span data-ttu-id="e5935-104">Felder sind *Member* Ihres enthaltenden Typs.</span><span class="sxs-lookup"><span data-stu-id="e5935-104">Fields are *members* of their containing type.</span></span>
