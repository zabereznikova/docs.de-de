<span data-ttu-id="dd7c3-138">Benutzerdefinierte Operatoren müssen immer als `public` und `static` deklariert werden.</span><span class="sxs-lookup"><span data-stu-id="dd7c3-138">User-defined operators must always be declared as `public` and `static`.</span></span> <span data-ttu-id="dd7c3-139">Weitere Informationen finden Sie unter [Operatorüberladung](../../language-reference/operators/operator-overloading.md).</span><span class="sxs-lookup"><span data-stu-id="dd7c3-139">For more information, see [Operator overloading](../../language-reference/operators/operator-overloading.md).</span></span>

Außerdem verfügen Schnittstellen wie Klassen und Strukturen standardmäßig über `internal`-Zugriff.</span><span class="sxs-lookup"><span data-stu-id="dd7c3-143">Interfaces declared directly within a namespace can be `public` or `internal` and, just like classes and structs, interfaces default to `internal` access.</span></span> <span data-ttu-id="dd7c3-144">Für Schnittstellenmember ist standardmäßig `public` festgelegt, da es der Zweck einer Schnittstelle ist, anderen Typen den Zugriff auf eine Klasse oder Struktur zu ermöglichen.</span><span class="sxs-lookup"><span data-stu-id="dd7c3-144">Interface members are `public` by default because the purpose of an interface is to enable other types to access a class or struct.</span></span> <span data-ttu-id="dd7c3-145">Schnittstellenmemberdeklarationen können einen beliebigen Zugriffsmodifizierer aufweisen.</span><span class="sxs-lookup"><span data-stu-id="dd7c3-145">Interface member declarations may include any access modifier.</span></span> <span data-ttu-id="dd7c3-146">Das ist für statische Methoden besonders nützlich, um allgemeine Implementierungen bereitzustellen, die von allen Implementierern einer Klasse benötigt werden.</span><span class="sxs-lookup"><span data-stu-id="dd7c3-146">This is most useful for static methods to provide common implementations needed by all implementors of a class.</span></span>
