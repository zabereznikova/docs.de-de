# <a name="nullable-types-c-programming-guide"></a><span data-ttu-id="af592-103">Nullable-Typen (C#-Programmierhandbuch)</span><span class="sxs-lookup"><span data-stu-id="af592-103">Nullable types (C# Programming Guide)</span></span>

<span data-ttu-id="af592-104">Auf NULL festlegbare Typen sind Instanzen der <xref:System.Nullable%601?displayProperty=nameWithType>-Struktur.</span><span class="sxs-lookup"><span data-stu-id="af592-104">Nullable types are instances of the <xref:System.Nullable%601?displayProperty=nameWithType> struct.</span></span> <span data-ttu-id="af592-105">Nullable-Typen können alle Werte eines zugrunde liegenden `T`-Typs und einen zusätzlichen [NULL](../../language-reference/keywords/null.md)-Wert darstellen.</span><span class="sxs-lookup"><span data-stu-id="af592-105">Nullable types can represent all the values of an underlying type `T`, and an additional [null](../../language-reference/keywords/null.md) value.</span></span> <span data-ttu-id="af592-106">Der zugrunde liegende Typ `T` kann jeder [Werttyp](../../language-reference/keywords/value-types.md) sein, der keine NULL-Werte zulässt.</span><span class="sxs-lookup"><span data-stu-id="af592-106">The underlying type `T` can be any non-nullable [value type](../../language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="af592-107">`T` kann kein Referenztyp sein.</span><span class="sxs-lookup"><span data-stu-id="af592-107">`T` cannot be a reference type.</span></span>

<span data-ttu-id="af592-108">Sie können beispielsweise `null` oder jeden anderen Integerwert von <xref:System.Int32.MinValue?displayProperty=nameWithType> bis <xref:System.Int32.MaxValue?displayProperty=nameWithType> einem `Nullable<int>`-, [TRUE](../../language-reference/keywords/true-literal.md)- und [FALSE](../../language-reference/keywords/false-literal.md)-Wert zuweisen, oder `null` zu `Nullable<bool>`.</span><span class="sxs-lookup"><span data-stu-id="af592-108">For example, you can assign `null` or any integer value from <xref:System.Int32.MinValue?displayProperty=nameWithType> to <xref:System.Int32.MaxValue?displayProperty=nameWithType> to a `Nullable<int>` and [true](../../language-reference/keywords/true-literal.md), [false](../../language-reference/keywords/false-literal.md), or `null` to a `Nullable<bool>`.</span></span>

<span data-ttu-id="af592-109">Sie verwenden einen Nullable-Typ, wenn Sie den nicht definierten Wert eines zugrunde liegenden Typs darstellen müssen.</span><span class="sxs-lookup"><span data-stu-id="af592-109">You use a nullable type when you need to represent the undefined value of an underlying type.</span></span> <span data-ttu-id="af592-110">Eine boolesche Variable darf nur zwei Werte besitzen: TRUE und FALSE.</span><span class="sxs-lookup"><span data-stu-id="af592-110">A Boolean variable can have only two values: true and false.</span></span> <span data-ttu-id="af592-111">Es gibt keinen „nicht definierten“ Wert.</span><span class="sxs-lookup"><span data-stu-id="af592-111">There is no "undefined" value.</span></span> <span data-ttu-id="af592-112">In den meisten Programmierungsanwendungen – allen voran Datenbankinteraktionen – kann ein Variablenwert in einem nicht definierten Zustand vorkommen oder fehlen.</span><span class="sxs-lookup"><span data-stu-id="af592-112">In many programming applications, most notably database interactions, a variable value can be undefined or missing.</span></span> <span data-ttu-id="af592-113">Ein Feld in einer Datenbank kann z.B. die Werte TRUE und FALSE enthalten; ebenso kann es aber auch gar keinen Wert enthalten.</span><span class="sxs-lookup"><span data-stu-id="af592-113">For example, a field in a database may contain the values true or false, or it may contain no value at all.</span></span> <span data-ttu-id="af592-114">In diesem Fall verwenden Sie einen `Nullable<bool>`-Typ.</span><span class="sxs-lookup"><span data-stu-id="af592-114">You use a `Nullable<bool>` type in that case.</span></span>

<span data-ttu-id="af592-115">Nullable-Typen weisen die folgenden Eigenschaften auf:</span><span class="sxs-lookup"><span data-stu-id="af592-115">Nullable types have the following characteristics:</span></span>
  
- <span data-ttu-id="af592-116">Nullable-Typen stellen Werttypvariablen dar, denen der Wert `null` zugewiesen werden kann.</span><span class="sxs-lookup"><span data-stu-id="af592-116">Nullable types represent value-type variables that can be assigned the `null` value.</span></span> <span data-ttu-id="af592-117">Sie können keinen Nullable-Typ basierend auf einem Verweistyp erstellen.</span><span class="sxs-lookup"><span data-stu-id="af592-117">You cannot create a nullable type based on a reference type.</span></span> <span data-ttu-id="af592-118">(Verweistypen unterstützen immer den `null`-Wert.)</span><span class="sxs-lookup"><span data-stu-id="af592-118">(Reference types already support the `null` value.)</span></span>  
  
- <span data-ttu-id="af592-119">Die Syntax `T?` ist eine Kurzform für `Nullable<T>`.</span><span class="sxs-lookup"><span data-stu-id="af592-119">The syntax `T?` is shorthand for `Nullable<T>`.</span></span> <span data-ttu-id="af592-120">Die beiden Formen sind austauschbar.</span><span class="sxs-lookup"><span data-stu-id="af592-120">The two forms are interchangeable.</span></span>  
  
- <span data-ttu-id="af592-121">Sie weisen einem Nullable-Typ einen Wert genauso zu, wie Sie es für einen zugrunde liegenden Werttyp tun würden: `int? x = 10;` oder `double? d = 4.108;`.</span><span class="sxs-lookup"><span data-stu-id="af592-121">Assign a value to a nullable type just as you would for an underlying value type: `int? x = 10;` or `double? d = 4.108;`.</span></span> <span data-ttu-id="af592-122">Sie können ebenfalls den `null`-Wert zuweisen: `int? x = null;`.</span><span class="sxs-lookup"><span data-stu-id="af592-122">You also can assign the `null` value: `int? x = null;`.</span></span>  
  
- <span data-ttu-id="af592-123">Verwenden Sie die schreibgeschützten Eigenschaften <xref:System.Nullable%601.HasValue%2A?displayProperty=nameWithType> und <xref:System.Nullable%601.Value%2A?displayProperty=nameWithType>, um auf NULL zu prüfen und den Wert abzurufen, wie im folgenden Beispiel gezeigt: `if (x.HasValue) y = x.Value;`</span><span class="sxs-lookup"><span data-stu-id="af592-123">Use the <xref:System.Nullable%601.HasValue%2A?displayProperty=nameWithType> and <xref:System.Nullable%601.Value%2A?displayProperty=nameWithType> readonly properties to test for null and retrieve the value, as shown in the following example: `if (x.HasValue) y = x.Value;`</span></span>  
  
  - <span data-ttu-id="af592-124">Die Eigenschaft <xref:System.Nullable%601.HasValue%2A> gibt `true` zurück, wenn die Variable einen Wert enthält, oder sie gibt `false` zurück, wenn die Variable `null` ist.</span><span class="sxs-lookup"><span data-stu-id="af592-124">The <xref:System.Nullable%601.HasValue%2A> property returns `true` if the variable contains a value, or `false` if it's `null`.</span></span>
  
  - <span data-ttu-id="af592-125">Die Eigenschaft <xref:System.Nullable%601.Value%2A> gibt einen Wert zurück, sofern <xref:System.Nullable%601.HasValue%2A> `true` zurückgibt.</span><span class="sxs-lookup"><span data-stu-id="af592-125">The <xref:System.Nullable%601.Value%2A> property returns a value if <xref:System.Nullable%601.HasValue%2A> returns `true`.</span></span> <span data-ttu-id="af592-126">Andernfalls wird eine <xref:System.InvalidOperationException> ausgelöst.</span><span class="sxs-lookup"><span data-stu-id="af592-126">Otherwise, an <xref:System.InvalidOperationException> is thrown.</span></span>  
  
- <span data-ttu-id="af592-127">Sie können wie im folgenden Beispiel dargestellt auch die Operatoren `==` und `!=` mit einem Nullable-Typ verwenden: `if (x != null) y = x.Value;`</span><span class="sxs-lookup"><span data-stu-id="af592-127">You can also use the `==` and `!=` operators with a nullable type, as shown in the following example: `if (x != null) y = x.Value;`.</span></span> <span data-ttu-id="af592-128">Wenn `a` und `b` jeweils NULL sind, ergibt `a == b` `true`.</span><span class="sxs-lookup"><span data-stu-id="af592-128">If `a` and `b` are both null, `a == b` evaluates to `true`.</span></span>  

- <span data-ttu-id="af592-129">Ab C# 7.0 können Sie den [Musterabgleich](../../pattern-matching.md#the-is-type-pattern-expression) verwenden, um einen Wert eines Nullable-Typs zu untersuchen und abzurufen: `if (x is int valueOfX) y = valueOfX;`.</span><span class="sxs-lookup"><span data-stu-id="af592-129">Beginning with C# 7.0, you can use [pattern matching](../../pattern-matching.md#the-is-type-pattern-expression) to both examine and get a value of a nullable type: `if (x is int valueOfX) y = valueOfX;`.</span></span>
  
- <span data-ttu-id="af592-130">Der Standardwert von `T?` ist eine Instanz, deren <xref:System.Nullable%601.HasValue%2A>-Eigenschaft `false` zurückgibt.</span><span class="sxs-lookup"><span data-stu-id="af592-130">The default value of `T?` is an instance whose <xref:System.Nullable%601.HasValue%2A> property returns `false`.</span></span>  

- <span data-ttu-id="af592-131">Verwenden Sie die <xref:System.Nullable%601.GetValueOrDefault>-Methode, um entweder den zugewiesenen Wert oder den [Standardwert](../../language-reference/keywords/default-values-table.md) für den zugrunde liegenden Werttyp zurückzugeben, wenn der Wert des Nullable-Typs `null` ist.</span><span class="sxs-lookup"><span data-stu-id="af592-131">Use the <xref:System.Nullable%601.GetValueOrDefault> method to return either the assigned value, or the [default](../../language-reference/keywords/default-values-table.md) value of the underlying value type if the value of the nullable type is `null`.</span></span>  

- <span data-ttu-id="af592-132">Verwenden Sie die <xref:System.Nullable%601.GetValueOrDefault(%600)>-Methode, um entweder den zugewiesenen Wert oder den bereitgestellten Standardwert zurückzugeben, wenn der Wert des Nullable-Typs `null` ist.</span><span class="sxs-lookup"><span data-stu-id="af592-132">Use the <xref:System.Nullable%601.GetValueOrDefault(%600)> method to return either the assigned value, or the provided default value if the value of the nullable type is `null`.</span></span>
  
- <span data-ttu-id="af592-133">Verwenden Sie den [NULL-Sammeloperator](../../language-reference/operators/null-coalescing-operator.md), `??`, um einen Wert einem zugrunde liegenden Typ auf Grundlage eines Werts des Nullable-Typs zuzuweisen: `int? x = null; int y = x ?? -1;`</span><span class="sxs-lookup"><span data-stu-id="af592-133">Use the [null-coalescing operator](../../language-reference/operators/null-coalescing-operator.md), `??`, to assign a value to an underlying type based on a value of the nullable type: `int? x = null; int y = x ?? -1;`.</span></span> <span data-ttu-id="af592-134">Da `x` NULL ist, ist in diesem Beispiel der Ergebniswert von `y` `-1`.</span><span class="sxs-lookup"><span data-stu-id="af592-134">In the example, since `x` is null, the result value of `y` is `-1`.</span></span>

- <span data-ttu-id="af592-135">Wenn eine benutzerdefinierte Konvertierung zwischen zwei Datentypen definiert ist, kann die gleiche Konvertierung auch mit der Version dieser Datentypen verwendet werden, die NULL-Werte zulassen.</span><span class="sxs-lookup"><span data-stu-id="af592-135">If a user-defined conversion is defined between two data types, the same conversion can also be used with the nullable versions of these data types.</span></span>
  
- <span data-ttu-id="af592-136">Geschachtelte Nullable-Typen sind nicht zulässig.</span><span class="sxs-lookup"><span data-stu-id="af592-136">Nested nullable types are not allowed.</span></span> <span data-ttu-id="af592-137">Die folgende Zeile wird nicht kompiliert: `Nullable<Nullable<int>> n;`</span><span class="sxs-lookup"><span data-stu-id="af592-137">The following line doesn't compile: `Nullable<Nullable<int>> n;`</span></span>
